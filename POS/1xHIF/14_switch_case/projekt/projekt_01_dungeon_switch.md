# Jahresprojekt: Dungeon Crawler — switch-case

Baue switch-case in deinen Dungeon Crawler ein.

## Aufgaben

1. **Menüverwaltung:** Der Spieler wählt eine Aktion (1 = Angreifen, 2 = Inventar, 3 = Heilen, 4 = Fliehen). Lies die Eingabe als int oder String und verarbeite sie in einem switch.
2. **default:** Verwende `default` für ungültige Eingaben.
3. **Erweiterung:** Füge weitere Menüpunkte hinzu (z. B. 5 = Status anzeigen).
