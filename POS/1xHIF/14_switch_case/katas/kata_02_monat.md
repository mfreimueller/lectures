# K2: Monatsname

**Ziel:** switch-case mit default.

Der Benutzer gibt eine Zahl 1-12 ein. Gib den passenden Monatsnamen aus (1 = Jänner, 2 = Februar, ..., 12 = Dezember). Verwende default für ungültige Eingaben.

**Beispielablauf:**
```
Gib eine Zahl (1-12) ein: 5
Mai
```

```
Gib eine Zahl (1-12) ein: 13
Ungültige Eingabe!
```
