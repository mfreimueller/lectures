# Exercise: Mini-Taschenrechner mit switch

**Ziel:** Einen Taschenrechner mit switch-case bauen.

## Anforderungen

1. Der Benutzer gibt zwei Zahlen (Kommazahlen erlaubt) und einen Operator (`+`, `-`, `*`, `/`) ein.
2. Verwende `switch` auf dem Operator-String.
3. Gib das Ergebnis aus.
4. Mit `default` eine Fehlermeldung für ungültige Operatoren.

## Beispielablauf

```
Erste Zahl: 10
Zweite Zahl: 3
Operator (+ - * /): /
Ergebnis: 3.333...
```

```
Erste Zahl: 5
Zweite Zahl: 3
Operator (+ - * /): %
Ungültiger Operator!
```
