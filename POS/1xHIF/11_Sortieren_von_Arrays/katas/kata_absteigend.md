# K2: Absteigend sortieren

Modifiziere Bubblesort so, dass das Array absteigend (größte zuerst) sortiert wird.

Verwende das Array: `{5, 2, 8, 1, 9, 3}`

**Beispielausgabe:**

```
Vorher: 5 2 8 1 9 3
Nachher: 9 8 5 3 2 1
```

### Hinweise

- Ändere die Vergleichsoperation: Tausche, wenn das linke Element kleiner ist.
- Aus `arr[j] > arr[j+1]` wird `arr[j] < arr[j+1]`.
