# Dungeon Escape - JavaDoc

## Aufgaben
Dokumentiere alle vorhandenen Klassen und Methoden deines Dungeon Escape Spiels mit JavaDoc.

## JavaDoc-Anforderungen
Jede Klasse muss enthalten:
- @author (dein Name)
- @version (Versionsnummer)

Jede Methode muss enthalten:
- @param fuer jeden Parameter
- @return fuer den Rueckgabewert (ausser void)
- Beschreibung der Funktionalitaet

## Generierung
Generiere die HTML-Dokumentation:
- IntelliJ: Tools -> Generate JavaDoc
- Oder Kommandozeile: javadoc -d doc *.java

Oeffne die generierte index.html im Browser und ueberpruefe das Ergebnis.
