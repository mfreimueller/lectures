# Kata 03: Eigene Klasse dokumentieren

Waehle eine deiner bisherigen Klassen aus (z.B. Auto, Spieler, Raum)
und dokumentiere sie vollstaendig mit JavaDoc.

## Anforderungen
- Klassen-JavaDoc mit @author
- Methoden-JavaDoc mit @param und @return
- Generiere die HTML-Dokumentation

## Vorgehen
1. Kopiere den Code deiner Klasse
2. Fuege JavaDoc-Kommentare hinzu
3. Lasse einen Partner die Dokumentation reviewen
4. Generiere die HTML-Dokumentation (IntelliJ: Tools -> Generate JavaDoc)
5. Oeffne index.html im Browser
