OO-Grundlagen: Wiederholung

Klasse = Bauplan


public class Auto {
    private String marke;   // Attribut
    private int baujahr;

    public Auto(String marke, int baujahr) { // Konstruktor
        this.marke = marke;
        this.baujahr = baujahr;
    }

    public void hupen() {   // Methode
        System.out.println("Hupe!");
    }
}

Objekt = Instanz einer Klasse: Auto a = new Auto("BMW", 2020);

Assoziationen: Objekte als Attribute

hat-ein-Beziehung


public class Person {
    private Adresse adresse; // Person HAT EINE Adresse

    public String getOrt() {
        if (adresse != null) {
            return adresse.getOrt();
        }
        return "Unbekannt";
    }
}
  • Objekte werden als Referenz ubergeben
  • null prufen vor dem Zugriff
  • Kettenzugriff: person.getAdresse().getOrt()
