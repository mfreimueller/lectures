Das Problem mit vielen if-else

if (tag == 1) {
    System.out.println("Montag");
} else if (tag == 2) {
    System.out.println("Dienstag");
} else if (tag == 3) {
    System.out.println("Mittwoch");
} // ... bis Sonntag

Viel Code, unübersichtlich.

switch-case - Syntax

switch (variable) {
    case wert1:
        // Code für wert1
        break;
    case wert2:
        // Code für wert2
        break;
    default:
        // Code für alle anderen Werte
        break;
}

Wochentag mit switch

int tag = 3;
String name;

switch (tag) {
    case 1: name = "Montag";    break;
    case 2: name = "Dienstag";  break;
    case 3: name = "Mittwoch";  break;
    case 4: name = "Donnerstag";break;
    case 5: name = "Freitag";   break;
    case 6: name = "Samstag";   break;
    case 7: name = "Sonntag";   break;
    default: name = "Ungültig";break;
}

System.out.println(name);  // Mittwoch

break nicht vergessen!

int x = 1;

switch (x) {
    case 1: System.out.println("Eins");  // break fehlt!
    case 2: System.out.println("Zwei"); break;
    default: System.out.println("Sonst"); break;
}
// Ausgabe: Eins Zwei (Fall-through!)

Ohne break wird der nächste case ausgeführt -> Fall-through.

Fall-through nutzen

int monat = 2;
int tage;

switch (monat) {
    case 1: case 3: case 5: case 7:
    case 8: case 10: case 12:
        tage = 31; break;
    case 4: case 6: case 9: case 11:
        tage = 30; break;
    case 2:
        tage = 28; break;  // Schaltjahr ignorieren
    default:
        tage = 0; break;
}

if vs. switch

if-elseswitch
Beliebige BedingungenNur == mit Konstanten
Komplexe AusdrückeEinfache Werte (int, char, String)
Unübersichtlich bei vielen ZweigenÜbersichtlich bei vielen Zweigen

switch mit String (ab Java 7)

String befehl = "status";

switch (befehl) {
    case "kämpfen":
        System.out.println("Angriff!");
        break;
    case "fliehen":
        System.out.println("Flucht!");
        break;
    case "status":
        System.out.println("HP: 100");
        break;
    default:
        System.out.println("Unbekannt");
        break;
}
