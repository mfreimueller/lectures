# K1: this-Schlüsselwort

Schreibe eine Klasse `Produkt` mit den privaten Attributen `name` (String) und `preis` (double).

Erstelle einen Konstruktor `Produkt(String name, double preis)`, der `this` verwendet, weil die Parameter und Attribute die gleichen Namen haben.

Erzeuge im `main` zwei Produkte und gib sie auf der Konsole aus.

Zusatz: Füge eine Methode `void printInfo()` hinzu, die `this` zur Klarheit verwendet.
