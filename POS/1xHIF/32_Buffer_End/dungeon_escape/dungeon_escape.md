# Dungeon Escape - Prasentation

## Prasentation (5 Minuten pro Person)
Zeige dein Dungeon Escape Spiel der Klasse.

### Folgende Punkte sollten vorkommen:
1. Welche Features hast du implementiert?
2. Zeige eine kurze Demo des Spiels
3. Was war die groesste Herausforderung?
4. Was hast du dabei gelernt?

### Reflexion im Plenum
- Was hat dir am Projekt gefallen?
- Was wurdest du anders machen?
- Welche Themen waren besonders interessant?

### Abschluss
- Letzte Fragen zum Stoff
- Ausblick auf die 2. Klasse
- Verabschiedung
