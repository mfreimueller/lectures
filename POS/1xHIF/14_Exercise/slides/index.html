Häufige Fehler

Fehlendes Semikolon

String-Vergleich mit == statt equals()

if (name == "Max")  // FALSCH
if (name.equals("Max")) // RICHTIG

Off-by-one in Schleifen

for (int i = 1; i < 10; i++) // 1-9, nicht 1-10
for (int i = 1; i <= 10; i++) // 1-10, korrekt
