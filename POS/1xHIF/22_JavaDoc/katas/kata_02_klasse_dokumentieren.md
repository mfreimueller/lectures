# K2: Klasse dokumentieren

Schreibe eine Klasse `Rechner` mit folgenden Methoden:

- `int add(int a, int b)` — Addition
- `int subtract(int a, int b)` — Subtraktion
- `int multiply(int a, int b)` — Multiplikation
- `double divide(int a, int b)` — Division (prüfe auf Division durch 0)

Dokumentiere die Klasse mit:
- @author (dein Name)
- @since (aktuelles Datum)

Dokumentiere jede Methode mit:
- Beschreibung
- @param für alle Parameter
- @return für den Rückgabewert

Teste alle Methoden im main.
