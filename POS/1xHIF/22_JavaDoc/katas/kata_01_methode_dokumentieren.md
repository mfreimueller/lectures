# K1: Methode dokumentieren

Schreibe eine Methode `public static int max(int[] arr)` in einer Klasse `ArrayUtils`.

Dokumentiere die Methode vollständig mit JavaDoc:
- Beschreibung: "Findet das größte Element in einem Integer-Array"
- @param arr Das Array mit den Werten
- @return Das größte Element
- @see Math#max(int, int)

Implementiere die Methode. Teste sie im main mit einem Array.
