Das Problem

Scanner.nextInt() liefert eine Zahl.

Aber: Scanner.next() liefert einen String.

Was, wenn wir aus einem String eine Zahl machen müssen?

Integer.parseInt()

String zahlAlsText = "42";
int zahl = Integer.parseInt(zahlAlsText);

System.out.println(zahl + 10);  // 52 (Rechnung!)
// Ohne parseInt: "42" + 10 = "4210" (Text!)

Float.parseFloat()

String preisAlsText = "19.99";
double preis = Float.parseFloat(preisAlsText);

System.out.println(preis * 2);  // 39.98

Boolean.parseBoolean()

String wert1 = "true";
String wert2 = "false";
String wert3 = "maybe";

boolean b1 = Boolean.parseBoolean(wert1);  // true
boolean b2 = Boolean.parseBoolean(wert2);  // false
boolean b3 = Boolean.parseBoolean(wert3);  // false (nur "true" wird true)

Fehlerbehandlung (Vorschau)

String eingabe = "abc";
// int zahl = Integer.parseInt(eingabe);  // Programm stürzt ab!

// Bessere Variante (Vorschau auf Exception-Handling):
try {
    int zahl = Integer.parseInt(eingabe);
    System.out.println(zahl);
} catch (NumberFormatException e) {
    System.out.println("Keine gültige Zahl!");
}

Kata 2: Binär zu Dezimal

String binär = "1010";
int dezimal = 0;

for (int i = 0; i < binär.length(); i++) {
    char c = binär.charAt(i);
    int ziffer = Integer.parseInt(String.valueOf(c));
    dezimal = dezimal * 2 + ziffer;
}

System.out.println(binär + " = " + dezimal);  // 1010 = 10

Dungeon Escape: Heiltranke

Scanner scanner = new Scanner(System.in);

System.out.print("Wie viele Heiltränke? ");
String eingabe = scanner.nextLine();
int anzahl = Integer.parseInt(eingabe);

int hp = 50;
int heilung = anzahl * 15;
hp += heilung;

System.out.println("Du heilst " + heilung + " HP. Neue HP: " + hp);
