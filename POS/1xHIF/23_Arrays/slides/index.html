Array = Sammlung gleichartiger Elemente

Definition

Ein Array ist eine Sammlung von Elementen desselben Datentyps mit fester Lange.

Visualisierung


Noten: [ 2 | 3 | 1 | 4 | 5 ]
Index:   0   1   2   3   4

Deklaration: int[] noten;

Zwei Schreibweisen:


// Empfohlen (Typ gefolgt von []):
int[] noten;

// Auch moeglich ([] nach dem Namen):
int noten[];

Der Array wird nur deklariert, noch nicht mit Lange versehen.

Initialisierung: noten = new int[5];


int[] noten;          // Deklaration
noten = new int[5];   // Array mit 5 Elementen (Index 0-4)

// Oder in einer Zeile:
int[] noten = new int[5];

Standardwerte: int=0, double=0.0, boolean=false, Referenzen=null

Kurzform: int[] noten = {1, 2, 3, 4, 5};


// Kurzform mit Initialisierung:
int[] noten = {1, 2, 3, 4, 5};

// Gleich bedeutend:
int[] noten = new int[]{1, 2, 3, 4, 5};


// Auch fur Strings und Objekte:
String[] namen = {"Anna", "Ben", "Clara"};

Index-Zugriff: noten[0] bis noten[laenge-1]


int[] noten = {2, 3, 1, 4, 5};

System.out.println(noten[0]); // 2 (erstes Element)
System.out.println(noten[2]); // 1 (drittes Element)
System.out.println(noten[4]); // 5 (letztes Element)

noten[1] = 2; // Andern: noten[1] ist jetzt 2 statt 3

Achtung: Index von 0 bis laenge-1! noten[5] -> ArrayIndexOutOfBoundsException!

Array-Lange: noten.length


int[] noten = {2, 3, 1, 4, 5};
int laenge = noten.length; // 5

// Letztes Element:
int letztes = noten[noten.length - 1]; // 5

.length ist ein Attribut (keine Methode!), daher keine Klammern.

for uber Array


int[] noten = {2, 3, 1, 4, 5};

// Alle Elemente ausgeben:
for (int i = 0; i < noten.length; i++) {
    System.out.println("Index " + i + ": " + noten[i]);
}

Live-Coding: Summe aller Elemente


public class ArraySumme {
    public static void main(String[] args) {
        int[] zahlen = {5, 10, 15, 20, 25};
        int summe = 0;

        for (int i = 0; i < zahlen.length; i++) {
            summe = summe + zahlen[i];
            // oder: summe += zahlen[i];
        }

        System.out.println("Summe: " + summe); // 75
    }
}

Live-Coding: Minimum/Maximum suchen


int[] zahlen = {12, 7, 19, 3, 25, 8};
int min = zahlen[0];
int max = zahlen[0];

for (int i = 1; i < zahlen.length; i++) {
    if (zahlen[i] < min) {
        min = zahlen[i];
    }
    if (zahlen[i] > max) {
        max = zahlen[i];
    }
}
System.out.println("Min: " + min + ", Max: " + max);

Array als Parameter in Methoden


public class ArrayUtils {
    // Array als Parameter
    public static void printArray(int[] arr) {
        for (int i = 0; i < arr.length; i++) {
            System.out.print(arr[i] + " ");
        }
        System.out.println();
    }

    public static void main(String[] args) {
        int[] zahlen = {1, 2, 3, 4, 5};
        ArrayUtils.printArray(zahlen); // 1 2 3 4 5
    }
}

Array als Ruckgabewert


public class ArrayUtils {
    // Array als Ruckgabewert
    public static int[] createArray(int size) {
        int[] result = new int[size];
        for (int i = 0; i < size; i++) {
            result[i] = i * 10;
        }
        return result;
    }

    public static void main(String[] args) {
        int[] werte = ArrayUtils.createArray(5);
        // werte = {0, 10, 20, 30, 40}
    }
}

Dungeon Escape: Raum[] statt Einzel-Attribute

Bisher: Vier einzelne Attribute


private Raum norden;
private Raum sueden;
private Raum osten;
private Raum westen;

Neu: Ein Array von Raumen


public class Raum {
    public static final int NORDEN = 0;
    public static final int SUEDEN = 1;
    public static final int OSTEN = 2;
    public static final int WESTEN = 3;

    private Raum[] ausgaenge = new Raum[4];

    public void setAusgang(int richtung, Raum raum) {
        ausgaenge[richtung] = raum;
    }

    public Raum getAusgang(int richtung) {
        return ausgaenge[richtung];
    }
}

Dungeon Escape Aufgabe

  • Ersetze die vier Richtungs-Attribute durch ein Raum[4]-Array
  • Definiere Index-Konstanten: NORDEN=0, SUEDEN=1, OSTEN=2, WESTEN=3
  • Passe alle Bewegungsmethoden im Spieler an
  • Implementiere getAusgaenge() zuruck einen String mit allen Richtungen
  • Teste, ob alles noch funktioniert wie vorher

Zusammenfassung

  • Array = Sammlung gleichartiger Elemente mit fester Lange
  • Deklaration: int[] arr;
  • Initialisierung: new int[5] oder {1, 2, 3}
  • Index von 0 bis laenge-1
  • .length fur die Lange (ohne Klammern!)
  • For-Schleife: for (int i = 0; i < arr.length; i++)
  • Arrays als Parameter und Ruckgabewert von Methoden
