# K1: Erste Methode

**Ziel:** Die erste eigene Methode schreiben.

Schreibe eine Methode `public static void sayHello()`, die "Hallo!" auf der Konsole ausgibt. Rufe sie in der `main`-Methode auf.

**Beispielablauf:**
```
Hallo!
```

**Hinweis:** Die Methode ist `static`, weil wir sie aus `main` aufrufen (auch `main` ist static).
