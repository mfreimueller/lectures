# K4: Boolesche Methode

**Ziel:** Eine Methode mit boolean-Rückgabewert schreiben.

Schreibe eine Methode `public static boolean isEven(int number)`, die prüft, ob eine Zahl gerade ist (Rückgabe: `true` wenn gerade, `false` wenn ungerade). Rufe sie in `main` mit verschiedenen Werten auf.

**Beispielablauf:**
```
4 ist gerade: true
7 ist gerade: false
```

**Hinweis:** Mit dem Modulo-Operator `%` prüfst du, ob eine Zahl durch 2 teilbar ist.
