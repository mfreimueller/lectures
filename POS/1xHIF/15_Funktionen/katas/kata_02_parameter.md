# K2: Methode mit Parameter

**Ziel:** Eine Methode mit Parameter schreiben.

Schreibe eine Methode `public static void greet(String name)`, die "Hallo, [name]!" ausgibt. Rufe sie in `main` mit verschiedenen Namen auf.

**Beispielablauf:**
```
Hallo, Anna!
Hallo, Max!
```

**Hinweis:** Der Parameter `name` ist vom Typ `String`.
