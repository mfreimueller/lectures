# K3: Methode mit Rückgabewert

**Ziel:** Eine Methode mit Rückgabewert schreiben.

Schreibe eine Methode `public static int add(int a, int b)`, die zwei Zahlen addiert und das Ergebnis zurückgibt. Rufe sie in `main` auf und gib das Ergebnis aus.

**Beispielablauf:**
```
3 + 5 = 8
```

**Hinweis:** Der Rückgabetyp ist `int`, die Methode verwendet `return`.
