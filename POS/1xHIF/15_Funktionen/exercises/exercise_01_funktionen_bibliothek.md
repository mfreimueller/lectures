# Exercise: Funktionen-Bibliothek

**Ziel:** Erstelle eine Sammlung von nützlichen Methoden.

## Anforderungen

Definiere folgende Methoden in einer Klasse `FunktionenBibliothek`:

1. `public static double celsiusToFahrenheit(double c)` — rechnet Celsius in Fahrenheit um
2. `public static double fahrenheitToCelsius(double f)` — rechnet Fahrenheit in Celsius um
3. `public static int max(int a, int b, int c)` — gibt die größte von drei Zahlen zurück
4. `public static boolean isLeapYear(int year)` — prüft, ob ein Jahr ein Schaltjahr ist
5. `public static long factorial(int n)` — berechnet die Fakultät von n (n! = n * (n-1) * ... * 1)

Rufe alle Methoden in `main` mit verschiedenen Werten auf und gib die Ergebnisse aus.

## Beispielausgabe

```
30°C = 86.0°F
86°F = 30.0°C
Max von (3, 7, 5) = 7
2024 ist ein Schaltjahr: true
5! = 120
```
