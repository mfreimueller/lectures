# Große Übung: Taschenrechner mit Funktionen

**Ziel:** Baue einen Taschenrechner, der komplett aus Funktionen besteht.

## Anforderungen

Definiere folgende Methoden:

1. `public static double add(double a, double b)` — addiert zwei Zahlen
2. `public static double subtract(double a, double b)` — subtrahiert
3. `public static double multiply(double a, double b)` — multipliziert
4. `public static double divide(double a, double b)` — dividiert
5. `public static void printMenu()` — zeigt das Menü an
6. `public static void runCalculator()` — enthält die Hauptschleife

Die `main`-Methode ruft nur `runCalculator()` auf.

## Beispielablauf

```
=== Taschenrechner ===
1: Addieren
2: Subtrahieren
3: Multiplizieren
4: Dividieren
5: Beenden
Wähle eine Option: 1
Erste Zahl: 10
Zweite Zahl: 3
Ergebnis: 13.0
```
