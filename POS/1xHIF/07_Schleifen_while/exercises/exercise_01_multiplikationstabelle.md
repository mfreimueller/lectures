# Exercise: Multiplikationstabelle

Schreibe ein Programm, das für eine gegebene Zahl (z. B. 7) die Multiplikationstabelle von 1 bis 10 ausgibt. Verwende eine `while`-Schleife.

**Beispiel für die Zahl 7:**

```
7 x 1 = 7
7 x 2 = 14
7 x 3 = 21
7 x 4 = 28
7 x 5 = 35
7 x 6 = 42
7 x 7 = 49
7 x 8 = 56
7 x 9 = 63
7 x 10 = 70
```

### Hinweise

- Lege die Zahl als Variable an (z. B. `int zahl = 7`).
- Verwende eine Zählvariable von 1 bis 10.
- Berechne das Ergebnis mit `zahl * i`.
- Gib die Zeile formatiert aus.

### Erweiterung

- Frage den Benutzer, für welche Zahl die Tabelle ausgegeben werden soll.
- Frage den Benutzer, bis zu welchem Wert (statt fix 10) die Tabelle gehen soll.
