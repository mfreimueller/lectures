# K2: Summe berechnen

Berechne die Summe der Zahlen von 1 bis n (n ist eine festgelegte Zahl, z. B. 100). Gib das Ergebnis aus.

**Beispiel für n = 100:**

```
Summe 1..100 = 5050
```

### Hinweise

- Lege eine Variable `n` mit dem gewünschten Wert an.
- Verwende eine zweite Variable `summe`, die bei 0 startet.
- Addiere in der Schleife jede Zahl zur Summe.
