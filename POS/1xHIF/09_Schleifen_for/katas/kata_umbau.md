# K1: Umbau while → for

Gegeben ist eine `while`-Schleife, die 1 bis 10 zählt:

```java
int i = 1;
while (i <= 10) {
    System.out.println(i);
    i++;
}
```

Baue diese Schleife in eine `for`-Schleife um. Die Ausgabe soll identisch sein.

**Ausgabe:**

```
1
2
3
...
10
```

### Hinweise

- Die drei Teile der `for`-Schleife: Initialisierung, Bedingung, Inkrement.
- Setze `int i = 1` in die Initialisierung.
- Setze `i <= 10` in die Bedingung.
- Setze `i++` in das Inkrement.
