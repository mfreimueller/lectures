# K4: Potenzen

Berechne mit einer `for`-Schleife 2 hoch n für n = 1 bis 10. Gib "2^1 = 2", "2^2 = 4", ... aus.

**Beispielausgabe:**

```
2^1 = 2
2^2 = 4
2^3 = 8
2^4 = 16
2^5 = 32
2^6 = 64
2^7 = 128
2^8 = 256
2^9 = 512
2^10 = 1024
```

### Hinweise

- Starte mit einer Variable `ergebnis = 1` vor der Schleife.
- Multipliziere in jedem Durchlauf: `ergebnis = ergebnis * 2`.
- Gib die Potenzschreibweise formatiert aus.
