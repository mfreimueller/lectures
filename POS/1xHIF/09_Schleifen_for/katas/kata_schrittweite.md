# K3: Schrittweite

Gib mit einer `for`-Schleife alle geraden Zahlen von 0 bis 100 aus.

**Variante:** Gib zusätzlich alle Dreier-Schritte von 0 bis 30 aus.

**Ausgabe gerade Zahlen:**

```
0
2
4
...
100
```

**Ausgabe Dreier-Schritte:**

```
0
3
6
...
30
```

### Hinweise

- Verwende `i += 2` für gerade Zahlen.
- Verwende `i += 3` für Dreier-Schritte.
