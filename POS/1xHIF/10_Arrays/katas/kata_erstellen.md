# K1: Array erstellen und ausgeben

Erstelle ein `int`-Array mit 5 Zahlen (z. B. 10, 20, 30, 40, 50). Gib jedes Element mit einer `for`-Schleife aus.

**Beispielausgabe:**

```
Element 0: 10
Element 1: 20
Element 2: 30
Element 3: 40
Element 4: 50
```

### Hinweise

- Verwende die Kurzform mit geschweiften Klammern: `int[] zahlen = {10, 20, 30, 40, 50};`
- Verwende `zahlen.length` in der Schleifenbedingung.
- Greife mit `zahlen[i]` auf jedes Element zu.
