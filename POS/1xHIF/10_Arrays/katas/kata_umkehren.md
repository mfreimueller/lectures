# K4: Array umkehren

Erstelle ein Array mit den Werten 1, 2, 3, 4, 5 und gib es in umgekehrter Reihenfolge aus (ohne das Array zu verändern).

**Beispielausgabe:**

```
Original: 1 2 3 4 5
Umgekehrt: 5 4 3 2 1
```

### Hinweise

- Durchlaufe das Array rückwärts: von `length - 1` bis 0.
- Verwende `i--` in der Schleife.
- Das Array selbst bleibt unverändert — nur die Ausgabe ist umgekehrt.
