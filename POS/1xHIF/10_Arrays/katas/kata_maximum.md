# K3: Maximum suchen

Finde das größte Element in einem `int`-Array. Gib den Wert und die Position (Index) aus.

Verwende das Array: `{12, 7, 19, 4, 8, 15}`

**Beispielausgabe:**

```
Array: 12 7 19 4 8 15
Groesstes Element: 19 an Position 2
```

### Hinweise

- Starte mit der Annahme: Das erste Element ist das Maximum.
- Durchlaufe das Array und vergleiche jedes Element mit dem aktuellen Maximum.
- Merke dir sowohl den Wert als auch den Index.
