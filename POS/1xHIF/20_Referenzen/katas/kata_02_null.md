# K2: null und NullPointerException

Schreibe eine Klasse mit einer Methode `public void printName(Person p)`, die den Namen der Person ausgibt.

Weise einer Person-Variablen den Wert `null` zu und rufe `printName` mit dieser Variablen auf.

Was passiert?

Fange die NullPointerException mit try-catch ab und gib stattdessen "Keine Person vorhanden" aus.

Zusatz: Prüfe vor dem Methodenaufruf auf null (if-Bedingung).
