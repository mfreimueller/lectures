# K3: Objekt als Parameter

Schreibe eine Klasse `Person` mit Attribut `alter` (int) und einem Getter/Setter dafür.

Schreibe eine statische Methode `public static void altern(Person p)`, die das Alter der übergebenen Person um 1 erhöht.

Erzeuge im `main` eine Person mit Alter 20, rufe `altern` auf und gib das Alter danach aus.

Zeige, dass die Änderung außerhalb der Methode sichtbar ist. Ändere dann die Methode so, dass sie ein neues Person-Objekt erzeugt — die Änderung sollte dann nicht sichtbar sein.
