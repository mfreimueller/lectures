Strings in Java

Ein String ist eine Klasse (kein primitiver Datentyp).

Strings haben Methoden, die man aufrufen kann.

length() und equals()

String name = "Anna";
int laenge = name.length();      // 4

String pw = "geheim";
if (pw.equals("geheim")) {
    System.out.println("Passwort korrekt!");
}

// NICHT: if (pw == "geheim") - Fehler!
// == vergleicht Referenzen, nicht den Inhalt!

charAt()

String text = "Java";
char erster = text.charAt(0);   // 'J'
char zweiter = text.charAt(1);  // 'a'
char dritter = text.charAt(2);  // 'v'
char vierter = text.charAt(3);  // 'a'

// charAt(laenge - 1) = letztes Zeichen
char letzter = text.charAt(text.length() - 1);  // 'a'

startsWith() und endsWith()

String datei = "Program.java";

boolean istJava = datei.endsWith(".java");        // true
boolean istClass = datei.endsWith(".class");      // false
boolean startetMit = datei.startsWith("Prog");    // true

String url = "https://schule.at";
boolean istHttps = url.startsWith("https");       // true

Weitere nuetzliche Methoden

String s = "  Hallo Welt!  ";

s.toLowerCase();    // "  hallo welt!  "
s.toUpperCase();    // "  HALLO WELT!  "
s.trim();           // "Hallo Welt!" (entfernt Leerzeichen aussen)
s.contains("Welt"); // true
s.indexOf("Welt");  // 7 (Position)
s.substring(7);     // "Welt!"
s.substring(2, 7);  // "llo W"

Kata 1: Woerter zaehlen

String satz = "Heute ist ein schoener Tag";
int woerter = 1;

for (int i = 0; i < satz.length(); i++) {
    if (satz.charAt(i) == ' ') {
        woerter++;
    }
}

System.out.println("Woerter: " + woerter);  // 5

Kata 3: Caesar-Verschluesselung

String text = "HALLO";
int verschiebung = 3;
String verschluesselt = "";

for (int i = 0; i < text.length(); i++) {
    char c = text.charAt(i);
    c = (char) (c + verschiebung);
    verschluesselt += c;
}

System.out.println(verschluesselt);  // KDOOR

Dungeon Escape: Befehlsparsing

String befehl = "norden";

if (befehl.equals("norden")) {
    System.out.println("Du gehst nach Norden.");
} else if (befehl.equals("angreifen")) {
    System.out.println("Du greifst an!");
} else if (befehl.startsWith("gehe")) {
    System.out.println("Bewegung...");
} else {
    System.out.println("Unbekannter Befehl.");
}
