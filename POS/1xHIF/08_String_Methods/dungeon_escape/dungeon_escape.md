# Dungeon Escape - Woche 08: Befehlsparsing

**Konzepte:** String-Methoden (equals, startsWith)

### Aufgabe
Der Spieler gibt Textbefehle ein (Strings). Werte diese aus:
- "norden" / "sueden" / "osten" / "westen" -> Bewegung
- "angreifen" -> Kampf
- "fliehen" -> Flucht
- "status" -> HP anzeigen
- "hilfe" -> Befehlsliste anzeigen

Nutze equals() fuer genaue Uebereinstimmung.
