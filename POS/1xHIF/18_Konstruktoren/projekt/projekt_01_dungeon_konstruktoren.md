# Jahresprojekt: Dungeon Crawler — Konstruktoren

Baue Konstruktoren in den Dungeon Crawler ein.

## Aufgaben

1. **Player-Konstruktor:** `public Player(String name, int maxHp, int attack)` — setzt hp automatisch auf maxHp.
2. **Enemy-Klasse:** Erstelle eine Klasse `Enemy` mit Attributen `name`, `hp`, `attack`, `xpReward` und einem Konstruktor `Enemy(String name, int hp, int attack, int xpReward)`.
3. **Objekterzeugung:** Erzeuge Gegner mit unterschiedlichen Werten und kämpfe gegen sie.
