# K1: Default-Konstruktor

**Ziel:** Den Default-Konstruktor verwenden.

Schreibe eine Klasse `Book` mit den Attributen `title` (String) und `pages` (int). Verwende den Default-Konstruktor (den ohne Parameter) und setze die Werte danach über Setter-Methoden. Gib die Daten mit einer Methode `void printInfo()` aus.

**Beispielablauf:**
```
Titel: Java für Anfänger
Seiten: 350
```

**Hinweis:** Den Default-Konstruktor muss man nicht schreiben — Java erzeugt ihn automatisch, solange kein anderer Konstruktor definiert ist.
