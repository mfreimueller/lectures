# K3: Konstruktor-Überladung

**Ziel:** Mehrere Konstruktoren in einer Klasse.

Füge der `Book`-Klasse einen zweiten Konstruktor `Book(String title)` hinzu, der `pages` auf einen Standardwert (z. B. 0) setzt. Zeige beide Varianten in `main`.

**Beispielablauf:**
```
Titel: Java für Anfänger
Seiten: 350

Titel: Kurzgeschichte
Seiten: 0
```

**Hinweis:** Konstruktor-Überladung bedeutet: mehrere Konstruktoren mit unterschiedlichen Parameterlisten.
