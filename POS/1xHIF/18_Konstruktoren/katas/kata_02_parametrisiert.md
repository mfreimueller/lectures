# K2: Parametrisierter Konstruktor

**Ziel:** Einen Konstruktor mit Parametern schreiben.

Erweitere die `Book`-Klasse um einen Konstruktor `Book(String title, int pages)`, der beide Attribute direkt setzt. Erzeuge Bücher direkt mit Werten.

**Beispielablauf:**
```
Titel: Java für Anfänger
Seiten: 350

Titel: Clean Code
Seiten: 464
```

**Hinweis:** Der Konstruktor hat keinen Rückgabetyp (auch nicht `void`) und heißt genauso wie die Klasse.
