# Exercise: Student-Klasse mit Konstruktoren

**Ziel:** Eine Klasse `Student` mit mehreren Konstruktoren.

## Anforderungen

Schreibe eine Klasse `Student` mit den privaten Attributen `name` (String), `matrikelnummer` (String) und `note` (double).

Definiere:
1. Einen Konstruktor, der alle drei Attribute setzt: `Student(String name, String matrikelnummer, double note)`
2. Einen Konstruktor, der nur `name` und `matrikelnummer` setzt (note = 0): `Student(String name, String matrikelnummer)`
3. Eine Methode `void zeigeDaten()`, die alle Daten ausgibt

Erzeuge mehrere Studenten mit verschiedenen Konstruktoren und gib ihre Daten aus.

## Beispielablauf

```
Name: Anna Schmidt
Matrikelnummer: 2024-001
Note: 1.0

Name: Max Mustermann
Matrikelnummer: 2024-002
Note: 0.0
```
