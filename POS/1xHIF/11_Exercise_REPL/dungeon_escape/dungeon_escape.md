# Dungeon Escape - Woche 11: REPL-Architektur

**Konzepte:** while(true), Scanner, switch

### Aufgabe
Baue das gesamte Dungeon Escape als REPL um:
- while(true) { eingabe lesen -> ausfuehren -> status ausgeben }
- "kaempfen", "fliehen", "status", "inventar", "hilfe", "exit"
- Nach jeder Aktion: aktuellen Raum und HP ausgeben
- Die Schleife laeuft bis "exit" oder hp <= 0
