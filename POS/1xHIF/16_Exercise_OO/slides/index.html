Wiederholung: Klassen und Objekte

public class Auto {
    String marke;     // Attribut

    void fahre() {    // Methode
        System.out.println(marke + " faehrt!");
    }
}

// Objekt erzeugen und verwenden
Auto a = new Auto();
a.marke = "VW";
a.fahre();
  • Klasse = Bauplan, Objekt = Instanz
  • new erzeugt ein Objekt
  • Punkt-Operator (.) fuer Zugriff

Methoden mit Parametern

public class Bankkonto {
    double kontostand;

    void einzahlen(double betrag) {
        kontostand = kontostand + betrag;
    }

    void auszahlen(double betrag) {
        kontostand = kontostand - betrag;
    }
}
  • Parameter werden beim Aufruf uebergeben
  • Die Methode arbeitet mit dem Parameter

Methoden mit Rueckgabewert

public class Wuerfel {
    int seiten;

    int wuerfeln() {
        Random rand = new Random();
        return rand.nextInt(seiten) + 1;
    }
}

// Verwendung:
Wuerfel w = new Wuerfel();
w.seiten = 6;
int ergebnis = w.wuerfeln();
  • return gibt einen Wert zurueck
  • Der Rueckgabetyp steht vor dem Methodennamen
