06 - while und do-while

POS1 - 1AHIF

Wiederholung: for-Schleife

for-Schleife: bekanntes Durchlaufverhalten

for (int i = 0; i < 10; i++) { ... }

Frage: Wann weiss man vorher, wie oft die Schleife laufen muss?

while-Schleife

while (bedingung) {
    // wiederholter Code
}

Die Bedingung wird vor jedem Durchlauf geprueft.

Wird die Bedingung nie false -> Endlosschleife!

int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;  // Wichtig! Sonst Endlosschleife
}

while: Ziffernsumme

int zahl = 1234;
int summe = 0;

while (zahl > 0) {
    summe += zahl % 10;  // letzte Ziffer
    zahl /= 10;           // letzte Ziffer entfernen
}

System.out.println("Ziffernsumme: " + summe);

Durchlauf: 1234 -> 4, 123 -> 3, 12 -> 2, 1 -> 1, 0 -> Ende

do-while-Schleife

do {
    // wird mindestens einmal ausgefuehrt
} while (bedingung);

Der Code wird AUSGEFUEHRT, DANN die Bedingung geprueft.

int eingabe;
do {
    System.out.println("Bitte Zahl > 0 eingeben:");
    eingabe = scanner.nextInt();
} while (eingabe <= 0);

for vs. while vs. do-while

SchleifeWann verwenden?
forBekannte Anzahl Durchlaeufe
whileBedingung vor dem ersten Durchlauf
do-whileMindestens ein Durchlauf

Endlosschleifen

// Endlosschleife (Achtung!)
while (true) {
    System.out.println("Forever...");
}

Endlosschleifen sind nicht immer falsch!

Manche Programme laufen absichtlich "forever" (z.B. Betriebssysteme).

Wichtig: break als Ausstieg.

break in Schleifen

while (true) {
    int eingabe = scanner.nextInt();

    if (eingabe == 0) {
        break;  // Schleife beenden
    }

    System.out.println("Eingegeben: " + eingabe);
}

Dungeon Escape: Hauptschleife

int hp = 100;
boolean entkommen = false;

while (hp > 0 && !entkommen) {
    System.out.println("1: Kaempfen  2: Fliehen  3: Status");
    int wahl = scanner.nextInt();

    if (wahl == 1) {
        hp -= 10;
    } else if (wahl == 2) {
        entkommen = true;
    } else {
        System.out.println("HP: " + hp);
    }
}

Dungeon Escape - Aufgabe

  • Implementiere die Hauptspielschleife mit while
  • Verwende do-while fuer das Menue
  • Spielende bei hp <= 0 oder entkommen = true

Was wir heute gelernt haben

  • while-Schleife: Bedingung vor dem Durchlauf
  • do-while-Schleife: mindestens ein Durchlauf
  • Unterschiede zu for
  • Endlosschleifen vermeiden/steuern
  • break zum Abbruch
