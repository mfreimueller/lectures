# Dungeon Escape - Woche 06: Hauptspielschleife

**Konzepte:** while, do-while

### Aufgabe
Implementiere die Hauptspielschleife mit while(hp > 0 && !entkommen).
Ein Menue (do-while): 1 = kaempfen (hp - 15), 2 = fliehen (entkommen = true), 3 = status (hp anzeigen).
Gib bei jedem Durchlauf den Raum-Name aus.
