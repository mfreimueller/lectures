# K4: Menü

Zeige ein Menü in einer `do-while`-Schleife an. Reagiere auf die Eingabe mit `if-else`.

**Menü:**

```
=== MENU ===
1: Option A
2: Option B
3: Option C
0: Beenden
Auswahl:
```

**Beispielablauf (Benutzereingabe fett):**

```
Auswahl: 1
Option A ausgewählt
Auswahl: 3
Option C ausgewählt
Auswahl: 0
Programm beendet
```

### Hinweise

- Die Schleife läuft, solange die Auswahl nicht 0 ist.
- Verwende `if-else` oder `switch`, um die Auswahl zu verarbeiten.
