# K2: Attribut-Zugriff

**Ziel:** Methoden mit Rückgabewert in einer Klasse.

Füge zur `Person`-Klasse eine Methode `boolean istVolljährig()` hinzu, die prüft, ob das Alter >= 18 ist. Verwende die Methode in `main`, um eine entsprechende Meldung auszugeben.

**Beispielablauf:**
```
Anna ist volljährig: true
```
