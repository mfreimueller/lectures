Was ist OOP?

OOP = Objektorientierte Programmierung

  • Ein Programm besteht aus Objekten, die miteinander kommunizieren
  • Jedes Objekt hat Eigenschaften (Attribute) und Fähigkeiten (Methoden)
  • Analogie: LEGO-Steine - viele kleine Bausteine ergeben ein grosses Ganzes
  • Java ist eine objektorientierte Programmiersprache

Klasse vs. Objekt

Klasse = Bauplan

Objekt = Haus (gebaut nach dem Bauplan)

  • Eine Klasse definiert den Aufbau (Attribute und Methoden)
  • Mit new erzeugen wir konkrete Objekte (Instanzen)
  • Von einer Klasse können viele Objekte erzeugt werden
// Klasse = Bauplan
// Auto meinAuto = new Auto();  // Objekt = Haus

Klasse Auto - Attribute

public class Auto {
    String marke;
    int baujahr;
    String farbe;
}
  • Attribute sind Eigenschaften eines Objekts
  • Sie werden als Variablen innerhalb der Klasse deklariert
  • Jedes Auto-Objekt hat seine eigenen Werte für marke, baujahr, farbe

Klasse Auto - Methoden

public class Auto {
    String marke;
    int baujahr;

    void fahre() {
        System.out.println(marke + " fährt los!");
    }

    void hupen() {
        System.out.println("Hup Hup!");
    }
}
  • Methoden ohne static gehören zu einem Objekt
  • Sie können auf die Attribute des Objekts zugreifen

Attribut vs. Parameter vs. Variable

public class Auto {
    String marke;             // ATTRIBUT (gehört zum Objekt)

    void fahre(int tempo) {   // PARAMETER (wird beim Aufruf übergeben)
        String msg = "Start"; // LOKALE VARIABLE (nur in dieser Methode)
        System.out.println(marke + " fährt mit " + tempo + " km/h");
    }
}
  • Attribute: gelten für das ganze Objekt
  • Parameter: werden beim Methodenaufruf übergeben
  • Lokale Variablen: existieren nur innerhalb einer Methode

Objekte erzeugen mit new

// Objekt erzeugen
Auto meinAuto = new Auto();

// Attribute setzen
meinAuto.marke = "VW";
meinAuto.baujahr = 2020;

// Methoden aufrufen
meinAuto.fahre();
  • new Auto() erzeugt ein neues Auto-Objekt im Speicher
  • Der Datentyp der Variable ist Auto (der Klassenname)
  • Mit dem Punkt-Operator (.) greifen wir auf Attribute/Methoden zu

Live-Coding: Auto Klasse + Objekt

public class Auto {
    String marke;
    int baujahr;

    void fahre() {
        System.out.println(marke + " fährt los!");
    }
}
public class AutoTest {
    public static void main(String[] args) {
        Auto a1 = new Auto();
        a1.marke = "VW";
        a1.baujahr = 2020;

        Auto a2 = new Auto();
        a2.marke = "BMW";
        a2.baujahr = 2023;

        a1.fahre(); // VW fährt los!
        a2.fahre(); // BMW fährt los!
    }
}

this - Vorgriff

public class Auto {
    String marke;

    void setMarke(String marke) {
        this.marke = marke; // this.marke = Attribut
    }                       // marke = Parameter
}

Zusammenfassung

  • Klasse = Bauplan für Objekte
  • Objekt = konkrete Instanz einer Klasse (erzeugt mit new)
  • Attribute = Eigenschaften des Objekts
  • Methoden = Fähigkeiten des Objekts (ohne static)
  • Zugriff auf Attribute/Methoden mit dem Punkt-Operator (.)
