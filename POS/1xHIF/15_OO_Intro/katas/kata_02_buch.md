# Kata 2: Klasse Buch

| Feld | Wert |
|------|------|
| Konzepte | Attribute, Methoden mit Ausgabe |
| Schwierigkeit | 2/5 |
| Dauer | ca. 15 min |

### Aufgabenstellung

Erstelle eine Klasse `Buch` in einer eigenen Datei `Buch.java`:

1. Die Klasse hat die Attribute `titel` (String), `autor` (String) und `seiten` (int)
2. Die Methode `printInfo()` gibt alle Informationen formatiert aus

Erstelle eine Datei `BuchTest.java` mit einer main-Methode. Erzeuge zwei Buch-Objekte mit unterschiedlichen Werten und rufe printInfo() fuer beide auf.

### Beispiel-Output

```
Titel: Der Hobbit
Autor: J.R.R. Tolkien
Seiten: 310

Titel: 1984
Autor: George Orwell
Seiten: 328
```
