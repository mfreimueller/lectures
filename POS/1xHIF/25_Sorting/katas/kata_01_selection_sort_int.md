# Kata 01: Selection Sort (int[])

Schreibe eine Methode `void selectionSort(int[] arr)`, die ein int-Array mit Selection Sort sortiert.

Beispiel:

```java
int[] zahlen = {5, 2, 8, 1, 9};
selectionSort(zahlen);
System.out.println(java.util.Arrays.toString(zahlen));
// [1, 2, 5, 8, 9]
```

Hinweise:
- Verwende zwei verschachtelte for-Schleifen
- Suche das Minimum ab Position i
- Tausche das Minimum an Position i
