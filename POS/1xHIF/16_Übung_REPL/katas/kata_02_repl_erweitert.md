# K2: Zähl-REPL

**Ziel:** Eine REPL, die Befehle zählt.

Erweitere die Echo-REPL: Zähle mit, wie viele Befehle eingegeben wurden. Gib nach jeder Eingabe die Befehlsnummer aus.

**Beispielablauf:**
```
[1] > Hallo
Hallo
[2] > Test
Test
[3] > exit
Programm beendet. 3 Befehle verarbeitet.
```

**Hinweis:** Führe eine Zählvariable ein, die bei jeder Eingabe erhöht wird.
