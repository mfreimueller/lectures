# K1: Echo-REPL

**Ziel:** Eine einfache REPL schreiben, die Eingaben wiederholt.

Schreibe eine REPL (Read-Eval-Print-Loop), die jede Eingabe des Benutzers wiederholt. Bei der Eingabe "exit" bricht sie ab.

**Beispielablauf:**
```
> Hallo
Hallo
> Wie geht's?
Wie geht's?
> exit
Programm beendet.
```

**Hinweis:** Verwende eine `while`-Schleife mit einer boolean-Variable `running`.
