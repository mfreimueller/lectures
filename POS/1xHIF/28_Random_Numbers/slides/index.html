Math.random()

Liefert eine double-Zufallszahl zwischen 0.0 (inklusive) und 1.0 (exklusive).

double zufall = Math.random();
System.out.println(zufall); // z.B. 0.4738291

Ergebnis ist immer im Intervall [0.0, 1.0)

Umrechnung auf andere Bereiche

Ganze Zahl von 0 bis max-1:

int zahl = (int)(Math.random() * max);

Beispiel: 0 bis 9

int zahl = (int)(Math.random() * 10); // 0-9

Bereich mit Verschiebung

Ganze Zahl von min bis max (inklusive):

int zahl = (int)(Math.random() * (max - min + 1)) + min;

Beispiel: 10 bis 20

int zahl = (int)(Math.random() * 11) + 10; // 10-20

Live-Coding: Wuerfel 1-6

int wuerfel = (int)(Math.random() * 6) + 1;
System.out.println("Gewuerfelt: " + wuerfel);

Math.random() * 6 liefert 0.0 bis 5.999...,
(int) schneidet Nachkommastellen ab -> 0 bis 5,
+ 1 verschiebt auf 1 bis 6.

Random-Klasse

Import aus java.util.Random:

import java.util.Random;

Random rand = new Random();

Random-Methoden

  • rand.nextInt() - ganze Zahl (beliebiger int-Wert)
  • rand.nextInt(n) - Zahl von 0 bis n-1
  • rand.nextDouble() - double 0.0 bis 1.0
  • rand.nextBoolean() - true oder false

Random nextInt(n)

Liefert eine Zahl von 0 bis n-1:

Random rand = new Random();
int zahl = rand.nextInt(10); // 0-9

Achtung: rand.nextInt(6) liefert 0-5, nicht 1-6!

Live-Coding: Wuerfel mit Random

Random rand = new Random();
int wuerfel = rand.nextInt(6) + 1; // 1-6
System.out.println("Gewuerfelt: " + wuerfel);

nextInt(6) -> 0 bis 5
+ 1 -> 1 bis 6

Seed bei Random

Deterministische Zufallszahlen zum Testen:

Random rand = new Random(12345);
// Bei gleichem Seed immer die gleichen Zahlen
System.out.println(rand.nextInt(100));
System.out.println(rand.nextInt(100));
