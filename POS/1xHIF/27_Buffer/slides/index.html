27 - Puffer: Arrays und Sortieren

POS - 1AHIF

Wiederholung: Arrays

  • Deklaration: int[] zahlen = new int[5];
  • Initialisierung: int[] zahlen = {1, 2, 3};
  • Zugriff über Index: zahlen[index]
  • for-each: for (int z : zahlen) { }
  • Arrays.toString() für schnelle Ausgabe
  • Mehrdimensionale Arrays: int[][] matrix

Wiederholung: Sortieren

  • Selection Sort: Minimum suchen, an den Anfang tauschen
  • Insertion Sort: Element einordnen, nach rechts verschieben
  • Bubblesort: Benachbarte Elemente vergleichen und tauschen
  • Alle drei Algorithmen: O(n^2)

Arbeitsauftrag

  • Kata 1: Notenverwaltung - Durchschnitt, Min, Max, Sortieren
  • Kata 2: Telefonbuch - Namen suchen und Nummer ausgeben
  • Dungeon Escape: Highscore verbessern (absteigend sortieren)
