Wiederholung: Sortieralgorithmen

  • Selection Sort: Minimum suchen, an den Anfang tauschen
  • Insertion Sort: Element einordnen, grössere nach rechts verschieben
  • Beide: O(n^2) im Durchschnitt

Bubblesort Algorithmus

void bubbleSort(int[] arr) {
    for (int i = 0; i < arr.length - 1; i++) {
        for (int j = 0; j < arr.length - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}

Idee: Grössere Elemente steigen auf wie Blasen im Wasser
