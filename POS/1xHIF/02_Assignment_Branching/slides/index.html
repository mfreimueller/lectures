Zuweisung

int x;        // Deklaration
x = 5;        // Zuweisung: Variable = Wert
int y = 10;   // Deklaration + Initialisierung

x = x + 3;    // x ist jetzt 8
x = y;        // x ist jetzt 10 (Kopieren)
  • Rechts vom = steht der Wert/Ausdruck
  • Links vom = steht die Variable, die den Wert erhaelt
  • Der alte Wert wird ueberschrieben

Variable erhoehen

int zaehler = 0;

zaehler = zaehler + 1;   // zaehler ist 1
zaehler = zaehler + 1;   // zaehler ist 2

Dieses Muster kommt sehr haeufig vor!

Kurzformen:

zaehler += 1;    // gleiche Wirkung wie zaehler = zaehler + 1
zaehler++;       // noch kuerzer!

Zusammengesetzte Zuweisungen

int x = 10;

x += 5;    // x = x + 5  -> 15
x -= 3;    // x = x - 3  -> 12
x *= 2;    // x = x * 2  -> 24
x /= 4;    // x = x / 4  -> 6
x %= 3;    // x = x % 3  -> 0

Gilt auch fuer double:

double preis = 10.0;
preis *= 1.2;    // +20% -> 12.0

Vergleichsoperatoren

OperatorBedeutung
==gleich
!=ungleich
<kleiner
>groesser
<=kleiner oder gleich
>=groesser oder gleich

Ergebnis ist ein boolean

int alter = 17;

boolean volljaehrig = alter >= 18;   // false
boolean ist17 = alter == 17;         // true
boolean ungleich = alter != 18;      // true

Ein Vergleich ergibt immer true oder false.

Dieses Ergebnis kann man in einer boolean-Variable speichern oder direkt in if verwenden.

Achtung! = vs. ==

= ist Zuweisung, == ist Vergleich

int x = 5;         // Zuweisung (korrekt)

// if (x = 5) {   // Fehler! = ist keine Bedingung!
// }

if (x == 5) {      // Vergleich (korrekt)
    System.out.println("x ist 5");
}

Der Compiler meldet einen Fehler bei `if (x = 5)` - das ist gut so!

if-Anweisung

if (bedingung) {
    // Code der ausgefuehrt wird, wenn bedingung true ist
}

Die Bedingung ist ein boolean-Ausdruck.

int temperatur = 35;

if (temperatur > 30) {
    System.out.println("Es ist heiss!");
}

if-else

if (bedingung) {
    // Code fuer true
} else {
    // Code fuer false
}
int alter = 16;

if (alter >= 18) {
    System.out.println("Volljaehrig");
} else {
    System.out.println("Minderjaehrig");
}

Beispiel: Temperatur

int temperatur = 35;

if (temperatur > 30) {
    System.out.println("Es ist heiss - schwimmen gehen!");
} else {
    System.out.println("Angenehme Temperatur.");
}

Was passiert bei temperatur = 25?

Die else-Zweig wird ausgefuehrt.

Mehrere Befehle in if/else

if (x > 0) {
    y = 1;
    z = 2;
    System.out.println("x ist positiv");
} else {
    y = -1;
    z = -2;
    System.out.println("x ist negativ oder null");
}

Die geschweiften Klammern { } fassen mehrere Befehle zusammen.

Wichtig: Auch bei einem Befehl immer { } verwenden!

else if - Mehrere Bedingungen

if (bedingung1) {
    // Wenn bedingung1 true
} else if (bedingung2) {
    // Wenn bedingung1 false UND bedingung2 true
} else {
    // Wenn beide false
}

Es koennen beliebig viele else-if-Zweige sein.

Der erste true-Zweig wird ausgefuehrt, der Rest uebersprungen.

Beispiel: Notenspiegel

int punkte = 85;

if (punkte >= 90) {
    System.out.println("Note 1");
} else if (punkte >= 78) {
    System.out.println("Note 2");
} else if (punkte >= 65) {
    System.out.println("Note 3");
} else if (punkte >= 51) {
    System.out.println("Note 4");
} else {
    System.out.println("Note 5");
}

Live-Coding: Schaltjahr

int jahr = 2024;
boolean istSchaltjahr;

istSchaltjahr = (jahr % 400 == 0) ||
                (jahr % 4 == 0 && jahr % 100 != 0);

if (istSchaltjahr) {
    System.out.println(jahr + " ist ein Schaltjahr.");
} else {
    System.out.println(jahr + " ist kein Schaltjahr.");
}

Ausgabe: 2024 ist ein Schaltjahr.
