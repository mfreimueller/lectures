# Dungeon Escape - Woche 07: Methoden-Struktur

**Konzepte:** Methoden, Scanner

### Aufgabe
Lagere diese Funktionen in Methoden aus:
- zeigeStatus(String name, int hp) -> gibt Namen und HP aus
- zeigeRaum(String raum) -> gibt Raum-Beschreibung aus
- int kaempfe(int hp, int schaden) -> zieht Schaden ab, gibt neue HP zurueck
- warteAufEingabe() -> Scanner, gibt int zurueck

Das Hauptprogramm ruft nur noch Methoden auf.
