# K2: String to Double

**Ziel:** `Double.parseDouble` kennenlernen.

Der Benutzer gibt seine Körpergröße in Metern ein (z. B. "1,85"). Wandle den Text mit `Double.parseDouble` in eine Gleitkommazahl um und gib die Größe in Zentimetern aus.

**Beispielablauf:**
```
Gib deine Größe in Metern ein: 1,85
Deine Größe in Zentimetern: 185.0 cm
```

**Hinweis:** Achte auf das deutsche Dezimaltrennzeichen (Komma). Java erwartet einen Punkt. Ersetze das Komma vor dem Parsen mit `.` (mit `replace(',', '.')`).
