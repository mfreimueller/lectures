# K1: String to Int

**Ziel:** `Integer.parseInt` kennenlernen.

Der Benutzer gibt sein Alter als Text ein. Wandle den Text mit `Integer.parseInt` in eine Ganzzahl um und gib aus, ob er volljährig ist (18 oder älter).

**Beispielablauf:**
```
Gib dein Alter ein: 20
Du bist volljährig.
```

```
Gib dein Alter ein: 15
Du bist nicht volljährig.
```

**Hinweis:** Verwende `Scanner.nextLine()` für die Eingabe und `Integer.parseInt()` für die Umwandlung.
