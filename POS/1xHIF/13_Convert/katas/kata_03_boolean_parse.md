# K3: Boolean.parseBoolean

**Ziel:** `Boolean.parseBoolean` kennenlernen.

Der Benutzer gibt "true" oder "false" ein. Wandle den Text mit `Boolean.parseBoolean` um. Gib eine entsprechende Meldung aus — z. B. "Du hast true eingegeben" oder "Du hast false eingegeben".

**Beispielablauf:**
```
Gib true oder false ein: true
Du hast true eingegeben.
```

```
Gib true oder false ein: false
Du hast false eingegeben.
```

**Hinweis:** `Boolean.parseBoolean` gibt bei allen anderen Eingaben (außer "true") `false` zurück. Das ist ein bekanntes Verhalten.
