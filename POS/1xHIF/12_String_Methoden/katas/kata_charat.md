# K2: Zeichen durchlaufen

Gib jedes Zeichen eines Strings mit einer `for`-Schleife einzeln aus. Verwende `charAt`.

Verwende den String: `"Java"`

**Beispielausgabe:**

```
Zeichen 0: J
Zeichen 1: a
Zeichen 2: v
Zeichen 3: a
```

### Hinweise

- Die Schleife läuft von 0 bis `text.length() - 1`.
- Mit `text.charAt(i)` bekommst du das Zeichen an Position i.
