# K4: startsWith / endsWith

Prüfe, ob ein eingegebener Satz mit "Hallo" beginnt und mit einem Punkt endet.

**Beispielablauf (Benutzereingabe fett):**

```
Gib einen Satz ein: Hallo Welt.
Beginnt mit Hallo: true
Endet mit Punkt: true

Gib einen Satz ein: Wie geht es?
Beginnt mit Hallo: false
Endet mit Punkt: false
```

### Hinweise

- Verwende `startsWith("Hallo")` und `endsWith(".")`.
- Beide Methoden geben `boolean` zurück.
