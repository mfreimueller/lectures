Frage: Was macht System.out.print() anders als println()?

Compiler vs. Interpreter

  • Compiler uebersetzt den gesamten Quellcode auf einmal in Maschinensprache
  • Interpreter fuehrt den Quellcode Zeile fuer Zeile aus
  • Java verwendet beides: Compiler + Interpreter (JVM)

Compiler: javac HalloWelt.java -> HalloWelt.class

Interpreter: java HalloWelt -> fuehrt Bytecode aus

JVM - Java Virtual Machine

  • Java-Code wird in Bytecode uebersetzt (.class-Datei)
  • Die JVM fuehrt den Bytecode aus
  • Vorteil: Plattformunabhaengigkeit
    • Einmal schreiben, ueberall ausfuehren
    • Windows, Mac, Linux - die JVM macht den Unterschied

Der Java-Prozess

// 1. Quelltext schreiben (.java)
public class HalloWelt {
    public static void main(String[] args) {
        System.out.println("Hallo!");
    }
}

// 2. Compilieren: javac HalloWelt.java -> HalloWelt.class
// 3. Ausfuehren:  java HalloWelt -> "Hallo!"

In IntelliJ passieren Schritt 2 und 3 automatisch beim Klick auf Run.

Was sind Variablen?

Eine Variable ist ein named Platzhalter im Speicher fuer einen Wert.

  • Jede Variable hat einen Typ und einen Namen
  • Deklaration: int alter;
  • Initialisierung: alter = 17;
  • Beides zusammen: int alter = 17;

Datentyp: int

int = ganze Zahlen (ohne Nachkommastellen)

int alter = 17;
int jahr = 2026;
int anzahlSchueler = 28;
int minus = -5;
  • Wertebereich: ca. -2,1 Mrd. bis +2,1 Mrd.
  • Typisch fuer Zaehlungen, Indizes, Alter, etc.

Datentyp: double

double = Gleitkommazahlen (mit Nachkommastellen)

double preis = 19.99;
double pi = 3.14159;
double temperatur = -5.5;
double prozent = 0.15;
  • Genauer als float (double = doppelte Genauigkeit)
  • Punkt (.) als Dezimaltrenner, NICHT Komma (,)

Datentypen: boolean und char

boolean

boolean istFertig = true;
boolean istGeburtstag = false;
  • Nur zwei Werte: true oder false
  • Wichtig fuer Bedingungen (naechste Woche)

char

char buchstabe = 'A';
char ziffer = '7';
  • Ein einzelnes Zeichen in Hochkomma (''), nicht in Anfuerhrungszeichen ("")

Arithmetische Operatoren

int a = 10;
int b = 3;

int summe     = a + b;    // 13
int differenz = a - b;    // 7
int produkt   = a * b;    // 30

Punkt-vor-Strich-Regel gilt auch in Java!

Ganzzahldivision

int a = 7;
int b = 2;

int ergebnis = a / b;    // 3, NICHT 3.5!
// int / int = int (Nachkommastellen werden abgeschnitten)

Wichtig: Wenn beide Operanden int sind, ist das Ergebnis int!

double ergebnisExakt = 7.0 / 2;    // 3.5
double ergebnisExakt2 = (double) 7 / 2;  // 3.5 (Cast)

Modulo-Operator %

Der %-Operator gibt den Rest einer Division zurueck.

int rest = 7 % 2;    // 1 (7 / 2 = 3 Rest 1)
int rest2 = 10 % 3;  // 1 (10 / 3 = 3 Rest 1)
int rest3 = 12 % 4;  // 0 (12 / 4 = 3 Rest 0)

Anwendung: Ist eine Zahl gerade? zahl % 2 == 0

Komplexe Ausdruecke

int ergebnis = 5 + 3 * 2;        // 11 (Punkt vor Strich)
int ergebnis2 = (5 + 3) * 2;     // 16 (Klammern zuerst)

double mittelwert = (10.0 + 20.0 + 30.0) / 3;  // 20.0
// Achtung: (10 + 20 + 30) / 3 waere 20 (int!), aber 10+20+30 / 3 waere...?
  • Klammern immer verwenden, wenn es unklar ist
  • Bei Division lieber double verwenden

Live-Coding: Celsius -> Fahrenheit

public class CelsiusToFahrenheit {
    public static void main(String[] args) {
        double celsius = 25.0;
        double fahrenheit = celsius * 9.0 / 5.0 + 32.0;

        System.out.printf("%.1f°C = %.1f°F\n", celsius, fahrenheit);
    }
}

Ausgabe: 25,0°C = 77,0°F

Live-Coding: Rechteck

public class Rechteck {
    public static void main(String[] args) {
        int laenge = 8;
        int breite = 5;

        int flaeche = laenge * breite;
        int umfang = 2 * (laenge + breite);

        System.out.println("Laenge: " + laenge);
        System.out.println("Breite: " + breite);
        System.out.println("Flaeche: " + flaeche);
        System.out.println("Umfang: " + umfang);
    }
}

Live-Coding: Brutto-Netto-Rechner

public class BruttoNetto {
    public static void main(String[] args) {
        double brutto = 120.0;
        double mwst = 0.2;  // 20%

        double netto = brutto / (1 + mwst);
        double steuer = brutto - netto;

        System.out.printf("Brutto:  %.2f\n", brutto);
        System.out.printf("Netto:   %.2f\n", netto);
        System.out.printf("MWST:    %.2f\n", steuer);
    }
}
