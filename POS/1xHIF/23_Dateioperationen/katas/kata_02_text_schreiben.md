# K2: Text in Datei schreiben

Schreibe ein Programm, das den Benutzer auffordert, Zeilen einzugeben. Jede Zeile wird mit `PrintWriter` in eine Datei `ausgabe.txt` geschrieben.

Das Programm endet, wenn der Benutzer "exit" eingibt.

Verwende try-with-resources.

Beispiel-Ablauf:
```
Gib Text ein (exit zum Beenden):
Hallo Welt
Java ist toll
exit
```

Die Datei `ausgabe.txt` sollte dann zwei Zeilen enthalten.
