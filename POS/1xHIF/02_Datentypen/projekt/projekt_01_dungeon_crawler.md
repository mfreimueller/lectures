# Jahresprojekt — Dungeon Crawler: Spieler-Eigenschaften

Der Spieler-Charakter bekommt erste Eigenschaften. Lege Variablen für die grundlegenden Attribute des Helden an und gib sie aus.

### Aufgaben

1. Lege eine Variable für den Spielernamen (`String`) an.
2. Lege eine Variable für die Lebenspunkte (`int`) an — Startwert: 100.
3. Lege eine Variable für die Angriffskraft (`int`) an — Startwert: 15.
4. Lege eine Variable für den Rüstungswert (`double`) an — Startwert: 2.5.
5. Lege eine Variable an, die speichert, ob der Spieler lebt (`boolean`).
6. Gib alle Werte formatiert auf der Konsole aus.

### Beispiel-Output

```
=== Spieler-Charakter ===
Name: HeldenName
LP: 100
Angriff: 15
Ruestung: 2.5
Lebend: true
=========================
```

### Hinweis

Verwende für den Spielernamen einen festen Wert (z. B. "Aragorn"). Die Eingabe über die Tastatur lernen wir später.
