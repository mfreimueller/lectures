# Kata 2: Richtigen Typ wählen

| Feld | Wert |
|------|------|
| Konzepte | Datentypen, Typwahl, Variablen |
| Schwierigkeit | 2/5 |
| Dauer | ca. 10 min |

### Aufgabenstellung

Für jeden der folgenden Werte sollst du den passenden Datentyp auswählen und die Variable deklarieren und initialisieren:

1. "Ist die Tür offen?" → Wert: `true`
2. Die Kreiszahl Pi → Wert: `3.14159`
3. Der Anfangsbuchstabe deines Vornamens
4. Die Anzahl der Schüler in der Klasse
5. Dein vollständiger Name

Gib alle Variablen formatiert auf der Konsole aus.

### Beispiel-Output

```
Tuer offen: true
Kreiszahl Pi: 3.14159
Anfangsbuchstabe: M
Schueleranzahl: 24
Name: Max Mustermann
```

### Erweiterung

Überlege dir drei weitere Werte aus dem Alltag und weise sie passenden Variablen zu.
