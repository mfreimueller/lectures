# Kata 3: Personen-Steckbrief

| Feld | Wert |
|------|------|
| Konzepte | Datentypen, Variablen, Konsolenausgabe |
| Schwierigkeit | 2/5 |
| Dauer | ca. 15 min |

### Aufgabenstellung

Erstelle einen Steckbrief für eine fiktive Person. Lege dafür die folgenden Variablen an:

- Name (`String`)
- Alter (`int`)
- Körpergröße in Metern (`double`)
- Ersten Buchstaben des Vornamens (`char`)
- Ob die Person volljährig ist (`boolean`)

Gib alle Informationen strukturiert auf der Konsole aus.

### Beispiel-Output

```
=== Personen-Steckbrief ===
Name: Anna Mueller
Alter: 21
Groesse: 1.72 m
Initial: A
Volljaehrig: true
===========================
```

### Erweiterung

Füge weitere sinnvolle Eigenschaften hinzu, z. B. Lieblingszahl (`int`), Haarfarbe (`String`) oder ob die Person einen Führerschein hat (`boolean`).
