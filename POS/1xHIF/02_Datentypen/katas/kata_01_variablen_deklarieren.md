# Kata 1: Variablen deklarieren

| Feld | Wert |
|------|------|
| Konzepte | Variablen, Datentypen, Deklaration, Initialisierung |
| Schwierigkeit | 1/5 |
| Dauer | ca. 10 min |

### Aufgabenstellung

Deklariere und initialisiere für jeden der fünf Datentypen (`boolean`, `int`, `double`, `String`, `char`) mindestens eine Variable. Gib jede Variable mit `System.out.println` auf der Konsole aus.

Achte auf sprechende Variablennamen!

### Beispiel-Output

```
Wahrheitswert: true
Ganze Zahl: 42
Kommazahl: 3.14
Text: Hallo Welt!
Buchstabe: A
```

### Erweiterung

Deklariere zwei Variablen vom selben Typ und gib beide aus. Probiere aus, was passiert, wenn du einer `int`-Variable eine `double`-Zahl zuweisen willst.
