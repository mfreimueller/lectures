# Kata 3: Taschenrechner für eine Formel

| Feld | Wert |
|------|------|
| Konzepte | Arithmetische Operatoren, Formeln, Variablen |
| Schwierigkeit | 2/5 |
| Dauer | ca. 15 min |

### Aufgabenstellung

Berechne die Fläche eines Rechtecks (Länge × Breite) und den Umfang (2 × (Länge + Breite)) mit gegebenen Werten. Verwende sprechende Variablennamen.

Gegebene Werte:
- Länge: 8.5
- Breite: 4.2

### Beispiel-Output

```
Rechteck-Berechnung
Laenge: 8.5
Breite: 4.2
Flaeche: 35.7
Umfang: 25.4
```

### Erweiterung

Berechne auch das Volumen eines Quaders (Länge × Breite × Höhe). Füge eine Höhen-Variable hinzu.
