# Kata 2: Ganzzahl- vs. Gleitkommadivision

| Feld | Wert |
|------|------|
| Konzepte | Division, Datentypen, Ganzzahl, Gleitkomma |
| Schwierigkeit | 2/5 |
| Dauer | ca. 10 min |

### Aufgabenstellung

Berechne `5 / 2` auf drei verschiedene Arten und gib jedes Ergebnis aus:

1. Beide Zahlen als `int`
2. Beide Zahlen als `double`
3. Eine Zahl als `int`, die andere als `double`

Erkläre die unterschiedlichen Ergebnisse in einem Kommentar.

### Beispiel-Output

```
int / int: 5 / 2 = 2
double / double: 5.0 / 2.0 = 2.5
int / double: 5 / 2.0 = 2.5
```

### Erweiterung

Probiere auch `7 / 3` und `1 / 3` in beiden Varianten aus. Was fällt dir auf?
