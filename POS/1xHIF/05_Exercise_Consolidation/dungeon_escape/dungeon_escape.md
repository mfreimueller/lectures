# Dungeon Escape - Woche 05: Code-Knacker

**Konzepte:** for, if, Modulo

### Aufgabe
Der Spieler findet eine Tuer mit einem 4-stelligen Zahlenschloss.
Finde den Code durch Brute-Force (for-Schleife ueber alle 0000-9999).
Bedingungen: Quersumme = 14, Zahl ist gerade, dritte Ziffer = 2 * erste Ziffer.

Der Code ist 2462.
