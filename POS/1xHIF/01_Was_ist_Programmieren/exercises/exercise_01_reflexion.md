# Exercise: Reflexion -- Was ist Programmieren?

Beantworte die folgenden Fragen schriftlich in ganzen Sätzen.

### Fragen

1. Was ist ein Programm? Nenne mindestens drei Beispiele für Programme aus dem Alltag.
2. Warum gibt es verschiedene Programmiersprachen? Was ist der Vorteil davon?
3. Was bedeutet "Kompilieren"? Erkläre den Vorgang in eigenen Worten.
4. Was bedeutet "Interpretieren"? Wie unterscheidet es sich vom Kompilieren?
5. Welche Aufgabe hat die Java Virtual Machine (JVM)?
6. Was versteht man unter "Computational Thinking"? Erkläre an einem Beispiel.
