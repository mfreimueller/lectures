# Kata 01: Code-Golf - Primzahlen

Gib alle Primzahlen bis n auf der Konsole aus.
Gewonnen hat, wer den Code in den wenigsten Zeichen schreibt.

## Aufgabe
Schreibe ein Programm, das alle Primzahlen von 2 bis n (Eingabe in main hartkodiert) ausgibt.

## Beispiel (n = 20)
```
2
3
5
7
11
13
17
19
```

## Regeln
- Das Programm muss vollstaendig lauffaehig sein
- Keine externen Bibliotheken (ausser java.util)
- Die main-Methode muss enthalten sein
- Leerzeichen und Zeilenumbrueche zaehlen nicht mit
