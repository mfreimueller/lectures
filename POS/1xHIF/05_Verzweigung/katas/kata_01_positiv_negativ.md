# Kata 1: Einfaches if — Positiv, negativ oder null

| Feld | Wert |
|------|------|
| Konzepte | if, else, Vergleichsoperatoren |
| Schwierigkeit | 1/5 |
| Dauer | ca. 10 min |

### Aufgabenstellung

Schreibe ein Programm, das eine Zahl (in einer Variable) prüft und ausgibt, ob sie positiv, negativ oder null ist. Verwende if und else.

Teste mit verschiedenen Werten: 7, -3, 0.

### Beispiel-Output

```
Zahl: 7
Die Zahl ist positiv.
---
Zahl: -3
Die Zahl ist negativ.
---
Zahl: 0
Die Zahl ist null.
```

### Erweiterung

Prüfe zusätzlich, ob die Zahl gerade oder ungerade ist (Hinweis: Modulo-Operator %).
