# Kata 2: Notenstufen

| Feld | Wert |
|------|------|
| Konzepte | if-else if-else, Vergleichsoperatoren |
| Schwierigkeit | 2/5 |
| Dauer | ca. 15 min |

### Aufgabenstellung

Schreibe ein Programm, das zu einer Punktezahl (0–100) die Note (1–5) ausgibt. Verwende eine if-else if-else-Kaskade.

Notenschlüssel:
- 90–100: Note 1
- 75–89: Note 2
- 60–74: Note 3
- 40–59: Note 4
- 0–39: Note 5

### Beispiel-Output

```
Punkte: 85
Note: 2
---
Punkte: 42
Note: 4
---
Punkte: 110
Ungueltige Punktezahl!
```

### Erweiterung

Prüfe zuerst, ob die Punktezahl im gültigen Bereich (0–100) liegt. Gib eine Fehlermeldung aus, wenn nicht.
