# Dungeon Escape - Woche 04: Kampfrunde

**Konzepte:** for-Schleife

### Aufgabe
Fuehre 3 Angriffe mit einer for-Schleife aus. Der Gegner hat 50 HP, dein Schaden ist 12.
Gib Schaden pro Runde und Gesamtschaden aus. Bei gegnerHp <= 0 brich ab.

### Beispiel-Output
Runde 1: Gegner HP = 38
Runde 2: Gegner HP = 26
Runde 3: Gegner HP = 14
