for-Schleife - Das Problem

Wir wollen 5x "Hallo" ausgeben:

System.out.println("Hallo");  // 1
System.out.println("Hallo");  // 2
System.out.println("Hallo");  // 3
System.out.println("Hallo");  // 4
System.out.println("Hallo");  // 5

Was, wenn wir 1000x ausgeben wollen?

Losung: for-Schleife

for-Schleife - Syntax

for (Initialisierung; Bedingung; Inkrement) {
    // Wiederholter Code
}

Beispiel:

for (int i = 0; i < 5; i++) {
    System.out.println("Hallo");
}

Ausgabe: Hallo (5x)

Die Zaehlvariable

for (int i = 0; i < 5; i++) {
    System.out.println("Durchlauf: " + i);
}

Zaehlvariablen-Optionen:

  • Start: int i = 0 (oder 1, 10, ...)
  • Bedingung: i < n (oder i <= n, i > 0, ...)
  • Schritt: i++ (oder i--, i += 2, ...)

Durchlauf der for-Schleife

for (int i = 0; i < 3; i++) {
    System.out.println(i);
}
  1. Initialisierung: i = 0
  2. Bedingung: 0 < 3? true -> Ausgabe: 0
  3. Inkrement: i = 1
  4. Bedingung: 1 < 3? true -> Ausgabe: 1
  5. Inkrement: i = 2
  6. Bedingung: 2 < 3? true -> Ausgabe: 2
  7. Inkrement: i = 3
  8. Bedingung: 3 < 3? false -> Ende

Summe 1..n

int n = 10;
int summe = 0;

for (int i = 1; i <= n; i++) {
    summe = summe + i;  // oder summe += i
}

System.out.println("Summe 1.." + n + " = " + summe);

Summe 1..10 = 55

Produkt (Fakultaet)

int n = 5;
int produkt = 1;

for (int i = 1; i <= n; i++) {
    produkt = produkt * i;  // produkt *= i
}

System.out.println(n + "! = " + produkt);

5! = 120

1x1-Reihe

int zahl = 3;

for (int i = 1; i <= 10; i++) {
    System.out.println(zahl + "x" + i + " = " + (zahl * i));
}

Ausgabe: 3x1=3, 3x2=6, ..., 3x10=30

Primzahl-Prüfung

int zahl = 17;
boolean istPrim = true;

for (int i = 2; i < zahl; i++) {
    if (zahl % i == 0) {
        istPrim = false;
        break;  // Schleife vorzeitig beenden
    }
}

if (istPrim) {
    System.out.println(zahl + " ist eine Primzahl.");
}

Typische Fehler

  • Endlosschleife: for (int i = 0; i < 10; )
  • Off-by-one: i <= statt i < (oder umgekehrt)
  • Zaehlvariable aendern: i = 5 im Schleifenkoerper
  • Semikolon zu frueh: for (int i=0; i<10; i++); { }

Dungeon Escape: Kampfrunde

int gegnerHp = 50;
int schaden = 12;

for (int runde = 1; runde <= 3; runde++) {
    System.out.println("Runde " + runde + ":");
    gegnerHp -= schaden;
    System.out.println("  Gegner HP: " + gegnerHp);
}

System.out.println("Gegner besiegt!");

Dungeon Escape - Aufgabe

  • Fuehre 3 Angriffe mit for-Schleife aus
  • Gib Schaden pro Runde aus
  • Berechne Gesamtschaden
  • Erweiterung: break bei gegnerHp <= 0
