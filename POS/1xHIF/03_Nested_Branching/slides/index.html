Geschachtelte ifs

Ein if innerhalb eines anderen if:

if (hatSchlüssel) {
    if (codeRichtig) {
        System.out.println("Tür geöffnet!");
    }
}

Erst die äussere, dann die innere Bedingung prüfen.

Geschachtelt vs. &&

// Geschachtelt
if (hatSchlüssel) {
    if (codeRichtig) {
        System.out.println("Tür geöffnet!");
    }
}

// Gleichwertig mit &&
if (hatSchlüssel && codeRichtig) {
    System.out.println("Tür geöffnet!");
}

Kurzschluss-Evaluierung (Short-circuit)

if (x != 0 && 10 / x > 2) {  // Sicher, wenn x = 0
    // ...
}

// Bei &&: Wenn erste Bedingung false,
// wird die zweite NICHT mehr ausgewertet.

Das verhindert Fehler (z.B. Division durch Null).

Beispiel 1: Passwort-Check

String pw = "geheim123";
boolean langGenug = pw.length() >= 8;
boolean hatZiffer = false;

// Prüfe auf Ziffer (vereinfacht)
if (langGenug && pw.contains("1")) {
    System.out.println("Passwort ist gültig!");
} else {
    System.out.println("Passwort zu schwach.");
}

Beispiel 2: Stein-Schere-Papier

int spieler = 0;  // 0 = Stein
int gegner = 2;   // 2 = Papier

// Papier wickelt Stein ein
if (spieler == gegner) {
    System.out.println("Unentschieden!");
} else if ((spieler == 0 && gegner == 2) ||  // Stein vs. Schere?
           (spieler == 1 && gegner == 0) ||  // Schere vs. Papier?
           (spieler == 2 && gegner == 1)) {  // Papier vs. Stein?
    System.out.println("Spieler gewinnt!");
} else {
    System.out.println("Gegner gewinnt!");
}

Beispiel 3: Dreieckstyp

int a = 5, b = 5, c = 5;

if (a == b && b == c) {
    System.out.println("Gleichseitig");
} else if (a == b || b == c || a == c) {
    System.out.println("Gleichschenklig");
} else {
    System.out.println("Allgemein");
}

Dungeon Escape: Verschlossene Tür

Der Spieler findet eine Tür mit einem magischen Schloss.

boolean hatSchlüssel = true;
int code = 2462;
int eingabeCode = 2462;

if (hatSchlüssel && code == eingabeCode) {
    System.out.println("Die Tür schwingt auf!");
} else {
    System.out.println("Die Tür bleibt verschlossen.");
}

Dungeon Escape - Aufgabe

  • Boolean: hatSchlüssel, kenntZauber
  • Tür öffnet sich nur wenn beide Bedingungen erfüllt sind
  • Verwende && für die Kombination
  • Tipp: mit ! (NOT) Bedingungen umkehren
