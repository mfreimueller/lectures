# Dungeon Escape - Woche 03: Verschlossene Tuer

**Konzepte:** Logische Operatoren (&&, ||, !), geschachtelte ifs

### Aufgabe
Der Spieler findet eine magisch versperrte Tuer. Sie oeffnet sich nur, wenn:
- Der Spieler den goldenen Schluessel hat (boolean hatSchluessel)
- Der Spieler den richtige Zauber spricht (boolean kenntZauber)

Verwende &&, um beide Bedingungen zu pruefen.

### Erweiterung
Ein alternatives Schloss oeffnet sich, wenn der Spieler EINEN Schluessel ODER einen Dietrich hat.
