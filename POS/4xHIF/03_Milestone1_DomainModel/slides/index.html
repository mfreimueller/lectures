03 - Milestone 1: Domain Model

POS - 4xHIF

Milestone 1 Presentations

  • 5-10 minutes per team
  • All team members must speak
  • Present your PlantUML diagram on screen
  • Be ready for questions from the class

What to Present

  1. Domain — What problem are you solving? Who are the users?
  2. Entities — At least 6, with attributes and relationships
  3. Value Objects — Email, Money, Address, etc.
  4. Business Logic — What makes this more than CRUD?
  5. Project Setup — Spring Boot + JPA repo ready

Grading Criteria

CriterionWeight
Completeness (≥6 entities, relationships)30%
Correct UML (multiplicity, relationship types)25%
Value Objects & RichTypes identified20%
Clarity of presentation15%
Project repo + Spring Boot setup10%

After Presentations

  • Incorporate feedback from the class
  • Update your PlantUML model in the repo
  • Add Spring Data JPA dependency to pom.xml
  • Create entity stubs (empty classes with @Entity)
