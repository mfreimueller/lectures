# Katas - Week 02: PlantUML_ValueObjects

## K1: PlantUML from Code
**Concepts:** PlantUML, class diagram, relationships
**Task:** Given a set of Java entity classes, write PlantUML code that generates their class diagram with correct relationships (association, aggregation, composition, inheritance).
**Duration:** 20 min

## K2: ValueObject Implementation
**Concepts:** immutability, equals/hashCode, factory methods, validation
**Task:** Implement `Email` as a Value Object with factory method, validation, and structural equality. Then implement a `Money` class with currency support.
**Duration:** 25 min

## K3: RichType Refactoring
**Concepts:** primitive obsession, rich types, type safety
**Task:** Refactor a given class that uses String and Long for everything (primitive obsession) into a model with proper RichTypes.
**Duration:** 20 min
