# Your tasks

1. **Clone** the repository
3. **Create a new branch:** `git checkout -b profile-yourname`
4. **Edit index.html** and add your profile (use the template for this)
5. **Commit your progress:** Use good commit messages
6. **Push:** `git push origin profile-yourname`
7. **Create a Pull Request** from your branch to master (use proper messages)
8. **Solve merge conflicts** until you can resolve the pull request
