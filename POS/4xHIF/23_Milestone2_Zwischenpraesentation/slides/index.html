23 - Milestone 2: Zwischenpräsentation

POS - 4xHIF

Präsentationsformat

  • Dauer: 10-15 Minuten pro Team
  • Format: Live-Demo + Architektur-Überblick
  • Alle Teammitglieder sollten etwas sagen
  • Im Anschluss: 5 Minuten Feedback

Erwartete Inhalte

  1. Domain Model — Kurzer Recap (Entity-Relationship-Diagramm)
  2. REST APIs — Endpoints demonstrieren (Swagger UI oder curl)
  3. Thymeleaf UI — Live-Demo der Web-Oberfläche
  4. Architektur — Schichten, Patterns, Entscheidungen
  5. Testing — Testabdeckung zeigen
  6. Lessons Learned — Was hat gut funktioniert? Was nicht?
