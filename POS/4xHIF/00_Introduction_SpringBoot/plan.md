## Previously Learned Content

*No previous content (first week of 4th grade). Students come from 3xHIF with knowledge of Javalin, JDBC, layered architecture, and modern Java features.*

## Learning Objectives

- Understand the 4th grade POS curriculum, grading, and the group project
- Explain the transition from Javalin (3rd grade) to Spring Boot
- Create a Spring Boot project using Spring Initializr
- Understand the Maven/Gradle project structure
- Implement a simple REST endpoint using `@RestController` and `@GetMapping`
- Set up a GitHub Classroom repository and collaborate in teams

## Presentation (slides/)

Introduces the course structure, grading, and the year-long group project. Recaps the transition from Javalin to Spring Boot, then guides students through creating their first Spring Boot project, writing a simple REST controller, and setting up GitHub Classroom for team collaboration.

## Katas (katas/)

1. **K1: Spring Boot Setup** — Create a Spring Boot project via Spring Initializr, explore the generated structure.
2. **K2: First REST Endpoint** — Implement a GET /hello endpoint returning a JSON greeting.
3. **K3: GitHub Classroom** — Set up the team repository, push the project, verify collaboration.
