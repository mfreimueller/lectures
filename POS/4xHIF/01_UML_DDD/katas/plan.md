# Katas - Week 01: UML_DDD

## K1: UML from Code
**Concepts:** association, aggregation, composition, inheritance, class diagram
**Task:** Given a set of Java classes, draw a UML class diagram showing all relationships, multiplicities, and navigability.
**Duration:** 20 min

## K2: DDD Event Storming
**Concepts:** domain events, bounded context, ubiquitous language
**Task:** Run a simplified event storming session for a library domain: identify domain events, commands, aggregates, and contexts on sticky notes.
**Duration:** 25 min

## K3: Domain Brainstorming
**Concepts:** project scoping, domain evaluation
**Task:** Brainstorm 3 potential domains for the group project. For each, evaluate: at least 6 entities, meaningful business logic, CRUD + search requirements.
**Duration:** 20 min
