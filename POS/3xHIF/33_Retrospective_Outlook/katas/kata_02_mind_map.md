# Kata 2: Topic Mind Map

| Field | Value |
|-------|-------|
| Concepts | Knowledge organization, visualization |
| Difficulty | 1/5 |
| Duration | approx. 15 min |

### Task

Create a mind map (text-based or drawn) of all topics covered in POS. Organize them into groups and show connections. Include:
1. Java Language Features
2. Testing and Design
3. Streams and Functional Programming
4. Reflection and Annotations
5. Databases (JDBC, JPA)
6. REST APIs and Architecture
7. POS-WMC Project

### Example Output

```
POS
├── Modern Java
│   ├── var, Records, Text Blocks
│   ├── Pattern Matching, Sealed Classes
│   └── Optional, Switch, Unnamed Variables
├── Testing
│   └── JUnit 5, AssertJ, TDD, Mockito
├── Design
│   └── SOLID Principles
├── Streams
│   ├── Intermediate (filter, map)
│   └── Terminal (collect, groupingBy)
├── Reflection & Annotations
├── Databases
│   ├── JDBC → PreparedStatement, DAO
│   └── JPA → Entities, JPQL, Inheritance
└── REST
    ├── HTTP, Javalin
    ├── DTOs, Validation
    └── Layered Architecture
```
