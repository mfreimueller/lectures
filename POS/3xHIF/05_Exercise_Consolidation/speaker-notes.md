## 2. Konsolidierungs-Katas (2h Theorie + 3h Praxis)

### K1: Contact Manager
- Records für Contact
- Optional für optionale Felder (phone, email)
- Streams für Filtern/Sortieren
- **Zeit:** 30 min

### K2: Expression Evaluator
- sealed Classes für Expression-Typen (Add, Subtract, Multiply, Number)
- Pattern Matching switch für Auswertung
- **Zeit:** 30 min

### K3: CSV Processor
- Files API für Datei-I/O
- Records für Datenzeilen
- Streams für Verarbeitung
- Pattern Matching für Typ-Dispatching
- **Zeit:** 45 min

### K4: Mini Project
- Eigenes kleines Projekt — alle Features kombinieren
- **Zeit:** 45 min

### K5: Code Analysis
- Gegebenen Code auf Anti-Patterns prüfen
- Verbesserungsvorschläge machen
- **Zeit:** 30 min

## PLÜ 1 Vorbereitung

- var und Records: Syntax, Verwendung, Einschränkungen
- Interfaces: Default/Static Methods, Konfliktregeln
- Pattern Matching instanceof: Syntax, Scope
- Text Blocks: Syntax, Whitespace-Handling
- Optional: Erzeugung, Konsum, Transformation, Best Practices
- switch Expressions: `->`, yield, Exhaustiveness
- sealed Classes: Syntax, permits, exhaustive switch
- Unnamed Variables: Einsatzorte
- Moderne APIs: String, Files, Collections
