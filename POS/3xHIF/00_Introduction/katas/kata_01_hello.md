# Kata 1: Hello World

| Field | Value |
|-------|-------|
| Concepts | var, System.out, Maven |
| Difficulty | 1/5 |
| Duration | approx. 10 min |

### Task

Write a Java program that:
1. Uses `var` to declare a greeting string variable
2. Prints the greeting to the console
3. Also prints the current date and time using `java.time.LocalDateTime`

### Example Output

```
Hello, POS3!
Current time: 2026-05-13T10:30:00
```

### Extension

Read a name from the command line argument and print a personalized greeting.
