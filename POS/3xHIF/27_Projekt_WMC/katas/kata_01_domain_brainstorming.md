# Kata 1: Project Domain Brainstorming

| Field | Value |
|-------|-------|
| Concepts | Domain modeling, requirements analysis |
| Difficulty | 1/5 |
| Duration | approx. 15 min |

### Task

Brainstorm 3 different project domains that interest you. For each domain:
1. Describe the domain in 2-3 sentences
2. List at least 6 potential database tables
3. Identify at least one 1:1, one 1:n, and one inheritance relationship
4. Evaluate which domain fits the project requirements best

### Example Output

```
Domain 1: Library Management
- Tables: User, Book, Author, Category, Loan, Review, Address
- 1:1: User - Address
- 1:n: Category - Book
- Inheritance: PhysicalBook / EBook extends Book
- Rating: 8/10 - well-suited for all requirements
```

### Extension

Write a one-page project vision document describing your chosen domain, target users, and core features.
