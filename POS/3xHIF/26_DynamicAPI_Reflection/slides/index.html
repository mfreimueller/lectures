Review: Reflection

  • Reflection allows inspecting classes, methods, fields at runtime
  • Class.forName(), obj.getClass(), Class<?> API
  • Method.invoke() for dynamic method calls
  • Field.get() / Field.set() for reading/writing fields

Review: Annotations

  • Annotations add metadata to code (classes, methods, fields)
  • @Retention(RUNTIME) makes them available at runtime
  • @Target specifies where annotations can be used
  • Processing: obj.getClass().isAnnotationPresent(MyAnnotation.class)

Motivation

  • Spring Boot uses annotations like @RestController, @GetMapping
  • Routes are auto-discovered without manual registration
  • This is called "convention over configuration"
  • Today we build our own mini version of this!

We will create a framework that scans for annotations and auto-registers routes in Javalin.

Annotation Design


@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.TYPE)
public @interface Controller {
    String path() default "";
}

@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.METHOD)
public @interface Get {
    String path() default "";
}

@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.METHOD)
public @interface Post {
    String path() default "";
}

Using the Annotations


@Controller(path = "/api/users")
public class UserController {

    private final UserService service;

    public UserController(UserService service) {
        this.service = service;
    }

    @Get(path = "/{id}")
    public UserResponse getUser(long id) {
        return service.getUser(id)
            .orElseThrow(() -> new ResourceNotFoundException("User", id));
    }

    @Post
    public UserResponse createUser(CreateUserRequest request) {
        return service.createUser(request);
    }
}

Scanning for Classes


// Classpath scanning using reflection
public class ClasspathScanner {

    public static List<Class<?>> scan(String packageName) throws Exception {
        var classes = new ArrayList<Class<?>>();
        var packagePath = packageName.replace('.', '/');
        var urls = Thread.currentThread()
            .getContextClassLoader()
            .getResources(packagePath);

        while (urls.hasMoreElements()) {
            var url = urls.nextElement();
            var file = new File(url.toURI());
            scanDirectory(file, packageName, classes);
        }
        return classes;
    }

    private static void scanDirectory(File dir, String packageName,
                                       List<Class<?>> classes) {
        for (var file : dir.listFiles()) {
            if (file.isDirectory()) {
                scanDirectory(file, packageName + "." + file.getName(), classes);
            } else if (file.getName().endsWith(".class")) {
                var className = packageName + "." +
                    file.getName().replace(".class", "");
                classes.add(Class.forName(className));
            }
        }
    }
}

Processing @Controller


public class RouteRegistrar {

    public static void registerControllers(Javalin app,
                                            List<Class<?>> classes) {
        for (var clazz : classes) {
            if (!clazz.isAnnotationPresent(Controller.class)) continue;

            var controller = clazz.getAnnotation(Controller.class);
            var basePath = controller.path();

            // Instantiate controller (simple DI)
            var instance = clazz.getDeclaredConstructor().newInstance();

            for (var method : clazz.getMethods()) {
                if (method.isAnnotationPresent(Get.class)) {
                    var get = method.getAnnotation(Get.class);
                    var fullPath = basePath + get.path();
                    var paramTypes = method.getParameterTypes();

                    app.get(fullPath, ctx -> {
                        // Resolve arguments and invoke
                        var args = resolveArguments(ctx, paramTypes);
                        var result = method.invoke(instance, args);
                        if (result != null) ctx.json(result);
                    });
                }
                // Similar for @Post, @Put, @Delete...
            }
        }
    }
}

Resolving Handler Arguments


private static Object[] resolveArguments(Context ctx,
                                          Class<?>[] paramTypes) {
    var args = new Object[paramTypes.length];
    for (int i = 0; i < paramTypes.length; i++) {
        if (paramTypes[i] == String.class) {
            args[i] = ctx.body();  // Default: request body
        } else if (paramTypes[i] == Long.class || paramTypes[i] == long.class) {
            args[i] = Long.parseLong(ctx.pathParam("id"));
        } else {
            // Try to deserialize from body
            args[i] = ctx.bodyAsClass(paramTypes[i]);
        }
    }
    return args;
}

@PathVariable Support


@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.PARAMETER)
public @interface PathVariable {
    String value();
}

// Enhanced argument resolver:
if (param.isAnnotationPresent(PathVariable.class)) {
    var pathVar = param.getAnnotation(PathVariable.class);
    var value = ctx.pathParam(pathVar.value());
    // Convert to correct type
    if (paramType == Long.class || paramType == long.class)
        args[i] = Long.parseLong(value);
    else if (paramType == Integer.class || paramType == int.class)
        args[i] = Integer.parseInt(value);
    else
        args[i] = value;
}

Putting It All Together


public class App {
    public static void main(String[] args) throws Exception {
        var app = Javalin.create().start(7070);

        // Scan for controllers
        var controllers = ClasspathScanner.scan("com.example.controllers");

        // Auto-register routes
        RouteRegistrar.registerControllers(app, controllers);

        System.out.println("Server started with auto-registered routes!");
    }
}

This is how Spring Boot discovers @RestController classes!

Summary

  • Custom annotations define metadata for route mapping
  • Classpath scanning finds annotated classes at startup
  • Reflection processes annotations and dynamically invokes methods
  • Javalin routes are registered programmatically based on annotations
  • This pattern is the foundation of Spring Boot, Micronaut, Quarkus
