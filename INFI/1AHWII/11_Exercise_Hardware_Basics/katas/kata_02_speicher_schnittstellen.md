# Kata 2: Speichermedien & Schnittstellen

## Ziel
Die Unterschiede zwischen HDD und SSD verstehen sowie die wichtigsten Schnittstellen zuordnen können.

## Aufgabenstellung
1. Vervollständige die folgende Tabelle zum Vergleich von HDD und SSD:

| Eigenschaft | HDD | SSD |
|---|---|---|
| Speichertechnologie | | |
| Typische Lesegeschwindigkeit | | |
| Stoßempfindlichkeit | | |
| Lebensdauer (Schreibzyklen) | | |
| Typische Geräuschentwicklung | | |

2. Nenne drei verschiedene Schnittstellen, an die eine SSD angeschlossen werden kann. Ordne jede Schnittstelle einer ungefähren maximalen Datenrate zu.
3. Erkläre in eigenen Worten, warum eine NVMe-SSD deutlich schneller ist als eine SATA-SSD.
4. Ein Kunde möchte einen alten PC mit einer HDD aufrüsten. Er überlegt, ob er eine SATA-SSD oder eine NVMe-SSD einbauen soll. Sein Mainboard hat einen M.2-Slot (PCIe 3.0 x4) und SATA-III-Anschlüsse. Welche Option würdest du ihm empfehlen? Begründe deine Antwort.

## Hinweise
- SATA III bietet maximal ~550 MB/s.
- NVMe (PCIe 3.0 x4) bietet bis zu ~3.500 MB/s.
- Die Lebensdauer einer SSD wird in TBW (Total Bytes Written) angegeben.
- Notiere deine Antworten in vollständigen Sätzen.
