27 - PowerPoint Präsentationen

INFI - 1AHWII

PowerPoint-Oberfläche

  • Menüband (Ribbon): Enthält alle Befehle, gruppiert in Registerkarten (Start, Einfügen, Entwurf, Übergänge, Animationen, Bildschirmpräsentation)
  • Folienbereich: Großflächige Ansicht der aktuellen Folie – hier wird bearbeitet
  • Folienübersicht (links): Miniaturansichten aller Folien zum schnellen Navigieren
  • Notizenbereich: Unterhalb der Folie – für Sprechnotizen, die in der Referentenansicht sichtbar sind
  • Statusleiste: Unten: Foliensymbol, Nummer, Rechtschreibprüfung, Ansichtsmodus, Zoom
  • Registerkarten: Kontextabhängige Zusatzregister (z. B. Bildtools bei Bildauswahl)

Neue Präsentation erstellen und speichern

  • Neue Präsentation: Datei → Neu → Leere Präsentation (oder Vorlage wählen)
  • Tastenkürzel: Strg + N für neue, leere Präsentation
  • Speichern: Strg + S (Schnellspeichern) oder F12 (Speichern unter)
  • Wichtige Dateiformate:
    FormatBeschreibung
    .pptxPowerPoint-Präsentation (Standard seit 2007)
    .pptÄlteres PowerPoint-Format (kompatibel)
    .pdfPortable Document Format (druckfertig)
    .ppsxBildschirmpräsentation (startet direkt im Vollbild)
  • AutoWiederherstellen speichert Änderungen regelmäßig im Hintergrund

Folienlayouts

  • Ein Layout bestimmt die Anordnung von Platzhaltern (Titel, Text, Bilder, etc.)
  • Auswahl: Start → Layout oder Rechtsklick auf Folie → Layout
  • Wichtige Layouts:
    • Titelfolie: Großer Titel, Untertitel – für die erste Folie
    • Titel und Inhalt: Standard – Titel + Aufzählungstext oder Tabelle/Grafik
    • Vergleich: Zwei Inhaltsbereiche nebeneinander
    • Nur Titel: Nur ein Titelplatzhalter – für Bilder oder Diagramme
    • Leer: Keine Platzhalter – maximale Freiheit
  • Nachträgliches Ändern des Layouts ist jederzeit möglich (Inhalte bleiben erhalten)

Texte auf Folien einfügen und formatieren

  • Text direkt in einen Platzhalter oder über Einfügen → Textfeld eingeben
  • Schriftformatierung: Start → Schriftart (Schriftart, -größe, -farbe, Fett, Kursiv, Unterstrichen)
  • Absatzformatierung: Ausrichtung, Aufzählungszeichen, Nummerierung, Zeilenabstand
  • WordArt: Einfügen → WordArt für dekorative Texteffekte
  • SmartArt: Einfügen → SmartArt für Diagramme, Prozesse, Hierarchien
  • Tipp: Weniger Text pro Folie – Stichpunkte reichen, ausformulieren im Vortrag

Bilder und Grafiken einfügen

  • Bild einfügen: Einfügen → Bilder (von Datei) oder Online-Grafiken (Bildersuche)
  • Formen: Einfügen → Formen (Rechtecke, Kreise, Pfeile, Linien, etc.)
  • Symbole und Icons: Einfügen → Symbole für moderne Vektor-Icons
  • Bildformatierung: Bildtools → Format (Korrekturen, Farbe, Effekte, Rahmen, Zuschneiden)
  • Anordnung: Bild in den Vorder-/Hintergrund, Ausrichten, Gruppieren
  • Tipp: Bilder nicht verzerren – Umschalt gedrückt halten beim Größenändern

Folienübergänge

  • Übergänge bestimmen, wie eine Folie erscheint (beim Wechsel zur nächsten Folie)
  • Einstellung: Registerkarte Übergänge – auf Folie anwenden oder auf alle
  • Kategorien:
    • Einfach: Kein Übergang, Wischen, Einblenden, Überblenden
    • Dezent: Auflösen, Würfel, Vorhang, Verschieben
    • Dynamisch: Fenster, Ränder, Zerfallen, Wechseln
  • Effekt-Optionen: Richtung (von links, von unten, etc.)
  • Dauer einstellbar (Sekundenbruchteile bis mehrere Sekunden)
  • Tipp: Zurückhaltend einsetzen – einheitlicher Übergang für alle Folien wirkt professionell

Animationen

  • Animationen bewegen einzelne Elemente auf einer Folie (Text, Bilder, Formen)
  • Einstellung: Registerkarte Animationen → Animationsbereich
  • Arten:
    • Eingang: Element erscheint (z. B. Einblenden, Wischen, Vergrößern)
    • Hervorhebung: Element wird betont (Farbänderung, Vergrößerung, Drehen)
    • Ausgang: Element verschwindet (Verblassen, Ausfliegen)
    • Bewegungspfade: Element bewegt sich auf einem Pfad (Linie, Kurve, Zeichnen)
  • Steuerung: Auslösen (per Mausklick), mit vorherigem/nachherigem, Dauer, Verzögerung
  • Tipp: Animationen sparsam und mit Bedacht einsetzen – sonst wirkt es unruhig

Präsentationstechniken

  • Bildschirmpräsentation starten:
    • F5: Ab der ersten Folie starten
    • Umschalt + F5: Ab der aktuellen Folie starten
  • Während der Präsentation:
    • Mit Pfeiltasten oder Klicken vor-/rückwärts navigieren
    • Esc beendet die Präsentation
    • B / W: Schwarzen / weißen Bildschirm anzeigen
  • Referentenansicht: Vortragender sieht Notizen, nächste Folie und Zeit – Publikum nur die Folie
  • Bildschirmpräsentation einrichten: Folien manuell vorgeführt, unbemerkt, Endlosschleife

Folienmaster – Grundkonzept

  • Der Folienmaster ist die Vorlage für das gesamte Design aller Folien
  • Aufruf: Ansicht → Folienmaster (eigene Registerkarte im Menüband)
  • Änderungen im Folienmaster wirken sich auf alle Folien aus:
    • Hintergrundfarbe und -grafiken
    • Schriftarten und -größen für Überschriften und Text
    • Platzierung von Logo und Fußzeile
  • Jedes Layout (Titelfolie, Inhalt, etc.) ist ein eigenes Layout innerhalb des Masters
  • Vorteil: Design-Änderungen in wenigen Klicks für die gesamte Präsentation

Tipps für gute Präsentationen

  • Weniger ist mehr: Keine überladenen Folien – max. 6–7 Zeilen pro Folie
  • Kontraste: Heller Hintergrund + dunkle Schrift (oder umgekehrt) – nie Gelb auf Weiß
  • Grafiken statt Text: Bilder, Diagramme und Icons sagen mehr als tausend Worte
  • Die 10-20-30-Regel (Guy Kawasaki):
    • Maximal 10 Folien (für 1 Stunde Vortrag)
    • Maximal 20 Minuten Vortragszeit
    • Mindestens 30 Punkt Schriftgröße
  • Einheitliches Design: Eine Schriftart, zwei Farben, konsistente Übergänge
  • Vorbereitung: Probelauf, technische Checks (Beamer, Adapter, Präsentationsmodus)

Zusammenfassung

  • PowerPoint-Oberfläche: Menüband, Folienbereich, Folienübersicht, Notizenbereich
  • Folienlayouts bestimmen die Anordnung von Inhalten (Titelfolie, Inhalt, Vergleich, Leer)
  • Texte, Bilder, Formen und SmartArt über die Registerkarte Einfügen
  • Übergänge: Folienwechsel gestalten (einfach, dezent, dynamisch)
  • Animationen: Einzelelemente bewegen (Eingang, Hervorhebung, Ausgang, Bewegungspfade)
  • Präsentation: F5 startet, Referentenansicht für Notizen und Vorschau
  • Folienmaster: Zentrale Design-Vorlage für das gesamte Präsentationsdesign
  • Gute Präsentationen: Weniger Text, starke Kontraste, einheitliches Design
