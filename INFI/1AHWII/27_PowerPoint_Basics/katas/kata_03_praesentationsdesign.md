# Kata 3: Präsentationsdesign

## Ziel
Eine Präsentation gestalterisch aufwerten, den Folienmaster anpassen und ein professionelles, einheitliches Design erstellen.

## Aufgabenstellung
1. Erstelle eine neue Präsentation zum Thema "Meine Lieblingsreise" (oder ein frei gewähltes Thema) mit 4–5 Folien.
2. Passe den **Folienmaster** an:
   - Ansicht → Folienmaster
   - Wähle eine eigenständige **Hintergrundfarbe** (z. B. ein dunkles Blau oder ein helles Grau)
   - Ändere die **Schriftart** für Überschriften auf "Calibri Light" (oder eine andere serifenlose Schrift)
   - Füge im Master ein **Logo oder eine Grafik** (z. B. ein Symbol aus Einfügen → Symbole) in der oberen rechten Ecke ein
   - Schließe die Master-Ansicht
3. Erstelle die Folien mit abwechslungsreichen Layouts:
   - Folie 1: Titelfolie
   - Folie 2: Titel und Inhalt (Aufzählung zu Reisezielen)
   - Folie 3: Vergleich (Vorher/Nachher, oder "Strand vs. Berge")
   - Folie 4: Nur Titel mit einem großen Bild
   - Folie 5: Leer mit einem zentrierten Zitat
4. Füge eine **Fußzeile** mit laufender Foliennummer ein (aber nicht auf der Titelfolie).
5. Wende einen **einheitlichen Folienübergang** auf alle Folien an (z. B. "Überblenden" mit 1 Sekunde Dauer).
6. Füge auf mindestens einer Folie eine **Animation** hinzu, die den Inhalt schrittweise aufbaut.

## Hinweise
- Änderungen im Folienmaster wirken sich auf alle Folien aus – das spart viel Zeit bei der Gestaltung.
- Ein einheitliches Farbschema (max. 3 Farben) und eine durchgängige Schriftart sorgen für ein professionelles Erscheinungsbild.
- Die Fußzeile wird im Folienmaster für alle Layouts zentral konfiguriert (außer Titelfolie).
- Für das Logo eignen sich die integrierten Symbole (Einfügen → Symbole) besonders gut, da sie als Vektorgrafiken skalieren.
