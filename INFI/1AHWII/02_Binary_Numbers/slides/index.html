02 - Binary Numbers

INFI - 1AHWII

Wiederholung: Binärzahlen

  • Binärzahlen verwenden die Basis 2
  • Ziffern: 0 und 1
  • Stellenwert = 2Position (von rechts, beginnend bei 0)
  • Beispiel: 10112 = 1·2³ + 0·2² + 1·2¹ + 1·2⁰ = 1110

Wiederholung: Stellenwerttabelle

  • Beispiel: 1 0 0 1 1 0 1 0
    • 128 + 0 + 0 + 16 + 8 + 0 + 2 + 0 = 15410

Dezimal zu Binär – Methode 1: Subtraktion

  • Wandle 4210 in Binär um:
  • Größter Stellenwert ≤ 42: 32 → Bit setzen, Rest = 10
  • Nächster: 16 > 10 → Bit = 0
  • 8 ≤ 10 → Bit setzen, Rest = 2
  • 4 > 2 → Bit = 0
  • 2 ≤ 2 → Bit setzen, Rest = 0
  • 1 > 0 → Bit = 0
  • Ergebnis: 001010102

Dezimal zu Binär – Methode 2: Division

  • Teile die Zahl wiederholt durch 2 und notiere die Reste
  • Beispiel: 4210
    • 42 ÷ 2 = 21 Rest 0
    • 21 ÷ 2 = 10 Rest 1
    • 10 ÷ 2 = 5 Rest 0
    • 5 ÷ 2 = 2 Rest 1
    • 2 ÷ 2 = 1 Rest 0
    • 1 ÷ 2 = 0 Rest 1
  • Reste von unten nach oben lesen: 1010102

Binäre Addition – Grundregeln

0 + 00 + 11 + 01 + 1
= 0= 1= 1= 0, Übertrag 1
  • 1 + 1 + 1 (mit Übertrag) = 1, Übertrag 1
  • Die binäre Addition funktioniert genau wie die dezimale Addition

Binäre Subtraktion – Grundlagen

  • 0 - 0 = 0
  • 1 - 0 = 1
  • 1 - 1 = 0
  • 0 - 1 = 1, mit Borger (leihe von links)
  • Beispiel: 1010 - 0110
    • 0 - 0 = 0
    • 1 - 1 = 0
    • 0 - 1 → borgen von links: 10 - 1 = 1
    • 0 (nach Borgen) - 0 = 0
    • Ergebnis: 0100 = 4

  • Alternativ: Subtraktion durch Addition des Zweierkomplements
  • a - b = a + (Zweierkomplement von b)

Zweierkomplement – Motivation

  • Wie stellt man negative Zahlen im Binärsystem dar?
  • Lösung: Zweierkomplement
  • Vorteile:
    • Subtraktion wird zur Addition
    • Kein separater Subtrahierer nötig
    • Eine eindeutige Darstellung der Null

Zweierkomplement – Berechnung

  • Schritte für -5 (8 Bit):
    1. Binärwert von +5: 00000101
    2. Alle Bits umkehren (Einerkomplement): 11111010
    3. +1 addieren: 11111011
  • Probe: 5 + (-5) = 0
    • 00000101 + 11111011 = 1 00000000
    • Der Überlauf wird ignoriert, Ergebnis = 0

Zusammenfassung

  • Binär = Stellenwertsystem mit Basis 2
  • Umrechnung Dezimal ↔ Binär durch Division oder Subtraktion
  • Binäre Addition: 1+1 = 0 mit Übertrag
  • Binäre Subtraktion: Borger bei 0-1
  • Zweierkomplement: Bitweise negieren + 1
  • Vorzeichenbehaftet: MSB = Vorzeichenbit
