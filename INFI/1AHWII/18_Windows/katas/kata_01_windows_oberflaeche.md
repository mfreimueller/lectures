# KATA 1: Windows-Oberfläche

**Dauer:** 15 Minuten

## Ziel
Die Windows-Benutzeroberfläche mit ihren Elementen beschreiben und bedienen können.

## Aufgaben

### Aufgabe 1: Elemente der Windows-Oberfläche (7 Min.)

Beschriften Sie die folgenden Elemente der Windows-Oberfläche und nennen Sie ihre Funktion:

a) Taskleiste
b) Startmenü
c) Infobereich (System Tray)
d) Aufgabenansicht (Task View)
e) Aero Snap

### Aufgabe 2: Tastenkombinationen (4 Min.)

Ordnen Sie den folgenden Aktionen die richtige Tastenkombination zu:

| Aktion | Tastenkombination |
|--------|------------------|
| Suchfunktion öffnen | |
| Aufgabenansicht öffnen | |
| Fenster an linke Bildschirmseite andocken | |
| Einstellungen öffnen | |
| Task-Manager öffnen | |

Wählen Sie aus: `Win + I`, `Win + S`, `Win + Tab`, `Win + Pfeiltaste links`, `Strg + Shift + Esc`.

### Aufgabe 3: Beschreibung (4 Min.)

Erklären Sie in 3–4 Sätzen, wie ein Benutzer ein Programm startet, zwischen mehreren offenen Fenstern wechselt und einen virtuellen Desktop hinzufügt.

---

## Lösungshinweise

**Aufgabe 1:**
a) Zeigt laufende Programme, Fixieren möglich
b) Zentrale Anlaufstelle zum Starten von Programmen
c) Uhrzeit, Lautstärke, Netzwerk, Benachrichtigungen
d) Fensterwechsel und virtuelle Desktops (Win + Tab)
e) Fenster an Seiten/Ecken ziehen für automatische Größenanpassung

**Aufgabe 2:**
| Aktion | Kombination |
|--------|------------|
| Suche | Win + S |
| Aufgabenansicht | Win + Tab |
| Fenster andocken | Win + Pfeiltaste links |
| Einstellungen | Win + I |
| Task-Manager | Strg + Shift + Esc |

**Aufgabe 3:** Individuelle Antwort, soll die Bedienung von Startmenü, Alt+Tab und Aufgabenansicht beschreiben.
