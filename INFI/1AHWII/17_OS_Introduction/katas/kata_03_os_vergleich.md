# KATA 3: Betriebssysteme im Vergleich

**Dauer:** 15 Minuten

## Ziel
Die Unterschiede zwischen Windows, Linux und macOS verstehen und bewerten können.

## Aufgaben

### Aufgabe 1: Vergleichstabelle (8 Min.)

Ergänzen Sie die folgende Tabelle zu den drei Betriebssystemen:

| Merkmal | Windows | Linux | macOS |
|---------|---------|-------|-------|
| Entwickler | | | |
| Kernel-Typ | | | |
| Primäres Dateisystem | | | |
| Standard-Paketverwaltung | | | |
| GUI-Name | | | |
| Lizenzmodell | | | |

### Aufgabe 2: Einsatzzweck (4 Min.)

Für jedes der folgenden Szenarien wählen Sie das Ihrer Meinung nach am besten geeignete Betriebssystem aus und begründen Sie kurz:

a) Ein Webserver mit vielen gleichzeitigen Anfragen, der kostengünstig betrieben werden soll
b) Ein Arbeitsplatz für professionelle Videobearbeitung in einem Kreativstudio
c) Ein Firmen-Desktop-PC in einer großen Organisation mit zentraler Verwaltung

### Aufgabe 3: Offene Frage (3 Min.)

Warum gibt es von Linux so viele verschiedene Distributionen (Ubuntu, Debian, Fedora, Arch, …), während Windows und macOS nur von einem Hersteller stammen? Welche Vor- und Nachteile hat dies?

---

## Lösungshinweise

**Aufgabe 1:**
| Merkmal | Windows | Linux | macOS |
|---------|---------|-------|-------|
| Entwickler | Microsoft | Open-Source-Gemeinschaft | Apple |
| Kernel-Typ | Hybrid-Kernel (NT) | Monolithischer Kernel | Hybrid-Kernel (XNU) |
| Primäres Dateisystem | NTFS | ext4, Btrfs, XFS | APFS |
| Standard-Paketverwaltung | MSI, Store | apt, dnf, pacman | .app, Homebrew |
| GUI-Name | Windows Explorer | GNOME, KDE, … | Aqua (Finder) |
| Lizenzmodell | Proprietär | Open Source (GPL) | Proprietär |

**Aufgabe 2:** Diskussionsfrage. a) Linux (kostenlos, stabil, geringer Ressourcenverbrauch), b) macOS (Final Cut, ProRes-Hardware), c) Windows (Active Directory, Gruppenrichtlinien, große Enterprise-Unterstützung).

**Aufgabe 3:** Linux-Kernel ist Open Source (GPL) → jeder kann eigene Distribution erstellen. Vorteile: Vielfalt, Spezialisierung. Nachteile: Fragmentierung, Inkompatibilitäten.
