01 - Introduction

INFI - 1AHWII

Was ist Informatik?

  • Informatik = Information + Automatik
  • Wissenschaft von der systematischen Verarbeitung von Informationen
  • Teilgebiete:
    • Theoretische Informatik
    • Praktische Informatik
    • Technische Informatik
    • Angewandte Informatik

Teilgebiete der Informatik

TeilgebietBeschreibung
Theoretische InformatikAlgorithmen, Berechenbarkeit, formale Sprachen
Praktische InformatikProgrammierung, Softwareentwicklung
Technische InformatikHardware, Schaltungen, Rechnerarchitektur
Angewandte InformatikDatenbanken, Netzwerke, KI

Definition eines Computers

  • Ein Computer ist eine elektronische Rechenanlage
  • Eingabe → Verarbeitung → Ausgabe (EVA-Prinzip)
  • Besteht aus:
    • Hardware (physische Komponenten)
    • Software (Programme und Daten)

Das EVA-Prinzip

SchrittBeschreibungBeispiel
EingabeDaten aufnehmenTastatur, Maus, Mikrofon
VerarbeitungDaten verarbeitenCPU, GPU
AusgabeErgebnisse ausgebenBildschirm, Drucker, Lautsprecher

Hardware vs. Software

  • Hardware = alle physischen Bestandteile
    • Beispiele: Prozessor, RAM, Festplatte, Mainboard
    • Anfassbar, sichtbar
  • Software = Programme und Daten
    • Beispiele: Betriebssystem, Browser, Spiele
    • Nicht anfassbar, aber ausführbar

Hardware-Komponenten im Überblick

KomponenteFunktion
Prozessor (CPU)Befehle ausführen, Rechnen
Arbeitsspeicher (RAM)Daten kurzzeitig speichern
Festplatte (SSD/HDD)Daten dauerhaft speichern
MainboardVerbindet alle Komponenten
NetzteilStromversorgung

Bits und Bytes

  • Bit = kleinste Informationseinheit (0 oder 1)
  • Byte = 8 Bit
  • Ein Bit kann 2 Zustände annehmen (0 oder 1)
  • Mit n Bits können 2n verschiedene Werte dargestellt werden

Binärzahlen – Stellenwertsystem

  • Binärzahlen verwenden Basis 2
  • Jede Stelle hat einen Stellenwert (2n)
  • Beispiel: 10112
    • = 1×23 + 0×22 + 1×21 + 1×20
    • = 8 + 0 + 2 + 1 = 1110

Datengrößen (KB, MB, GB, TB)

  • 1 Byte = 8 Bit
  • 1 Kilobyte (KB) = 1.024 Byte
  • 1 Megabyte (MB) = 1.024 KB = 1.048.576 Byte
  • 1 Gigabyte (GB) = 1.024 MB
  • 1 Terabyte (TB) = 1.024 GB

Arten von Computern

  • Embedded System – in Geräte eingebettet (z. B. Mikrowelle, Auto)
  • PC / Laptop – Allzweckrechner für zu Hause / Büro
  • Server – stellt Dienste für andere Rechner bereit
  • Mainframe – Großrechner für Unternehmen
  • Supercomputer – Höchstleistungsrechner für Forschung

Zusammenfassung

  • Informatik = Information + Automatik
  • Computer arbeitet nach dem EVA-Prinzip
  • Hardware (physisch) vs. Software (logisch)
  • Bit = kleinste Einheit, Byte = 8 Bit
  • Binärzahlen: Basis 2, Stellenwerte 2n
  • Datengrößen: KB, MB, GB, TB (Faktor 1.024)
  • Computerarten: Embedded bis Supercomputer
