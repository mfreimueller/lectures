# Kata 1: VM einrichten

## Ziel
Eine virtuelle Maschine mit Ubuntu Linux in VirtualBox einrichten.

## Aufgabenstellung
1. Laden Sie <strong>VirtualBox</strong> von virtualbox.org herunter und installieren Sie es.
2. Laden Sie die aktuelle <strong>Ubuntu LTS</strong>-Version (ubuntu.com) als ISO-Datei herunter.
3. Erstellen Sie eine neue VM mit folgenden Einstellungen:
   - Name: `Ubuntu-VM`
   - Typ: Linux, Version: Ubuntu (64-bit)
   - Arbeitsspeicher: 2048 MB
   - Festplatte: 20 GB (dynamisch alloziert)
4. Legen Sie die ISO-Datei in das virtuelle CD-Laufwerk ein.
5. Starten Sie die VM und installieren Sie Ubuntu mit deutscher Tastaturbelegung.
6. Legen Sie einen Benutzer mit Ihrem Vornamen an.

## Hinweise
- Aktivieren Sie in den BIOS-Einstellungen Ihres PCs die Virtualisierung (VT-x/AMD-V).
- Installieren Sie nach der Ubuntu-Installation die <strong>Gast-Erweiterungen</strong> (Devices → Insert Guest Additions CD image).
- Notieren Sie sich Ihren Benutzernamen und Ihr Passwort.

## Abgabe
Bestätigen Sie mit einem Screenshot des Ubuntu-Desktops nach erfolgreichem Login.
