# Kata 2: USB-Standards

## Ziel
Die verschiedenen USB-Versionen, ihre Eigenschaften und Steckertypen sicher zuordnen können.

## Aufgabenstellung

### Teil 1: Vergleichstabelle
Vervollständige die folgende Tabelle:

| USB-Version | Max. Datenrate | Typische Steckerfarbe | Steckertypen |
|---|---|---|---|
| USB 2.0 | | | |
| USB 3.0 / 3.1 Gen 1 | | | |
| USB 3.1 Gen 2 | | | |
| USB4 | | | |

### Teil 2: Zuordnung
Ordne die folgenden Geräte der jeweils sinnvollsten USB-Version zu und begründe:
1. Externe 4K-Webcam für Videokonferenzen
2. Maus mit Kabel
3. Externe NVMe-SSD für Videobearbeitung
4. Tastatur mit Hintergrundbeleuchtung
5. USB-C-Docking-Station mit zwei externen Monitoren

### Teil 3: Wahr oder falsch?
1. USB 3.0 ist abwärtskompatibel zu USB 2.0.
2. USB-C unterstützt immer USB4.
3. USB Power Delivery (USB PD) ermöglicht das Laden von Laptops über USB-C.
4. Ein USB 2.0-Gerät funktioniert an einem USB 3.0-Anschluss langsamer als an einem USB 2.0-Anschluss.

## Hinweise
- Die Tabelle in Teil 1 sollte alle relevanten Kenndaten enthalten.
- Bei Teil 2 kommt es auf die passende Begründung an, nicht nur auf die Nennung der Version.
- Bei Teil 3: Markiere jede Aussage als "Wahr" oder "Falsch" und korrigiere falsche Aussagen.
