# Kata 3: Grafik-APIs

## Ziel
Die wichtigsten Grafik-APIs (DirectX, OpenGL, Vulkan) unterscheiden und ihre Einsatzgebiete zuordnen können.

## Aufgabenstellung
1. Erstelle eine Tabelle mit folgenden Informationen für jede der drei APIs:
   - Entwickler / Organisation
   - Unterstützte Plattformen
   - Besonderheiten
   - Aktuelle Version
2. Ordne die APIs den folgenden Einsatzszenarien zu und begründe:
   - Ein Windows-only-Spiel aus dem Jahr 2010
   - Ein modernes Multiplattform-Spiel (Windows, Linux, macOS)
   - Eine wissenschaftliche Simulation, die höchste Leistung benötigt
   - Eine professionelle CAD-Anwendung
3. Welche Vorteile bietet Vulkan gegenüber OpenGL? Nenne mindestens drei.

## Hinweise
- DirectX wird nur von Microsoft-Plattformen unterstützt.
- OpenGL ist älter, aber auf vielen Systemen verfügbar.
- Vulkan bietet eine feinere Kontrolle über die Hardware.
- Metal ist Apples eigene API (macOS, iOS) – wird in dieser Kata nicht vertieft.
