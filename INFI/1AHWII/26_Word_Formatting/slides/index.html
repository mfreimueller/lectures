26 - Word - Formatierung & Layout

INFI - 1AHWII

Wiederholung: Schrift- und Absatzformatierung

  • Schriftformatierung: Schriftart, -größe, -schnitt (fett, kursiv), Farbe, Effekte
  • Absatzformatierung: Ausrichtung, Einzug, Zeilenabstand, Abstände
  • Formatierungen in der Registerkarte Start (Zeichen- und Absatzgruppe)
  • Direkte Formatierung: einzeln auf ausgewählten Text angewandt
  • Probleme: aufwändig bei vielen Absätzen, kein einheitliches Layout
  • Lösung: Formatvorlagen für konsistente Dokumente

Formatvorlagen – Was sind sie?

  • Eine Formatvorlage ist ein Satz von Formatierungsmerkmalen unter einem Namen
  • Enthält Schriftart, -größe, -farbe, Absatzausrichtung, Zeilenabstand, Rahmen, etc.
  • Word unterscheidet:
    • Absatzvorlagen (Überschrift 1, Normal, Zitat)
    • Zeichenvorlagen (Hervorhebung, starker Ausdruck)
    • Listen- und Tabellenvorlagen
  • Vorteile: Einheitlichkeit, schnelle Änderung, automatische Verzeichnisse

Formatvorlagen erstellen und anpassen

  • Bereich "Formatvorlagen" in der Registerkarte Start
  • Neue Vorlage: kleines Symbol unten rechts → "Neue Formatvorlage"
  • Name vergeben, Formatierung festlegen (Schrift, Absatz, Rahmen, etc.)
  • Vorlage ändern: Rechtsklick auf Vorlage → "Ändern"
  • Alle Absätze mit dieser Vorlage werden automatisch aktualisiert
  • Vorlage als Standard speichern (in der Dokumentvorlage Normal.dotm)

Formatvorlagen: Überschriften, Text, Zitate

  • Überschrift 1–3: Hierarchische Gliederung, Grundlage für Inhaltsverzeichnis
  • Normal: Standard-Absatzvorlage für Fließtext
  • Zitat / Intensiv: für hervorgehobene Textstellen
  • Eigene Vorlagen: z. B. "Code", "Infobox", "Aufgabe"
  • Überschriften-Vorlagen mit Nummerierung (Gliederung): Start → Absatz → Nummerierung
  • Tipp: Überschriften-Ebenen nicht direkt formatieren, sondern Vorlagen anpassen

Seitenlayout

  • Registerkarte Layout → Seiten einrichten
  • Ausrichtung: Hochformat / Querformat
  • Größe: A4, Letter, benutzerdefiniert
  • Spalten: Text in zwei oder mehr Spalten anordnen (Layout → Spalten)
  • Seitenränder: oben/unten/links/rechts über "Ränder" einstellen
  • Einstellungen gelten für das gesamte Dokument oder ab einem Abschnittsumbruch

Kopf- und Fußzeilen einfügen

  • Einfügen → Kopfzeile / Fußzeile
  • Enthalten meist: Dokumenttitel, Kapitelname, Datum, Seitenzahl
  • Unterschiedliche Kopf-/Fußzeilen möglich:
    • Für erste Seite (Deckblatt)
    • Für gerade/ungerade Seiten (Bücher)
  • Bearbeitung: Doppelklick in den Kopf-/Fußzeilenbereich
  • Abschnittsumbrüche erlauben unterschiedliche Kopf-/Fußzeilen pro Abschnitt

Seitenzahlen und Datumsfelder

  • Einfügen → Seitenzahl (unten, oben, Seitenrand, aktuelle Position)
  • Formatierung: Zahlenformat, beginnend mit ... (z. B. Seite 1 ab Kapitel 2)
  • Felder in Word: automatisch aktualisierte Inhalte
  • Datum/Uhrzeit: Einfügen → Datum und Uhrzeit (automatisch aktualisierbar)
  • Weitere Felder: "Dokumenteigenschaften", "Dateiname", "Autor"
  • Tastenkürzel: Alt+Umschalt+D für Datum, Alt+Umschalt+T für Uhrzeit

Umbrüche

  • Seitenumbruch: Strg+Enter – neuen Text auf nächster Seite beginnen
  • Spaltenumbruch: Text in nächste Spalte verschieben
  • Abschnittsumbruch: trennt Dokument in Abschnitte mit eigenem Layout
  • Typen: Nächste Seite, Fortlaufend, Gerade Seite, Ungerade Seite
  • Verwendung: andere Kopf-/Fußzeile, andere Spaltenanzahl, Seitenränder
  • Einfügen: Layout → Umbrüche

Inhaltsverzeichnis automatisch erstellen

  • Grundlage: Formatvorlagen Überschrift 1–3 müssen vergeben sein
  • Einfügen → Inhaltsverzeichnis → Automatische Tabelle (oder benutzerdefiniert)
  • Word durchsucht das Dokument nach Überschriften-Vorlagen
  • Aktualisieren: Rechtsklick auf Inhaltsverzeichnis → "Feldfunktion aktualisieren"
  • Optionen: Ebenenanzahl, Füllzeichen, Formatvorlagen für Einträge
  • Auch für Abbildungsverzeichnis und Tabellenverzeichnis nutzbar

Bilder und Grafiken einfügen und positionieren

  • Einfügen → Bilder (Datei) oder Online-Grafiken
  • Textumbruch bestimmt, wie Text um das Bild fließt:
  • Optionen: "Im Fließtext" (Standard), "Quadrat", "Passend", "oben/unten"
  • Weitere: "Hinter den Text", "Vor den Text", "Durch"
  • Position: "Verankern" am Absatz, "Mit Text verschieben"
  • Bildtools: Format → Korrekturen, Farbe, Effekte, Größe, Zuschneiden

Seriendruck – Grundlagen

  • Seriendruck erstellt mehrere personalisierte Dokumente aus einer Vorlage
  • Anwendungen: Briefe, E-Mails, Etiketten, Umschläge, Kataloge
  • Briefvorlage: Word-Dokument mit Platzhaltern (Seriendruckfelder)
  • Datenquelle: Excel-Tabelle, Access, Outlook-Kontakte, CSV-Datei
  • Felder: Vorname, Nachname, Straße, PLZ, Ort, Anrede, ...
  • Start: Sendungen → Seriendruck starten → Schritt-für-Schritt-Assistent

Seriendruck – Briefe zusammenführen, Etiketten

  • Briefe zusammenführen:
    • Vorlage + Datenquelle → Seriendruckfelder einfügen
    • Vorschau: "Ergebnisse anzeigen" prüft alle Datensätze
    • Abschluss: "Alle" → Einzeldokument, E-Mail oder direkt drucken
  • Etiketten: Sendungen → Etiketten → Datenquelle wählen
  • Adressetiketten, Namensschilder, Ordneretiketten
  • Tipp: Platzhalter mit "Wenn… dann… sonst" für geschlechtsspezifische Anrede

Zusammenfassung

  • Formatvorlagen: konsistente Formatierung, schnelle Änderungen, Basis für Verzeichnisse
  • Seitenlayout: Ausrichtung, Größe, Spalten, Ränder über Layout-Registerkarte
  • Kopf-/Fußzeilen: Seitenzahlen, Datumsfelder, verschiedene pro Abschnitt
  • Abschnittsumbrüche trennen Layout-Bereiche im Dokument
  • Inhaltsverzeichnis: automatisch aus Überschriften-Vorlagen erstellt
  • Bilder: positionieren mit Textumbruch (Quadrat, Passend, etc.)
  • Seriendruck: personalisierte Briefe und Etiketten aus Datenquelle
