# Kata 2: Logische Ausdrücke

## Ziel
Logische Ausdrücke auswerten und vereinfachen.

## Aufgabenstellung
1. Werten Sie folgende logische Ausdrücke aus (A = wahr, B = falsch, C = wahr):
   - A ∧ B
   - A ∨ B
   - ¬A ∨ C
   - (A ∧ C) ∨ (B ∧ C)
2. Vereinfachen Sie folgende Ausdrücke mithilfe der logischen Identitäten:
   - A ∧ (A ∨ B)
   - A ∨ (A ∧ B)
   - (A ∨ B) ∧ (A ∨ ¬B)
3. Zeigen Sie durch Wahrheitstabelle, dass A → B (Implikation) äquivalent zu ¬A ∨ B ist.

## Hinweise
- Nutzen Sie die Absorptionsgesetze: A ∧ (A ∨ B) ≡ A und A ∨ (A ∧ B) ≡ A.
- Eine Implikation A → B ist nur falsch, wenn A wahr und B falsch ist.
- Überprüfen Sie Ihre Vereinfachungen mit einer Wahrheitstabelle.
