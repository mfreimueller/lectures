25 - Word - Textverarbeitung

INFI - 1AHWII

Die Word-Oberfläche

  • Menüband (Ribbon): Enthält alle Befehle, gruppiert in Registerkarten
  • Registerkarten: Start, Einfügen, Entwurf, Layout, Referenzen, Sendungen, Überprüfen, Ansicht
  • Schnellzugriffsleiste: Anpassbare Symbolleiste über dem Menüband
  • Statusleiste: Unten – zeigt Seitenzahl, Wortanzahl, Ansichtsmodus, Zoom
  • Minimieren des Menübands: Strg + F1 oder Doppelklick auf Registerkarte

Dokumente erstellen, öffnen und speichern

  • Neues Dokument: Datei → Neu oder Strg + N
  • Öffnen: Datei → Öffnen oder Strg + O
  • Speichern: Strg + S (Schnellspeichern) oder F12 (Speichern unter)
  • Wichtige Dateiformate:
    FormatBeschreibung
    .docxWord-OpenXML-Format (Standard seit 2007)
    .docÄlteres Word-Format (kompatibel)
    .pdfPortable Document Format (druckfertig)
    .rtfRich Text Format (plattformunabhängig)
  • AutoWiederherstellen speichert Dokumente in regelmäßigen Abständen

Texteingabe und -bearbeitung

  • Text einfach tippen – Word führt automatischen Zeilenumbruch durch
  • Einfügen vs. Überschreiben: Mit Einfg (Insert) zwischen den Modi wechseln
  • Mit Entf Zeichen rechts vom Cursor löschen, mit Rück (Backspace) links
  • Rückgängig: Strg + Z – macht letzte Aktion rückgängig
  • Wiederholen: Strg + Y – stellt rückgängig gemachte Aktion wieder her
  • Text markieren: Mit Maus ziehen, Shift + Pfeiltasten, oder Strg + A (alles markieren)

Schriftformatierung

  • Über das Menüband Start → Schriftart oder das Minisymbolleisten-Kontextmenü
  • Schriftart: Z. B. Calibri, Arial, Times New Roman, Courier New
  • Schriftgröße: In Punkt (pt) – Standard 11 pt (neu), 12 pt (klassisch)
  • Schriftfarbe: Über das Farbsymbol – auch Hervorhebungsfarbe (Marker)
  • Textauszeichnungen:
    • Fett: Strg + B
    • Kursiv: Strg + I
    • Unterstrichen: Strg + U
    • Durchgestrichen

Absatzformatierung

  • Ausrichtung (Start → Absatz):
    • Linksbündig (Strg + L)
    • Zentriert (Strg + E)
    • Rechtsbündig (Strg + R)
    • Blocksatz (Strg + J)
  • Einzüge: Erste Zeile, hängend, links/rechts im Dialogformat
  • Zeilenabstand: 1,0 / 1,5 / 2,0 – über Start → Zeilenabstand
  • Absatzabstand: Abstand vor/nach Absätzen – im Dialog Absatz → Abstände

Aufzählungszeichen und Nummerierungen

  • Über Start → Absatz:
    • Aufzählungszeichen: Punkte, Striche, Symbole
    • Nummerierungen: 1., 2., 3. / a., b., c. / I., II., III.
    • Mehrstufige Listen: Gliederungen, z. B. 1 / 1.1 / 1.1.1
  • Aufzählungs-/Nummerierungs-Bibliothek bietet vordefinierte Formate
  • Eigene Symbole und Formate definieren über Neues Aufzählungszeichen definieren
  • Mit Tab die Ebene erhöhen, mit Shift + Tab verringern
  • Kontrollkästchen-Aufzählungen für Checklisten

Tabellen einfügen und formatieren

  • Tabelle einfügen:
    • Über Einfügen → Tabelle – Raster auswählen
    • Oder: Tabelle einfügen für genaue Zeilen-/Spaltenanzahl
    • Tabelle zeichnen (frei definierbar)
    • Excel-Tabelle einbetten
  • Formatieren mit Tabellenentwurf (erscheint bei Auswahl):
    • Tabellenvorlagen (Farben, Rahmen)
    • Kopfzeile hervorgehoben, Streifenmuster
  • Layout-Tab: Zellen verbinden, teilen, Größe anpassen
  • Tabellenbeschriftung: Rechtsklick → Beschriftung einfügen

Seitenränder und Seitenausrichtung

  • Seitenränder über Layout → Seitenränder:
    • Normal (2,54 cm oben/unten, links/rechts)
    • Schmal, Mittel, Breit – oder benutzerdefiniert
  • Seitenausrichtung:
    • Hochformat (Standard)
    • Querformat – für breite Tabellen oder Grafiken
  • Ausrichtung für einzelne Abschnitte über Layout → Umbrüche → Abschnittsumbrüche
  • Spaltenlayout: Layout → Spalten (z. B. für Zeitungen/Flyer)
  • Seitenhintergrund: Entwurf → Seitenfarbe / Seitenränder

Rechtschreibprüfung und Autokorrektur

  • Rechtschreibprüfung:
    • Automatisch: rot-welllinig = Rechtschreibfehler
    • Blau-welllinig = Grammatikfehler
    • Überprüfen mit F7 oder Überprüfen → Rechtschreibung
  • Autokorrektur:
    • Korrigiert häufige Tippfehler automatisch (z. B. "dass" statt "das")
    • Wörterbuch erweiterbar unter Datei → Optionen → Dokumentprüfung → Autokorrektur-Optionen
  • Kontextmenü (rechtsklick) schlägt Korrekturen vor
  • Thesaurus: Überprüfen → Thesaurus oder Shift + F7

Tipps und Tricks

  • Wichtige Tastenkombinationen:
    TasteAktion
    Strg + ZRückgängig
    Strg + YWiederholen
    Strg + C / VKopieren / Einfügen
    Strg + XAusschneiden
    Strg + FSuchen
    Strg + HErsetzen
  • Format übertragen (Format-Pinsel):
    • Einmalig: Auf formatierte Stelle klicken, Symbol für Format übertragen, Ziel markieren
    • Mehrfach: Doppelklick auf Symbol; mit Esc beenden
  • F4: Wiederholt den letzten Befehl

Zusammenfassung

  • Word-Oberfläche: Menüband, Registerkarten, Schnellzugriffsleiste
  • Dokumente speichern als .docx (Standard) oder .pdf
  • Schriftformatierung: Schriftart, -größe, -farbe, Fett/Kursiv/Unterstrichen
  • Absatzformatierung: Ausrichtung, Einzüge, Zeilen- und Absatzabstand
  • Aufzählungen und Nummerierungen für strukturierte Listen
  • Tabellen einfügen, formatieren und beschriften
  • Seitenränder und -ausrichtung im Layout-Tab einstellen
  • Rechtschreibprüfung mit F7 und Autokorrektur nutzen
  • Format übertragen und Tastenkombinationen sparen Zeit
