03 - Hexadezimalzahlen

INFI - 1AHWII

Wiederholung: Binärzahlen

  • Binärsystem = Basis 2
  • Ziffern: 0 und 1
  • Jede Stelle hat einen Stellenwert (2n)
  • Beispiel: 10112 = 1×2³ + 0×2² + 1×2¹ + 1×2⁰ = 8 + 0 + 2 + 1 = 1110
  • Nachteil: Lange Zahlen, schwer lesbar

Das Hexadezimalsystem

  • Hexadezimalsystem = Basis 16
  • 16 Ziffern: 0–9 und A–F
  • A=10, B=11, C=12, D=13, E=14, F=15
  • Kompakte Darstellung von Binärzahlen
  • 4 Binärziffern entsprechen 1 Hex-Ziffer

Stellenwertsystem

Stelle…3210
Stellenwert…16³16²16¹16⁰
Wert…4096256161
Beispiel: 2A3₁₆ = 2×256 + 10×16 + 3×1
                = 512 + 160 + 3
                = 675₁₀

Hex in Dezimal umrechnen

Jede Hex-Ziffer mit ihrem Stellenwert multiplizieren:

B7F₁₆ = 11×256 + 7×16 + 15×1
      = 2816 + 112 + 15
      = 2943₁₀
1A4₁₆ = 1×256 + 10×16 + 4×1
      = 256 + 160 + 4
      = 420₁₀

Dezimal in Hex umrechnen

Durch 16 dividieren, Reste notieren (von unten nach oben):

420 ÷ 16 = 26 Rest 4 → Ziffer 4
 26 ÷ 16 =  1 Rest 10 → Ziffer A
  1 ÷ 16 =  0 Rest  1 → Ziffer 1

Ergebnis: 420₁₀ = 1A4₁₆

Binär in Hex umrechnen

Binärzahl in 4er-Gruppen teilen, jede Gruppe einzeln umwandeln:

1011 0110₂ = ?
1011₂ = B₁₆
0110₂ = 6₁₆
→ 1011 0110₂ = B6₁₆

Hex in Binär umrechnen

Jede Hex-Ziffer durch 4 Binärziffern ersetzen:

3F₁₆ = ?
3₁₆ = 0011₂
F₁₆ = 1111₂
→ 3F₁₆ = 0011 1111₂
A9₁₆ = ?
A₁₆ = 1010₂
9₁₆ = 1001₂
→ A9₁₆ = 1010 1001₂

Anwendung: RGB-Farbcodes

  • Farben im Web werden hexadezimal angegeben: #RRGGBB
  • Jede Farbe (Rot, Grün, Blau) von 00 bis FF (0–255)
  • #FF0000 = Rot
  • #00FF00 = Grün
  • #0000FF = Blau
  • #FFFFFF = Weiß, #000000 = Schwarz
  • #FFA500 = Orange (FF Rot, A5 Grün, 00 Blau)

Anwendung: Speicheradressen

  • Speicheradressen werden hexadezimal dargestellt
  • Beispiel: 0x7FFD4F8A (32-Bit-Adresse)
  • Kompakte Darstellung großer Zahlenbereiche
  • Jedes Byte = 2 Hex-Ziffern
  • Debugger und Hex-Editoren zeigen Speicher in Hex

Anwendung: MAC-Adressen

  • MAC-Adresse = 6 Bytes, dargestellt in Hex
  • Format: 00:1A:2B:3C:4D:5E
  • Ersten 3 Bytes = Hersteller (OUI)
  • Letzten 3 Bytes = Geräte-ID
  • Beispiel: 00:1A:2B:3C:4D:5E ≠ 00:1A:2B:3C:4D:5F

Anwendung: Weitere Beispiele

  • IPv6-Adressen: fe80::1 (hexadezimale Notation)
  • ASCII-Codes: 0x41 = 'A', 0x42 = 'B'
  • Assembler-Programmierung: Opcodes in Hex
  • Cheat-Codes: Speicherwerte in Hex-Editoren ändern

Zusammenfassung

  • Hexadezimalsystem: Basis 16, Ziffern 0–9 und A–F
  • 1 Hex-Ziffer = 4 Binärziffern
  • Umrechnung: Hex ↔ Dez ↔ Binär
  • Hexadezimale Addition mit Übertrag
  • Wichtige Anwendungen: Farben, Speicher, MAC-Adressen
