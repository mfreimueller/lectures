# Kata 3: Hexadezimale Addition

## Ziel
Hexadezimale Addition durchführen und Übertrag korrekt behandeln.

## Aufgabenstellung
1. Berechnen Sie folgende hexadezimale Additionen:
   - 1A₃₁₆ + 2B₁₆
   - FF₁₆ + 1₁₆
   - A0₁₆ + 60₁₆
2. Berechnen Sie die folgenden Additionen mit mehrstelligen Zahlen:
   - 3F7₁₆ + A2C₁₆
   - FFFF₁₆ + 1₁₆
   - 7A5₁₆ + 8B6₁₆
3. Überprüfen Sie Ihre Ergebnisse, indem Sie alle Zahlen in Dezimal umrechnen und die dezimale Addition durchführen.

## Hinweise
- Addieren Sie stellenweise von rechts nach links.
- Wenn das Ergebnis pro Stelle ≥ 16 ist, bilden Sie einen Übertrag zur nächsten Stelle.
- Der Übertrag wird wie im Dezimalsystem behandelt: Ergebnis − 16, Übertrag + 1.
- Beispiel: F(15) + 3 = 18 → 18−16 = 2, Übertrag 1.
