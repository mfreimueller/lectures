23 - Excel - Diagramme & Datenanalyse

INFI - 1AHWII

Diagrammtypen im Überblick

  • Excel bietet viele Diagrammtypen für unterschiedliche Zwecke
  • Säulendiagramm: Vergleiche zwischen Kategorien
  • Liniendiagramm: Entwicklungen über Zeit (Trends)
  • Kreisdiagramm: Anteile an einem Ganzen (Prozent)
  • Balkendiagramm: Säulendiagramm um 90 Grad gedreht
  • Punktdiagramm (XY): Zusammenhang zweier Zahlenreihen

Diagramme erstellen

  • Datensatz markieren (inkl. Spaltenüberschriften)
  • Menüband → Einfügen → Diagramm auswählen
  • Excel schlägt passende Diagrammtypen vor (Empfohlene Diagramme)
  • Diagramm wird als Objekt im Arbeitsblatt eingefügt
  • Über Diagrammtools (Entwurf & Format) nachbearbeitbar

Diagramme formatieren

  • Achsen: Beschriftung, Skalierung, Einheiten anpassen
  • Titel: Diagramm- und Achsentitel hinzufügen/ändern
  • Legende: Position ändern, ein-/ausblenden
  • Datenbeschriftungen: Werte direkt an den Datenpunkten anzeigen
  • Farben & Stile: Designvorlagen oder manuelle Anpassung
  • Rechtsklick auf jedes Element → Kontextmenü für Formatierung

Säulen- und Balkendiagramme

  • Säulendiagramm: senkrechte Balken, ideal für Vergleiche
  • Balkendiagramm: waagrechte Balken, gut bei langen Kategorienamen
  • Gruppiert: mehrere Reihen nebeneinander
  • Gestapelt: Reihen übereinander → Gesamt und Anteile sichtbar
  • 100%-Gestapelt: Anteile auf 100% normiert

Liniendiagramme und Trendlinien

  • Liniendiagramm zeigt Werteverlauf über Zeit (z. B. Monate, Jahre)
  • Mehrere Datenreihen als separate Linien darstellbar
  • Trendlinie hinzufügen: Diagramm → "+" → Trendlinie
  • Typen: Linear, Exponentiell, Gleitender Mittelwert, ...
  • Formel und R² anzeigen: Trendlinie formatieren → Optionen
  • Prognose: Trendlinie in die Zukunft/reine Vergangenheit verlängern

Kreisdiagramme

  • Zeigen Anteile an einem Ganzen (muss 100 % ergeben)
  • Nur eine Datenreihe möglich
  • Varianten: Kreis (2D/3D), Ring, Kreis aus Kreis
  • Wann vermeiden?:
    • Mehr als 5–7 Kategorien (unübersichtlich)
    • Ähnliche Werte ( kaum erkennbare Unterschiede)
    • Zeitliche Entwicklungen (dafür Liniendiagramm)
  • Alternative: Säulendiagramm ist oft besser lesbar

Kombinationsdiagramme

  • Zwei unterschiedliche Diagrammtypen kombinieren
  • Beispiel: Säulen + Linie (z. B. Umsatz als Säule, Gewinnmarge als Linie)
  • Ermöglicht sekundäre Achse bei unterschiedlichen Größenordnungen
  • Erstellung: Diagramm → Entwurf → Diagrammtyp ändern → Kombination
  • Sekundäre Achse beschriften und formatieren

Pivot-Tabellen – Grundlagen

  • Pivot-Tabelle: interaktive Zusammenfassung großer Datenmengen
  • Daten markieren → Einfügen → Pivot-Tabelle
  • Aufbau: Zeilen, Spalten, Werte, Filter (vier Bereiche)
  • Datenfelder per Drag & Drop in die Bereiche ziehen
  • Werte: Summe, Anzahl, Mittelwert, Min, Max, ...
  • Pivot-Tabelle wird automatisch aktualisiert (Aktualisieren)

Pivot-Tabellen – Gruppieren & Filtern

  • Gruppieren: Zahlen in Bereiche (z. B. 0–100, 101–200)
  • Datumsangaben gruppieren: Nach Jahr, Quartal, Monat, Tag
  • Filtern: Manuell oder mit Slicer (Steuerelement)
  • Slicer: Schaltflächen zum schnellen Filtern (visuell)
  • Pivot-Chart: Diagramm aus Pivot-Tabelle erstellen
  • Tipp: Quell-Daten als Tabelle formatieren (Strg+T) → Pivot wird dynamisch

Bedingte Formatierung – Regeln

  • Start → Bedingte Formatierung → Regeln zum Hervorheben von Zellen
  • Beispiele: Größer als, Kleiner als, Gleich, Text, Datum
  • Farbskalen: 2- oder 3-Farb-Verlauf über Wertebereich
  • Datenbalken: Balkenlänge = Zellenwert (Balkendiagramm in Zellen)
  • Symbolsätze: Pfeile, Ampeln, Sterne als Wertkennzeichnung
  • Mehrere Regeln übereinander anwendbar (Priorität beachten)

Bedingte Formatierung mit Formeln

  • Regel basierend auf Formel für maximale Flexibilität
  • Start → Bedingte Formatierung → Neue Regel → Formel verwenden
  • Beispiel: Ganze Zeile hervorheben:
    =ODE.$A2>100 (Spalte A fix, Zeile relativ)
  • Formel muss einen WAHR/FALSCH-Wert zurückgeben
  • Achtung auf absolute/relative Bezüge ($ für Spalte fixieren)
  • Anwendungen: Schwellwerte, Abweichungen, Duplikate, Flash-Cards

Datenanalyse: Sortieren, Filtern, Teilergebnisse

  • Sortieren: Daten → Sortieren (ein/mehrere Spalten, benutzerdefiniert)
  • Filtern: AutoFilter (Daten → Filter) → Dropdown in Spaltenköpfen
  • Erweiterter Filter: mit Kriterienbereich (Daten → Erweitert)
  • Teilergebnisse: Daten → Teilergebnis
    • Gruppiert Daten und berechnet Zwischensummen
    • Vorher nach der Gruppierungsspalte sortieren!
  • Gliederungsebenen: 1, 2, 3 zum Ein-/Ausklappen

Zusammenfassung

  • Wähle den passenden Diagrammtyp (Säule, Linie, Kreis, Balken, Kombi)
  • Formatiere Achsen, Titel, Legende, Datenbeschriftungen
  • Trendlinien zeigen Entwicklungen und Prognosen
  • Pivot-Tabellen fassen große Datenmengen interaktiv zusammen
  • Bedingte Formatierung visualisiert Daten durch Farben, Balken, Symbole
  • Sortieren, Filtern und Teilergebnisse sind grundlegende Analysetools
