# Kata 3: CPU-Vergleich anhand von Datenblättern

## Ziel
Verschiedene CPUs anhand ihrer technischen Daten vergleichen und für gegebene Einsatzszenarien die passende CPU auswählen können.

## Aufgabenstellung
1. Recherchiere oder nutze die vorgegebenen Daten zu folgenden CPUs:
   - Intel Core Ultra 5 245K
   - Intel Core Ultra 7 265K
   - AMD Ryzen 5 9600X
   - AMD Ryzen 7 9800X3D
2. Erstelle eine Vergleichstabelle mit folgenden Merkmalen:
   - Kerne / Threads
   - Basistakt / Boost-Takt
   - L3-Cache-Größe
   - TDP (Thermal Design Power)
   - Sockel
   - Preis (ca.)
3. Ordne jede CPU einem der folgenden Einsatzszenarien zu und begründe:
   - Gaming-PC (maximale Gaming-Leistung)
   - Büro-PC (gutes Preis-Leistungs-Verhältnis)
   - Workstation für Videobearbeitung
4. Welche CPU würdest du für einen leisen Home-Server empfehlen? Achte besonders auf die TDP.

## Hinweise
- Datenblätter findest du auf den Herstellerseiten (Intel ARK, AMD Produktseiten) oder bei Vergleichsseiten wie Geizhals.
- Die TDP gibt an, wie viel Abwärme das Kühlsystem abführen muss (nicht der Stromverbrauch).
- CPUs mit 3D-V-Cache (AMD X3D) sind besonders für Gaming geeignet.
