# Kata 1: CPU-Komponenten zuordnen

## Ziel
Den internen Aufbau einer CPU verstehen und die Funktionen der einzelnen Komponenten (ALU, CU, Register, Cache) erklären können.

## Aufgabenstellung
1. Zeichne eine vereinfachte Skizze einer CPU und beschrifte die folgenden Komponenten:
   - ALU (Arithmetic Logic Unit)
   - CU (Control Unit)
   - Register (Programm Counter, Instruction Register, Akkumulator)
   - L1-Cache
   - L2-Cache
   - Datenbus
   - Adressbus
2. Beschreibe für jede Komponente ihre Aufgabe in 1–2 Sätzen.
3. Erstelle eine Tabelle, die die verschiedenen Registertypen mit ihrer Funktion auflistet.
4. Erkläre in eigenen Worten den Ablauf eines CPU-Befehlszyklus (Fetch → Decode → Execute).

## Hinweise
- Typische Registergröße: 64 Bit bei aktuellen x86-CPUs.
- Der Programm Counter (PC) zeigt auf die nächste auszuführende Speicheradresse.
- Der Befehlszyklus wiederholt sich millionenfach pro Sekunde.
