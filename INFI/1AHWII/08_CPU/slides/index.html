08 - CPU

INFI - 1AHWII

Was ist eine CPU?

  • CPU = Central Processing Unit (Hauptprozessor)
  • Das "Gehirn" des Computers – führt Programme und Berechnungen aus
  • Besteht aus Millionen bis Milliarden von Transistoren
  • Kommuniziert über den Chipsatz mit RAM und Peripherie

Aufbau der CPU – ALU

  • ALU (Arithmetic Logic Unit) – Rechenwerk
  • Führt arithmetische Operationen aus: +, −, ×, ÷
  • Führt logische Operationen aus: AND, OR, NOT, XOR
  • Vergleichsoperationen: <, >, =
  • Die ALU ist das Herz jeder Berechnung in der CPU

Steuerwerk und Register

  • CU (Control Unit) – Steuerwerk, interpretiert Befehle und steuert Datenfluss
  • Register – kleine, extrem schnelle Speicherzellen in der CPU
  • Wichtige Register: Programm Counter (PC), Instruction Register (IR), Akkumulator
  • Register sind die schnellste Speicherform im gesamten Computer

Cache-Speicher

  • Der Cache ist ein kleiner, schneller Zwischenspeicher in der CPU
  • L1-Cache: kleinste (32–64 KB pro Kern), schnellste
  • L2-Cache: mittelgroß (256–512 KB pro Kern)
  • L3-Cache: größter (8–32 MB), von allen Kernen geteilt
  • Je größer der Cache, desto weniger muss auf den langsamen RAM zugegriffen werden

Von-Neumann-Architektur

  • Grundlegendes Modell für die meisten modernen Computer
  • Vier Hauptkomponenten:
    • Steuerwerk (CU) – Befehlssteuerung
    • Rechenwerk (ALU) – Berechnungen
    • Speicher – RAM, hält Programme und Daten
    • Ein-/Ausgabe – Kommunikation nach außen
  • Prinzip: gemeinsamer Speicher für Code und Daten

Taktrate

  • Die Taktrate (Taktfrequenz) gibt an, wie viele Schwingungen pro Sekunde die CPU ausführt
  • Einheit: Hertz (Hz) – heute Gigahertz (GHz)
  • 1 GHz = 1 Milliarde Takte pro Sekunde
  • Pro Takt kann die CPU einen Befehlsschritt ausführen
  • Höhere Taktrate → mehr Befehle pro Sekunde (bei gleicher Architektur)

Cores und Threads

  • Core (Kern) – ein eigener vollständiger Prozessor auf dem Chip
  • Mehrere Kerne ermöglichen echtes paralleles Arbeiten
  • Typische Core-Zahlen: 2 (Dual-Core), 4 (Quad-Core), 6, 8, 16
  • Thread – ein Ausführungsstrang
  • Hyper-Threading (Intel) / SMT (AMD): 2 Threads pro Kern

CPU-Hersteller: Intel

  • Marktführer im Desktop- und Laptop-Bereich
  • Aktuelle Architekturen: Core (Ultra 5/7/9), Xeon
  • Sockel: LGA1700 (13./14. Gen), LGA1851 (Ultra 200S)
  • Besonderheiten: Hyper-Threading, Turbo Boost, AVX-Befehle
  • Performance-Cores (P-Cores) + Efficiency-Cores (E-Cores) seit Alder Lake

CPU-Hersteller: AMD

  • Stärkster Konkurrent von Intel, besonders im Server-Bereich (EPYC)
  • Aktuelle Architekturen: Ryzen (5/7/9), Threadripper, EPYC
  • Sockel: AM5 (LGA1718, ab Ryzen 7000)
  • Besonderheiten: SMT (Simultaneous Multithreading), 3D-V-Cache
  • Chiplet-Design: mehrere kleinere Chips in einem Package

CPU-Kühlung

  • Eine CPU erzeugt Abwärme – ohne Kühlung wird sie zerstört
  • Luftkühlung: Kühlkörper + Lüfter (Boxed-Kühler, Tower-Kühler)
  • Wasserkühlung: AiO (All-in-One) oder Custom-Loop
  • Wärmeleitpaste zwischen CPU und Kühler verbessert den Wärmeübergang
  • Typische Betriebstemperatur: 40–85 °C (je nach Last und Kühlung)

Zusammenfassung

  • CPU = ALU + Steuerwerk + Register + Cache
  • Von-Neumann-Architektur als Grundlage
  • Taktrate, Cores und Threads bestimmen die Leistung
  • CISC (x86) vs. RISC (ARM) – zwei Philosophien
  • Intel und AMD dominieren den Desktop-Markt
