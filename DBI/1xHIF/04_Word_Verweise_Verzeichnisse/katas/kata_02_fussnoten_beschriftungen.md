# Kata 2: Fußnoten & Beschriftungen

## Ziel

Lerne Fußnoten für wissenschaftliche Arbeiten und Beschriftungen für Bilder mit automatischer Nummerierung.

## Aufgabenstellung

### Teil A: Fußnoten

Fußnoten sind wichtig, um Quellen zu belegen.

1. Neues Word-Dokument
2. Schreibe folgenden Text (oder eigenen):

```
Wissenschaftliches Arbeiten

In der Wissenschaft ist es wichtig, Behauptungen mit Quellen zu belegen.
Man verwendet dazu Fußnoten oder Endnoten.

Laut einer aktuellen Studie ist die Nutzung von Fußnoten im Studium weit verbreitet.
Viele Studierende finden es aber zuerst kompliziert.

Aber mit etwas Übung ist es ganz einfach.
```

3. Füge Fußnoten hinzu:

**Erste Fußnote:**
- Klicke direkt hinter "verbreitet" (im zweiten Absatz)
- Menü: **Referenzen → Fußnote einfügen**
- Word springt automatisch nach unten zur Fußzeile
- Schreibe dort:
  ```
  Quelle: Statistische Auswertung unter 500 Studierenden, HTL Spengergasse, 2026.
  ```

**Zweite Fußnote:**
- Klicke direkt hinter "einfach" (am Ende)
- Wieder: **Referenzen → Fußnote einfügen**
- Schreibe:
  ```
  Tipp: Einfach mal ausprobieren!
  ```

4. Beobachte:
- Im Text siehst du hochgestellte Zahlen: `1` und `2`
- Unten in der Fußzeile stehen die Fußnoten mit denselben Zahlen
- Wenn du Text hinzufügst oder löschst, passen sich die Zahlen automatisch an!

### Teil B: Beschriftungen für Bilder

Wenn du Bilder in eine Arbeit einfügst, brauchen sie Bildunterschriften mit Nummerierung.

**Problem bei manuell:**
- Wenn du "Bild 1:" schreibst, dann "Bild 2:"
- Und dann löschst du Bild 1
- Dann ist Bild 2 immer noch "Bild 2" statt "Bild 1"
- Du musst ALLE von Hand umbenennen!

**Mit Word-Beschriftungen:**
- Automatische Nummerierung
- Nummerierung aktualisiert sich automatisch

#### Schritt 1: Bild einfügen

1. Füge ein Bild ein:
   - Menü: **Einfügen → Bilder → Dieses Gerät** (ein Foto aus deinen Dateien)
   - Oder: **Einfügen → Icons** (ein kostenloses Symbol)
   - Oder: Ein Screenshot

2. Füge noch mindestens ein zweites Bild hinzu.

#### Schritt 2: Beschriftung einfügen

1. Klicke auf das ERSTE Bild, sodass es markiert ist
2. **Rechtsklick → Beschriftung einfügen**
3. Oder: Menü **Referenzen → Beschriftung einfügen**
4. Einstellungen:
   - **Bezeichnung:** `Abbildung` (wählbar aus Liste)
   - **Position:** `Unterhalb des Elements`
   - **Beschriftung:** Gib hier deinen Text ein, z.B. `"Ein schönes Symbol"`
5. OK

Word fügt automatisch hinzu: `Abbildung 1: Ein schönes Symbol`

6. Wiederhole für das ZWEITE Bild:
- Markieren
- Rechtsklick → Beschriftung einfügen
- Beachte: Jetzt steht automatisch `Abbildung 2`!

#### Schritt 3: Automatische Aktualisierung testen

Teste, was passiert, wenn du ein Bild dazwischen einfügst:

1. Klicke zwischen Bild 1 und Bild 2
2. Füge ein DRITTES Bild ein
3. Gib diesem neuen Bild eine Beschriftung
4. Was steht da? `Abbildung 2:`

**ABER:** Das "alte" Bild 2 steht immer noch da mit "Abbildung 2"!
→ Die Beschriftungen müssen manuell aktualisiert werden!

#### Schritt 4: Nummerierung aktualisieren

1. `Strg+A` (Alles im Dokument markieren)
2. Drücke die Taste **`F9`**
3. Oder: Rechtsklick auf eine Beschriftung → **Feld aktualisieren**
4. Wähle: Ganzes Dokument aktualisieren

→ Jetzt sollten alle Nummern richtig sein!

---

### Teil C: Abbildungsverzeichnis (Optional)

Wenn du mehrere Bilder mit Beschriftungen hast, kann Word automatisch ein Abbildungsverzeichnis erstellen (wie Inhaltsverzeichnis, aber für Bilder).

1. Gehe an den Anfang des Dokuments
2. `Strg+Enter` für neue Seite
3. Schreibe: `"Abbildungsverzeichnis"`
4. Menü: **Referenzen → Abbildungsverzeichnis**
5. Wähle eine Vorlage
6. OK

Fertig! Genau wie Inhaltsverzeichnis, aber für Bilder.

---

## Hinweise

### Fußnoten:
- Immer direkt hinter das Wort klicken, das du belegen möchtest
- Nicht hinter ein Leerzeichen klicken!
- Fußnoten werden automatisch neu nummeriert
- Menü: Referenzen → Fußnote einfügen

### Beschriftungen:
- Immer zuerst das Bild markieren!
- `F9` = Alle Felder im Dokument aktualisieren
- Rechtsklick → Feld aktualisieren = Nur dieses Feld aktualisieren
- Abbildungsverzeichnis: Genau wie Inhaltsverzeichnis

## Erweiterung (optional)

### Endnoten statt Fußnoten

Endnoten sind wie Fußnoten, aber sie stehen am ENDE des Dokuments, nicht unten auf der Seite.

Teste:
- Menü: Referenzen → Fußnote einfügen (kleiner Pfeil daneben)
- Wähle: **Endnote**
- Position: Ende des Dokuments

### Literaturverzeichnis

Schaue dir an:
- Referenzen → Zitat einfügen
- Referenzen → Literaturverzeichnis

Für echte wissenschaftliche Arbeiten brauchst du das!
