## 2. Theorie: Seiteneinrichtung (20 min)

### Layout-Register

Menü: **Layout** (bzw. "Seitenlayout" in älteren Versionen)

**Wichtige Funktionen:**

#### Ränder

Menü: Layout → Ränder

- Normal, Schmal, Breit, Gespiegelt (für Broschüren)
- Oder: "Benutzerdefinierte Ränder" für genaue Werte

**Standard für Arbeiten:**
- Oben/Unten: 2,5 cm
- Links/Rechts: 2,5 cm (oder mehr bei Bindung)

#### Papierformat

Menü: Layout → Größe

- A4 (Standard)
- A3, Brief, etc.
- Oder: Benutzerdefinierte Größe

#### Ausrichtung

Menü: Layout → Ausrichtung

- **Hochformat** (Standard)
- **Querformat**

**Wichtig:** Ohne Abschnittswechsel ändert sich das gesamte Dokument!

## 3. Theorie: Kopf-/Fußzeilen & Seitenzahlen (35 min)

### Kopf- und Fußzeilen öffnen

**2 Möglichkeiten:**

1. **Einfügen → Kopfzeile** (oder Fußzeile) → Vordefiniert wählen
2. **Doppelklick** auf den oberen/unteren Seitenrand (oder genau dort, wo Kopfzeile sein soll)

Im Bearbeitungsmodus erscheint das Menü: **Kopf- und Fußzeile**

### Wichtige Optionen im Menü

| Option | Wirkung |
|--------|---------|
| **Verschieden auf erster Seite** | Titelblatt ohne Kopf-/Fußzeile |
| **Verschieden auf geraden & ungeraden Seiten** | Für Broschüren: Innenseiten/Innenseiten |
| **Mit vorherigem verknüpfen** (Button!) | SEHR WICHTIG: Verknüpfung zum vorherigen Abschnitt aufheben |

### Seitenzahlen

Einfügen → Seitenzahlen

**Positionen:**
- Seitenende (unten)
- Seitenanfang (oben)
- Seitenrand

**Formate:**
- Rechtsklick auf Seitenzahl → "Feld bearbeiten"
- Oder: Einfügen → Seitenzahlen → Seitenzahl formatieren

Hier einstellen:
- `1, 2, 3...` (Arabisch)
- `I, II, III...` (Römisch)
- `a, b, c...` (Buchstaben)

**Start bei:**
- Nützlich nach Abschnittswechsel: "1" oder "Fortfahren von vorherigem Abschnitt"

---

### Das wichtigste: Abschnittswechsel!

**Problem:**

Ich möchte:
- Seite 1 (Titelblatt): Ohne Seitenzahl
- Seite 2-4 (Inhaltsverzeichnis): Seitenzahlen I, II, III (römisch)
- Seite 5 ff. (Hauptteil): Seitenzahlen 1, 2, 3... (arabisch)

**Ohne Abschnittswechsel:** Nicht möglich!

**Mit Abschnittswechsel:** Einfach!

#### Demo: Abschnittswechsel erstellen

**Schritt für Schritt:**

1. Cursor an das ENDE von Seite 1 (Titelblatt) setzen
2. Menü: **Layout → Umbrüche → Nächste Seite**
   → Neuer Abschnitt!
3. Cursor an das ENDE von Inhaltsverzeichnis setzen
4. Wieder: **Layout → Umbrüche → Nächste Seite**
   → Noch ein Abschnitt!

Jetzt hast du 3 Abschnitte:
- Abschnitt 1: Titelblatt
- Abschnitt 2: Inhaltsverzeichnis
- Abschnitt 3: Hauptteil

**Jetzt: Seitenzahlen pro Abschnitt einstellen**

**WICHTIGSTER BUTTON:** "Mit vorherigem verknüpfen"

Wenn du in Abschnitt 2 bist und Seitenzahlen änderst...
...ändert sich standardmäßig auch Abschnitt 1!

**Mache zuerst:**
- Gehe in Abschnitt 2
- Menü: Kopf- und Fußzeile → **"Mit vorherigem verknüpfen" klicken (Button geht aus!)**
- Erst jetzt: Seitenzahlen ändern

Dasselbe für Abschnitt 3!

#### Zusammenfassung Abschnittswechsel

| Schritt | Aktion |
|---------|--------|
| 1 | Cursor ans Ende der Seite vor dem neuen Abschnitt |
| 2 | Layout → Umbrüche → Nächste Seite |
| 3 | In den neuen Abschnitt klicken (Kopf-/Fußzeile) |
| 4 | ⚠️ **"Mit vorherigem verknüpfen" DEAKTIVIEREN** ⚠️ |
| 5 | Erst jetzt Seitenzahlen formatieren, Kopfzeile ändern etc. |

## 4. Theorie: Wasserzeichen (5 min)

Menü: **Layout → Wasserzeichen**

Vordefiniert:
- "Entwurf", "Vertraulich", "Nicht kopieren"

Eigenes Wasserzeichen:
- Layout → Wasserzeichen → Benutzerdefiniertes Wasserzeichen
- Text-Wasserzeichen oder Bild-Wasserzeichen
