# Kata 1: Deckblatt gestalten

## Ziel

Lerne ein professionelles Deckblatt für eine Bewerbung zu gestalten.

Ein Deckblatt ist die ERSTE Seite der Bewerbungsmappe. Es ist optional, kann aber sehr professionell wirken.

## Aufgabenstellung

Gestalte ein Deckblatt für deine fiktive Bewerbung als Ferialpraktikant/in.

---

### Was gehört auf ein Deckblatt?

**Das muss drauf sein (Pflicht):**

| Element | Erklärung |
|---------|-----------|
| Titel | "BEWERBUNG" oder genauer "Bewerbung um die Stelle als..." |
| Dein Name | Vor- und Nachname, hervorgehoben |
| Kontaktdaten | Adresse, Telefonnummer, E-Mail |
| Ort und Datum | z.B. "Wien, 12.05.2026" |

**Das kann zusätzlich drauf sein (Optional):**

| Element | Erklärung |
|---------|-----------|
| Foto | Wenn du keins im Lebenslauf hast |
| Stellenbezeichnung | Genau: "als Ferialpraktikant/in im IT-Bereich" |
| Firmenlogo | Wenn bekannt und passend |
| Design/Grafik | Linie, Farbfläche, Symbol |

### Schritt für Schritt Anleitung

#### 1. Neues Dokument & Seitenränder

1. Neues Word-Dokument
2. Menü: **Layout → Seitenränder**
3. Wähle: **Schmal** (oder Normal, wenn du mehr Rand willst)

#### 2. Titel oben

1. Drücke ein paar mal `Enter` um nach unten zu kommen (oder arbeite von oben)
2. Schreibe: `BEWERBUNG`
3. Markiere es
4. Formatiere:
   - Schriftgröße: **28 pt - 36 pt** (groß!)
   - Fett
   - Schriftfarbe: Dunkelblau oder schwarz
   - Zentriert: Start → **Zentrieren**

#### 3. Stellenbezeichnung

1. `Enter` drücken
2. Schreibe: `als Ferialpraktikant/in im IT-Bereich`
3. Formatiere:
   - Kleiner als der Titel (z.B. 14 pt)
   - Evtl. nicht fett, oder etwas heller
   - Zentriert

#### 4. Platz nach unten

Drücke mehrmals `Enter` um Platz zu schaffen.

Tipp: Statt viele `Enter` kannst du auch arbeiten mit:
- Absatzabstand: Rechtsklick → Absatz → Abstand davor/nachher
- Oder: Tabellen/Textfelder für exakte Positionierung

#### 5. Deine Daten

Schreibe unterhalb oder links/rechts:

```
Max Mustermann
Hauptstraße 1
1010 Wien

Tel.: 0664/1234567
E-Mail: max.mustermann@email.at
```

#### 6. Ort und Datum

Ganz unten:

```
Wien, 12.05.2026
```

**Trennlinie einfügen:**
1. Menü: **Einfügen → Formen → Linie**
2. Linie ziehen
3. Form formatieren → Kontur → Farbe wählen, Stärke wählen

---

### Variante 3: Mit Farbfläche (für Fortgeschrittene)

Eine Farbfläche als Hintergrund:

1. Menü: **Einfügen → Formen → Rechteck**
2. Ziehe das Rechteck über die GANZE Seite
3. Form formatieren:
   - **Füllen:** Farbe wählen (z.B. hellblau oder dunkelblau)
   - **Kontur:** Keine Kontur
4. Form formatieren → **Anordnung → Hinter Text**
5. Jetzt kannst du weißen Text darüber schreiben!

---

### Styling-Regeln für ein gutes Deckblatt

| Regel | Erklärung |
|-------|----------|
| **Max. 2 Schriftarten** | Eine für Überschriften, eine für normalen Text. Nicht mehr! |
| **Max. 2-3 Farben** | Z.B. Dunkelblau + Schwarz + Weiß. Kein Regenbogen! |
| **Lesbarkeit geht vor Design** | Text muss man lesen können. Dunkler Text auf hellem Grund. |
| **Saubere Ausrichtung** | Alles entweder links, rechts oder zentriert. Nicht wild mischen. |
| **Weniger ist mehr** | Nicht überladen mit zu vielen Elementen. |

---

### Elemente zum Ausprobieren

| Element | Wie in Word |
|---------|--------------|
| **Fette Überschrift** | Schriftgröße 28-36 pt, Fett |
| **Trennlinie** | Einfügen → Formen → Linie |
| **Farbige Fläche** | Rechteck → Füllen → "Hinter Text" |
| **Rahmen um Text** | Start → Ränder → Außenränder |
| **Icons/Symbole** | Einfügen → Icons |
| **Kleines Logo/Zeichen** | Einfügen → Bilder → Aus Datei |

---

## Hinweise

- Ein Deckblatt ist **optional**. Viele Ferialpraktika-Bewerbungen funktionieren auch ohne.
- Wenn du es machst: Mache es **sauber und professionell**.
- Lieber **einfach und clean** als "überdesigned".
- **Konsistenz:** Wenn dein Deckblatt dunkelblau ist, sollte der Rest der Bewerbung auch dunkelblaue Elemente haben.
