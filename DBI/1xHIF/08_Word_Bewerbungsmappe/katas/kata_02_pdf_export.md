# Kata 2: PDF-Export & Final-Check

## Ziel

Lerne deine Bewerbung als PDF zu exportieren und einen finalen Check durchzuführen.

**Wichtig:** Zum Verschicken einer Bewerbung IMMER PDF verwenden, nie DOCX!

| DOCX | PDF |
|------|-----|
| Man kann es bearbeiten | Nicht so einfach änderbar |
| Format kann kaputt gehen | Format bleibt IMMER erhalten |
| Braucht Word zum Öffnen | Jeder kann PDF öffnen |

---

## Aufgabenstellung

Nimm deinen Lebenslauf und dein Bewerbungsschreiben (falls vorhanden) und gehe die komplette Checkliste durch.

---

### Schritt 1: Rechtschreibprüfung

**Für JEDES Dokument einzeln:**

1. Öffne das Dokument
2. Drücke **`F7`** oder:
   - Menü: **Überprüfen → Rechtschreibung und Grammatik**
3. Korrigiere alle gefundenen Fehler

**WICHTIG:**
- Die Rechtschreibprüfung findet **nicht** alle Fehler!
- Lese den Text danach **selbst** noch einmal durch.
- Besonders Namen und Adressen prüfen!

---

### Schritt 2: Layout prüfen

Gehe diese Checkliste durch:

- [ ] In ALLEN Dokumenten wird die **gleiche Schriftart** verwendet?
- [ ] Keine "wild" formatierten Bereiche?
- [ ] Konsistente Schriftgrößen?
- [ ] **Keine leeren Seiten** am Ende des Dokuments?
- [ ] Seitenzahlen (falls verwendet) stimmen?
- [ ] Die Anrede passt?
  - "Sehr geehrter Herr Müller," (richtig für männlich)
  - "Sehr geehrte Frau Schmidt," (richtig für weiblich)
  - "Sehr geehrte Damen und Herren," (wenn Ansprechpartner unbekannt)
- [ ] Das Datum ist **aktuell**? (Nicht von vor 3 Monaten!)

---

### Schritt 3: Optional: Alle Dokumente in EINEM Word-Dokument zusammenfügen

Wenn alles in **einer PDF** sein soll:

#### Variante A: Text aus Datei einfügen

1. Öffne das Dokument, das als **ERSTES** stehen soll (z.B. Deckblatt oder Anschreiben)
2. Gehe ans **ENDE** dieses Dokuments
3. Füge einen **Seitenumbruch** für einen neuen Abschnitt ein:
   - Menü: **Layout → Umbrüche → Nächste Seite**
4. Menü: **Einfügen → Objekt → Text aus Datei...**
5. Wähle das nächste Dokument aus (z.B. Lebenslauf)
6. Wiederhole bis alle Dokumente eingefügt sind
7. Speichere unter einem neuen Namen: `Bewerbung_Komplett.docx`

#### Variante B: Später im PDF zusammenfügen

Du kannst auch jedes Dokument **einzeln als PDF exportieren** und dann später mit einem PDF-Tool zusammenfügen.

Tools zum Zusammenfügen von PDFs:
- Adobe Acrobat Reader (kostenpflichtig für diese Funktion)
- Verschiedene Online-Tools (z.B. Smallpdf, ILovePDF)
- macOS: Vorschau.app kann PDFs zusammenfügen

---

### Schritt 4: Bilder komprimieren (wenn Bilder im Dokument sind)

Wenn du Fotos, Logos oder andere Bilder im Dokument hast, kann die PDF sehr groß werden.

**Vor dem Export komprimieren:**

1. Klicke auf **irgendein Bild** im Dokument
2. Menü: **Grafikformat** (erscheint nur, wenn Bild markiert ist)
3. Button: **Bilder komprimieren**
4. Einstellungen im Dialog:
   - ❌ **Haken entfernen** bei: "Nur dieses Bild anwenden"
   - → Damit werden **ALLE** Bilder im Dokument komprimiert!
   - Auflösung wählen: **E-Mail (96 ppi)** (reicht für Bildschirm und Druck)
5. Klicke **OK**

Jetzt ist das Dokument viel kleiner!

---

### Schritt 5: PDF exportieren

1. Menü: **Datei**
2. Klicke: **Exportieren**
3. Klicke: **PDF/XPS-Dokument erstellen**
4. Button: **PDF/XPS erstellen**

5. Im Dialog "Als PDF/XPS speichern":
   - Wähle den Speicherort aus
   - **WICHTIG: Gib einen SINNVOLLEN Dateinamen ein!**

     | ❌ Schlecht | ✅ Gut |
     |-------------|--------|
     | `Bewerbung.pdf` | `Bewerbung_Max_Mustermann_Ferialpraktikum.pdf` |
     | `doc1.pdf` | `Lebenslauf_Anna_Beispieldame_2026.pdf` |
     | `FINAL.pdf` | `Anschreiben_Sarah_Schmidt.pdf` |
     | `Neues Dokument.pdf` | |

   - **Optimieren für:**
     - Wenn kleine Datei per E-Mail: **"Mindestgröße (Per E-Mail senden)"**
     - Wenn gute Qualität für Druck: **"Standard (Online-Veröffentlichung)"**

6. Klicke auf den Button: **Optionen...** (ist ganz wichtig!)

   Im Dialog "Optionen":

   | Einstellung | Wert |
   |-------------|------|
   | Seiten | Alle |
   | Kommentare einfügen | ❌ Nein (Haken entfernen) |
   | Dokumentstruktur-Tags für Barrierefreiheit | ✅ Ja (Haken setzen) |
   | Bitmap-Text wenn Schriftarten nicht eingebettet | ✅ Ja |

7. Klicke **OK**
8. Klicke **Veröffentlichen**

---

### Schritt 6: PDF öffnen und PRÜFEN!

**Diesen Schritt niemals überspringen!**

Nach dem Export:

1. Gehe zum Speicherort der PDF
2. Doppelklick zum Öffnen
3. Blättere durch **ALLE Seiten** durch
4. Prüfe diese Punkte:

- [ ] Ist das **Layout** noch in Ordnung?
- [ ] Sind **alle Seiten** vorhanden?
- [ ] Sind **Bilder** sichtbar?
- [ ] Sind keine **Texte "abgeschnitten"** oder umgebrochen?
- [ ] Gibt es **leere Seiten** am Ende?
- [ ] Sind **Seitenzahlen** (falls vorhanden) korrekt?

**Wenn etwas nicht stimmt:**
1. Zurück zum Word-Dokument
2. Fehler beheben
3. **Neu exportieren!**
4. **Nochmal prüfen!**

---

### Schritt 7: Finale Prüfung des Dateinamens

Letzter Check:

| ✅ Gute Beispiele |
|-------------------|
| `Bewerbung_Max_Mustermann_Ferialpraktikum_IT.pdf` |
| `Lebenslauf_Anna_Beispieldame_2026.pdf` |
| `Anschreiben_Sarah_Schmidt_TechSolutions.pdf` |

| ❌ Schlechte Beispiele |
|-------------------------|
| `Bewerbung.pdf` |
| `Neues Dokument.pdf` |
| `doc1.pdf` |
| `FINAL.pdf` |
| `FINAL_FINAL.pdf` |
| `FINAL_FINAL_WIRKLICH_ENDGUELTIG.pdf` |

## Bonus: Wenn du die Bewerbung per E-Mail schickst

### Betreff der E-Mail

**GUT:**
```
Bewerbung Ferialpraktikant/in IT - Max Mustermann
```

**SCHLECHT:**
```
Bewerbung
```
```
hallo
```
```
blabla
```

### Inhalt der E-Mail

Das Bewerbungsschreiben steht im **PDF-Anhang**, nicht in der E-Mail selbst!

Die E-Mail selbst kann kurz sein:

```
Sehr geehrte Frau Schmidt,

anbei erhalten Sie meine Bewerbungsunterlagen für die ausgeschriebene Stelle als Ferialpraktikant/in im IT-Bereich.

Ich freue mich auf Ihre Rückmeldung.

Mit freundlichen Grüßen

Max Mustermann
Tel.: 0664/1234567
```

### Anhang

- Die PDF-Datei anhängen
- **NICHT** die DOCX-Datei anhängen!

---

## Zusammenfassung Final-Check

- [ ] Rechtschreibprüfung über **ALLE** Dokumente
- [ ] Danach nochmal **selbst lesen**!
- [ ] Layout konsistent?
- [ ] Keine leeren Seiten?
- [ ] Bilder komprimieren (falls vorhanden)
- [ ] Als **PDF** exportieren
- [ ] **Dateiname ist sinnvoll**
- [ ] PDF nach dem Export **öffnen und prüfen**!
- [ ] Bei E-Mail: **Betreff ist sinnvoll**

**Erst wenn alle Punkte geprüft sind: Verschicken!**
