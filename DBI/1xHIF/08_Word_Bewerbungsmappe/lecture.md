## 2. Theorie: Komplette Bewerbungsmappe (20 min)

### Was ist eine "Bewerbungsmappe"?

Alle Dokumente, die zu einer Bewerbung gehören:

**Bei PAPER-Bewerbung:**
- Alles in einem Ordner / einer Mappe
- Oft mit Trennblättern
- Unterschrift auf dem Anschreiben

**Bei E-MAIL-Bewerbung:**
- Alles in EINER oder MEHREREN PDF-Dateien
- Keine physische Mappe mehr
- Aber: Struktur beachten!

---

### Struktur: Welche Reihenfolge?

#### Standard-Reihenfolge:

1. **Deckblatt** (Optional)
2. **Anschreiben** (Bewerbungsschreiben)
3. **Lebenslauf**
4. **Anlagen / Zeugnisse**

---

#### Variante 1: Einfache Bewerbung (ohne Deckblatt)

```
┌─────────────────────────────────────┐
│  Seite 1: Anschreiben                │
│     - Absender                       │
│     - Empfänger                      │
│     - Betreff                        │
│     - Anrede                         │
│     - Text (4 Absätze)               │
│     - Unterschrift                   │
├─────────────────────────────────────┤
│  Seite 2: Lebenslauf                 │
│     - Persönliche Daten              │
│     - Bild (optional)                │
│     - Bildungsweg                    │
│     - Praktika                       │
│     - Kenntnisse                     │
│     - Unterschrift                   │
├─────────────────────────────────────┤
│  Seite 3+: Anlagen (Zeugnisse etc.) │
└─────────────────────────────────────┘
```

---

#### Variante 2: Mit Deckblatt (professionell)

**Was ist ein Deckblatt?**

Die ERSTE Seite der Bewerbung. Darauf steht:

```
┌─────────────────────────────────────────┐
│                                         │
│                                         │
│          BEWERBUNG                       │
│                                         │
│   als Ferialpraktikant/in im IT-Bereich │
│                                         │
│                                         │
│   Max Mustermann                        │
│   Hauptstraße 1                         │
│   1010 Wien                             │
│                                         │
│   Tel.: 0664/1234567                   │
│   E-Mail: max@mustermann.at             │
│                                         │
│                                         │
│   Wien, 12.05.2026                      │
│                                         │
│      (Optional: Foto / Grafik)          │
│                                         │
└─────────────────────────────────────────┘
```

**Vorteil Deckblatt:**
- Professional
- Übersichtlich
- Platz für Grafik/Foto (wenn man es nicht im Lebenslauf haben möchte)

**Nachteil Deckblatt:**
- Eine zusätzliche Seite
- Nicht unbedingt notwendig

**Tipp:** Bei HTL-Schülern für Ferialpraktika oft OHNE Deckblatt ausreichend.
**Aber:** Bei "richtigen" Jobs nach der HTL durchaus üblich.

## 3. Theorie: PDF-Export (15 min)

### Warum PDF und nicht DOCX?

| DOCX | PDF |
|------|-----|
| Jeder, der Word hat, kann es öffnen | JEDER kann es öffnen (auch ohne Word!) |
| Format kann "kaputt gehen" bei anderer Word-Version | Format bleibt IMMER erhalten! |
| Man kann es bearbeiten | Kann nicht so einfach bearbeitet werden |
| Zum Arbeiten | Zum VERSENDEN! |

**REGEL:** Zum Verschicken IMMER PDF!

---

### PDF-Export in Word: Schritt für Schritt

1. Menü: **Datei**
2. Klicke: **Exportieren**
3. Klicke: **PDF/XPS-Dokument erstellen**
4. Button: **PDF/XPS erstellen**
5. Dialog:
   - Speicherort wählen
   - **Dateiname: Sinnvoll wählen!** (z.B. `Bewerbung_Max_Mustermann.pdf`)
6. Button: **Optionen...** (ganz wichtig!)

**Optionen-Dialog:**

| Einstellung | Was es bedeutet | Empfehlung |
|-------------|-----------------|------------|
| Seiten: Alle | Welche Seiten exportieren? | Alle |
| Kommentare einfügen | Falls Kommentare im Text | Nein |
| Dokumentstruktur-Tags | Für Barrierefreiheit/Screenreader | Ja |
| Bitmap-Text | Falls Schriftarten fehlen | Ja |
| ISO 19005-1 (PDF/A) konform | Für Langzeitarchivierung | Nur wenn verlangt |

7. OK
8. Veröffentlichen

---

### Namenskonvention für PDFs

**Sehr wichtig!**

Stell dir vor, du arbeitest in der Personalabteilung und bekommst 50 Bewerbungen.

Welcher Dateiname ist hilfreich?

| ❌ Schlecht | ✅ Gut |
|-------------|--------|
| `Bewerbung.pdf` | `Bewerbung_Max_Mustermann_Ferialpraktikum.pdf` |
| `Lebenslauf.pdf` | `Lebenslauf_Anna_Beispieldame.pdf` |
| `Neues Dokument.pdf` | `Anschreiben_Sarah_Schmidt.pdf` |
| `FINAL.pdf` | `Bewerbung_2026_HTL_Spengergasse.pdf` |
| `FINAL_FINAL.pdf` | |
| `doc1.pdf` | |

**Regel für guten Dateinamen:**
`[Art]_[Vorname]_[Nachname]_[Optional: Stelle/Jahr].pdf`

Oder alles in einer PDF:
`Bewerbung_[Vorname]_[Nachname].pdf`

---

### Komprimieren vor dem Export

Wenn Bilder sehr groß sind, wird die PDF sehr groß.

**Vorher komprimieren:**

1. Ein Bild im Dokument markieren
2. Menü: **Grafikformat**
3. Button: **Bilder komprimieren**
4. Einstellungen:
   - Haken entfernen bei: **Nur dieses Bild anwenden**
   - Auflösung: **E-Mail (96 ppi)** oder **Web (150 ppi)**
5. OK

Jetzt ist das gesamte Dokument kleiner und die PDF wird kleiner.

## Final-Check Liste

Bevor eine Bewerbung verschickt wird:

- [ ] Rechtschreibprüfung über ALLE Dokumente laufen lassen
- [ ] Keine Fehler im Text
- [ ] Konsistente Schriftart (nicht 5 verschiedene!)
- [ ] Passt die Anrede? (Herr/Frau/Damen und Herren)
- [ ] PDF öffnen und durchblättern
- [ ] Keine leeren Seiten am Ende
- [ ] Dateiname ist sinnvoll
- [ - ] Wenn E-Mail: Betreff der E-Mail ist sinnvoll
