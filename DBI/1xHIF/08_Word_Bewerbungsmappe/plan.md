## Aufbau einer vollständigen Bewerbung

### Variante A: Ohne Deckblatt (einfach)

1. **Anschreiben** (1 Seite)
   - Mit Absender, Empfänger, Betreff, Anrede, Text, Unterschrift

2. **Lebenslauf** (1-2 Seiten)
   - Persönliche Daten, Bild (optional), Bildungsweg, Praktika, Kenntnisse

3. **Anlagen / Zeugnisse**
   - Schulzeugnisse (als Kopien)
   - Praktikumszeugnisse
   - Zertifikate
   - Als letzte Seiten in der PDF

### Variante B: Mit Deckblatt (professionell)

1. **Deckblatt** (1 Seite)
   - Titel: "Bewerbung"
   - Um welche Stelle?
   - Name, Kontaktdaten
   - Optional: Foto / Grafik

2. **Anschreiben** (1 Seite)

3. **Lebenslauf** (1-2 Seiten)

4. **Anlagen / Zeugnisse**

## Namenskonvention für PDFs

### Gute Beispiele:

```
Bewerbung_Max_Mustermann_Ferialpraktikum.pdf
Lebenslauf_Anna_Beispieldame.pdf
Anschreiben_Sarah_Schmidt_2026.pdf
```

### Schlechte Beispiele:

```
Bewerbung.pdf
Neues Dokument.pdf
doc1.pdf
final.pdf
FINAL_FINAL.pdf
```

**Regel:** Firmenname + Vorname Nachname + Stellennamen (oder Art des Dokuments)

## PDF-Export in Word

Datei → Exportieren → PDF/XPS-Dokument erstellen

**Wichtige Einstellungen:**

| Einstellung | Empfehlung |
|-------------|------------|
| Standard (Online-Veröffentlichung) | Gute Qualität, etwas größere Datei |
| Mindestgröße (Per E-Mail senden) | Kleine Datei, evtl. schlechtere Bildqualität |
| **Optionen...** → | |
| Seitenbereich | Alle |
| Kommentieren einfügen | Nein |
| Dokumentstruktur-Tags für Barrierefreiheit | Ja (empfohlen) |
| Bitmap-Text wenn Schriftarten nicht eingebettet werden können | Ja |

**Tipp:** Wenn Bilder sehr groß sind:
1. Bild markieren
2. Menü: Grafikformat → Komprimieren
3. Alle Bilder komprimieren
4. Dann erst PDF exportieren

## Final-Check vor dem Verschicken

### Check-Liste:

- [ ] Rechtschreibprüfung über alle Dokumente laufen lassen (`F7`)
- [ ] Alle Seitenzahlen (falls vorhanden) passen
- [ ] Konsistente Schriftart im ganzen Dokument
- [ ] Keine leeren Seiten am Ende
- [ ] PDF öffnen und nochmal durchblättern
- [ ] Dateiname ist sinnvoll
- [ ] PDF kann von jedem geöffnet werden (Test: An einem anderen Computer öffnen)
