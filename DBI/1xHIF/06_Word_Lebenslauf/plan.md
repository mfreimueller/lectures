## Wichtige Konzepte

### Aufbau Lebenslauf (klassisch)

| Bereich | Inhalt |
|---------|--------|
| **Persönliche Daten** | Name, Adresse, Telefon, E-Mail, Geburtsdatum/-ort (optional), Staatsbürgerschaft (optional) |
| **Foto** | Optional in AT, professionell (kein Selfie!), neutraler Hintergrund |
| **Bildungsweg** | Aktuellstes zuerst! Schulen mit Zeiträumen (Monat/Jahr - Monat/Jahr) |
| **Praktika/Berufserfahrung** | Auch Ferialpraktika! Was wurde gemacht? |
| **Kenntnisse/Fähigkeiten** | Sprachen, EDV, Führerschein, Soft Skills |
| **Interessen/Hobbies** | Kurz, aussagekräftig (nicht zu privat) |
| **Ort, Datum, Unterschrift** | Wichtig! |

### Dos & Don'ts

| ✅ Do | ❌ Don't |
|-------|----------|
| Aktuell bis heute/Monat | Veraltete Daten |
| Tabellarisch oder modern | Zu viel Text in Fließtext |
| 1 Seite (für Einsteiger) | Mehr als 2 Seiten |
| Rechtschreibprüfung | Fehler im Text |
| Ehrlich | Übertreibungen/Lügen |
| Einheitliche Schriftart | 5 verschiedene Schriftarten |
| Professionelles Foto (wenn) | Selfie, Party-Foto |
