## 2. Theorie: Tabellen (25 min)

### Tabelle erstellen

**Möglichkeiten:**
1. Menü: Einfügen → Tabelle → Raster ziehen
2. Oder: Tabelle einfügen → Zahlen eingeben
3. Oder: "Tabelle zeichnen" (für komplexere Tabellen)

**Live-Demo:** 5x3 Tabelle erstellen

### Tabellen bearbeiten

**Zeile/Spalte hinzufügen:**
- Rechtsklick in Tabelle → Einfügen
- Oder: Tabellenentwurf → "Neue Zeile unterhalb"

**Tipp:** `Tab` in letzter Zelle = Neue Zeile!

### Zellen zusammenführen

1. Mehrere Zellen markieren
2. Rechtsklick → Zellen verbinden
3. Oder: Tabellenentwurf → Zellen verbinden

### Tabellen formatieren

**Designs (schnell):**
- Menü: Tabellenentwurf → Tabellenformatvorlagen
- Vordefinierte Designs mit einem Klick

**Manuell (detailliert):**
- Rahmen: Tabellenentwurf → Rahmen
- Schattierung: Tabellenentwurf → Schattierung
- Ausrichtung innerhalb Zelle: Layout → Ausrichtung

### Tabellen-Einstellungen

Rechtsklick → Tabelleneigenschaften:
- Tabellenbreite festlegen
- Zeilenhöhe wiederholen auf jeder Seite (für Kopfzeilen)
- Textumfluss um die Tabelle

## 3. Theorie: Grafiken & SmartArt (25 min)

### Bilder einfügen

**Möglichkeiten:**
1. Menü: Einfügen → Bilder → Dieses Gerät
2. Oder: Online-Bilder (Stock-Fotos)
3. Oder: Screenshot

### Bild formatieren

Nachdem Bild markiert ist, erscheint Menü: **Grafikformat**

**Wichtige Funktionen:**
- **Zuschneiden:** Bild bescheiden, nicht löschen!
- **Position:** Textumfluss!
- **Ausrichten:** Links, Zentriert, Rechts
- **Korrekturen:** Helligkeit, Kontrast, Schärfe
- **Farben:** Sättigung, Farbton

### Textumfluss - das Wichtigste!

**Live-Demo mit verschiedenen Optionen:**

1. **Mit Text in Zeile** (Standard):
   - Bild wie ein Zeichen im Text
   - Bewegt sich mit dem Text

2. **Quadrat:**
   - Text umfließt das Bild rechteckig
   - Am häufigsten verwendet

3. **Eng:**
   - Text folgt der Kontur (bei freigestellten Bildern)

4. **Hinter Text:**
   - Bild im Hintergrund
   - Gut für Wasserzeichen/Hintergründe

5. **Vor Text:**
   - Bild überdeckt den Text
   - Vorsicht!

### SmartArt

Menü: Einfügen → SmartArt

**Einsatzzwecke:**
- **Liste:** Auflistungen visuell darstellen
- **Prozess:** Abläufe, Schritte zeigen
- **Hierarchie:** Organigramme, Strukturen
- **Zusammenhang:** Beziehungen visualisieren

**Demo:** Einfachen Prozess mit 3 Schritten erstellen

Kata-Blätter zeigen. Wichtige Hinweise:
- "Zusammenführen" für die großen Zellen
- Tabellen-Design verwenden für schöne Optik

## 5. Kata 2: Bild mit Textumfluss (20 min)

Kata-Blätter zeigen. Hinweise:
- Einfaches Bild verwenden (z.B. eigenes Foto oder Screenshot)
- Textumfluss: Ausprobieren, was am besten aussieht
- Positionierung: "Mit Textanker fixieren" vs. "Auf Seite positionieren"

| Konzept | Erklärung |
