## Wichtige Konzepte

### Tabellen

| Funktion | Menüpfad |
|----------|-----------|
| Tabelle einfügen | Einfügen → Tabelle |
| Zeile/Spalte einfügen | Tabellenentwurf (rechtsklicken) |
| Zellen zusammenführen | Rechtsklick → Zellen verbinden |
| Tabellen-Design | Tabellenentwurf → Formatvorlagen |

### Textumfluss-Arten

| Typ | Beschreibung |
|-----|--------------|
| Mit Text in Zeile | Bild verhält sich wie ein Zeichen |
| Quadrat | Text umfließt rechteckig herum |
| Eng | Text umfließt nah an der Kontur |
| Hinter Text | Bild im Hintergrund |
| Vor Text | Bild deckt Text ab |
