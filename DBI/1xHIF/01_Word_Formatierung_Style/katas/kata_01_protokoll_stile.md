# Kata 1: Protokoll mit Stilen formatieren

## Ziel

Lerne die integrierten Word-Stile (Überschrift 1/2/3, Standard) anzuwenden und verstehe den Vorteil von Stilen gegenüber direkter Formatierung.

## Aufgabenstellung

Ein Schulprotokoll soll mit den integrierten Word-Stilen konsistent formatiert werden.

### Schritt 1: Text kopieren

Kopiere folgenden Text und füge ihn in ein neues Word-Dokument ein:

```
Protokoll Klassenvorstandssitzung

Ort: Klassenzimmer 3.21
Datum: 12.05.2026
Anwesend: Schülervertreter, Lehrervertreter
Protokollführer: Max M.

1. Tagesordnung

1.1 Begrüßung
Der Klassenvorstand begrüßt alle Anwesenden. Es wird auf die Pünktlichkeit hingewiesen.

1.2 Protokoll letzte Sitzung
Das Protokoll der letzten Sitzung vom 28.04.2026 wurde genehmigt.

1.3 Neue Themen
1.3.1 Fahrtenwoche
Es wird über die Fahrtenwoche im Juni diskutiert. Kostenpunkt: ca. 150 Euro pro Schüler/in.

1.3.2 Projektwoche
Die Projektwoche findet im Herbst statt. Themenvorschläge werden gesammelt.

2. Sonstiges
Nächste Sitzung: 09.06.2026

Ende des Protokolls
```

### Schritt 2: Stile anwenden

Formatiere das Dokument mit folgenden Stilen:

| Text | Stil | Tastenkürzel |
|------|------|----------------|
| "Protokoll Klassenvorstandssitzung" | Überschrift 1 | Strg+Alt+1 |
| "1. Tagesordnung" | Überschrift 1 | Strg+Alt+1 |
| "2. Sonstiges" | Überschrift 1 | Strg+Alt+1 |
| "1.1 Begrüßung" | Überschrift 2 | Strg+Alt+2 |
| "1.2 Protokoll letzte Sitzung" | Überschrift 2 | Strg+Alt+2 |
| "1.3 Neue Themen" | Überschrift 2 | Strg+Alt+2 |
| "1.3.1 Fahrtenwoche" | Überschrift 3 | Strg+Alt+3 |
| "1.3.2 Projektwoche" | Überschrift 3 | Strg+Alt+3 |
| Alle anderen Texte | Standard | Strg+Umsch+N |

### Schritt 3: Navigation testen

Gehe zu:
- Menü: Ansicht → Gliederung
- Oder: Datei → Suchen ( öffnet Navigation)

Du solltest nun eine Übersicht über alle Überschriften sehen.

## Hinweise

- Stile findest du auch im Menü "Start" → Bereich "Stile"
- Falls du direkt formatiert hast, zerstörst du den Stil → mit Strg+Leertaste zurücksetzen!
- Die Überschriften-Ebenen sind wichtig für nächste Woche (Inhaltsverzeichnis)

## Erweiterung (optional)

Ändere den Stil "Überschrift 1":

1. Rechtsklick auf "Überschrift 1" im Stil-Menü → Ändern
2. Passe folgende Eigenschaften an:
   - Schriftart: Cambria (oder eine andere deiner Wahl)
   - Schriftfarbe: Dunkelblau
   - Größe: 14 pt

Beobachte: Alle Überschriften ändern sich automatisch!
