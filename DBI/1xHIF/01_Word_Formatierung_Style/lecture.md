## 2. Theorie: Stile vs. Direkte Formatierung (20 min)

### Was sind Stile?

| Art der Formatierung | Beschreibung | Nachteil |
|----------------------|--------------|----------|
| **Direkt** | Text markieren → Fett → Groß → Blau | Änderungen mühsam, inkonsistent |
| **Über Stil** | Text markieren → Stil "Überschrift 1" anwenden | Keine! Einmal Stil ändern = Alles ändert sich |

### Vorteile von Stilen:

1. **Konsistenz:** Alle Überschriften sehen gleich aus
2. **Schnelle Änderung:** Einmal Stil bearbeiten = überall ändern
3. **Automatische Verzeichnisse:** Später wichtig für Inhaltsverzeichnis
4. **Navigation:** Dokument über Überschriften durchsuchen

### Live-Demo: Integrierte Stile

1. Leeres Dokument öffnen
2. Mehrere Zeilen schreiben:

```
Überschrift 1
Ein bisschen Text...

Überschrift 2
Noch mehr Text...

Überschrift 2
Und weiter...

Überschrift 1
Ein neuer Abschnitt
```

3. Jede "Überschrift 1" → `Strg+Alt+1` (Tastenkürzel)
4. Jede "Überschrift 2" → `Strg+Alt+2`

5. Jetzt: **Rechtsklick auf Stil "Überschrift 1" → Ändern**
   - Schriftart ändern
   - Farbe ändern
   - → Alle Überschriften ändern sich automatisch!

### Wichtige Tastenkürzel:

| Kürzel | Aktion |
|--------|--------|
| `Strg+Alt+1` | Stil "Überschrift 1" |
| `Strg+Alt+2` | Stil "Überschrift 2" |
| `Strg+Alt+3` | Stil "Überschrift 3" |
| `Strg+Umsch+N` | Stil "Standard" (Fließtext) |

## 3. Theorie: Formatvorlagen & Design (25 min)

### Stil anpassen vs. Neuen Stil erstellen

**Szenario 1: Integrierten Stil anpassen**
- Rechtsklick auf "Überschrift 1" → Ändern
- Oder: Menü "Start" → Rechtsklick auf Stil → Ändern

**Szenario 2: Eigenen Stil erstellen**

Demo: "Mein Zitat"-Stil

1. Text schreiben: `"Ein Zitat aus einem Buch."`
2. Formatieren: Kursiv, Einrückung links 1cm, Farbe Grau
3. Markieren → Start → Stile → Auswahl speichern als neue Schnellformatvorlage
4. Name: "Zitat"
5. Fertig!

### Design-Register

Menü: `Design` (bzw. `Entwurf` in manchen Word-Versionen)

Zeigen:
- **Designs:** Vordefinierte Pakete (Schriftart + Farben + Abstände)
- **Farben:** Farbschema ändern
- **Schriftarten:** Schriftart-Paket ändern
- **Absatzabstand:** Vordefinierte Abstände

**Demo:** Ein Klick auf anderes Design → Komplettes Dokument ändert sich!

### Formatierung löschen

Wichtig: Wenn ein Stil "zerbrochen" ist (direkte Formatierung hinzugefügt):

1. Text markieren
2. `Strg+Leertaste` (direkte Formatierung entfernen, Stil bleibt)
3. Oder: Start → `Alle Formatierung löschen` (Button mit Radiergummi-Symbol)
