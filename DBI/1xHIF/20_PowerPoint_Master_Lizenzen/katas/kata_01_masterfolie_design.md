# Kata 1: Masterfolien-Design erstellen

## Ziel

Lerne, wie du in PowerPoint ein eigenes, einheitliches Design für alle Folien erstellst — mit Hintergrundfarbe, eigener Schriftart und einem Logo.

---

## Der wichtigste Tipp vorweg

Alles, was du auf der **Masterfolie** änderst, erscheint automatisch auf **allen Folien**. Das spart enorm viel Zeit!

**So öffnest du die Masteransicht:**  
**Ansicht → Folienmaster**

Jetzt wird oben in der Seitenleiste die Masterfolie und alle Layouts angezeigt.

---

## Aufgabenstellung

### Schritt 1: Masterfolie gestalten

Klicke in der Seitenleiste auf die **oberste Folie** (die größte — das ist die Masterfolie).

**Ändere den Hintergrund:**

1. Rechtsklick auf die Folie → **Hintergrund formatieren**
2. Wähle eine Farbe oder einen Farbverlauf
3. Tipp: Ein dunkler Hintergrund wirkt oft eleganter, heller ist besser lesbar

**Ändere die Schriftart:**

1. Markiere den Platzhalter "Titel" (oben)
2. **Start → Schriftart** → Wähle eine Schrift (z.B. Arial, Calibri, Verdana)
3. Stelle die Größe auf mindestens 36 pt ein
4. Mache den Titel **fett**

**Füge ein Logo ein:**

1. **Einfügen → Symbole → Symbol** (oder **Einfügen → Formen**)
2. Wähle ein Symbol, das zu deinem Thema passt (oder erstelle eine einfache Form)
3. Platziere es in einer Ecke (z.B. oben rechts)
4. Das Logo erscheint jetzt auf allen Folien!

---

### Schritt 2: Layouts anpassen

Unter der Masterfolie siehst du verschiedene Layouts.

**Titelfolie anpassen:**

1. Klicke auf das **Titelfolien-Layout** (meist das zweite von oben)
2. Gib ihr einen etwas anderen Hintergrund als der Masterfolie
3. Verschiebe den Titel-Platzhalter in die Mitte

**Layout "Titel und Inhalt" prüfen:**

1. Klicke auf dieses Layout
2. Kontrolliere, ob Schriftart und -größe passen
3. Ändere nichts — es soll ja gleich bleiben wie die Masterfolie

**Layout "Zwei Inhalte" vorbereiten:**

1. Falls vorhanden, kontrolliere die Anordnung
2. Hier landen später Text + Bild nebeneinander

---

### Schritt 3: Präsentation erstellen

Schließe die Masteransicht (**Masteransicht schließen** in der Menüleiste).

Jetzt erstellst du 4-5 Folien:

| Folie | Layout | Inhalt |
|-------|--------|--------|
| 1 | Titelfolie | Titel deiner Präsentation, Name, Datum |
| 2 | Titel und Inhalt | Einleitung zu deinem Thema |
| 3 | Zwei Inhalte | Ein Bild + erklärender Text |
| 4 | Titel und Inhalt | Weitere Informationen |
| 5 | Nur Titel | "Quellen" (für spätere Quellenangaben) |

**Tipp:** Nutze **Start → Neue Folie** und wähle das gewünschte Layout aus.

---

### Schritt 4: Testen

- Funktioniert das Logo auf allen Folien?
- Ist die Schrift einheitlich?
- Sieht die Titelfolie anders aus als die Inhaltsfolien?
- Wenn du etwas auf der Masterfolie änderst — aktualisieren sich alle Folien?

---

## Kontrolle

Deine Präsentation sollte diese Kriterien erfüllen:

- [ ] Einheitlicher Hintergrund auf allen Inhaltsfolien
- [ ] Gleiche Schriftart auf allen Folien
- [ ] Ein Logo / Symbol auf jeder Folie sichtbar
- [ ] Titelfolie hebt sich optisch ab
- [ ] Mindestens 4 Folien mit verschiedenen Layouts
- [ ] Eine leere Quellen-Folie am Ende

---

## Zusammenfassung

| Konzept | Erklärung |
|---------|-----------|
| Masterfolie | Zentrale Vorlage — Änderungen wirken auf alle Folien |
| Layouts | Varianten der Masterfolie (Titel, Inhalt, Bild+Text) |
| Hintergrund | Farbe oder Verlauf auf der Masterfolie festlegen |
| Schriftart | Einmal in der Masterfolie setzen, gilt überall |
| Logo | Einmal platzieren, erscheint auf allen Folien |

**Merke:** Alles, was auf allen Folien gleich sein soll, machst du in der Masterfolie!

---

## Erweiterung (optional)

1. Verwende einen **Farbverlauf** als Hintergrund statt einer einfachen Farbe
2. Füge **Folienübergänge** hinzu (Übergänge → z.B. "Wischen")
3. Erstelle ein **eigenes Farbdesign** unter Entwurf → Farben → Farben anpassen
4. Füge Kopf- und Fußzeile mit Foliennummer und Datum ein (Einfügen → Kopf- und Fußzeile)
