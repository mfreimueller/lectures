## 2. Theorie: Bildlizenzen (15 min)

### Warum ist das wichtig?

- Nicht jedes Bild aus dem Internet darf einfach verwendet werden
- Urheberrecht gilt auch bei Präsentationen in der Schule
- **Faustregel:** Wenn du ein Bild nicht selbst gemacht hast, prüfe die Lizenz!

### Die drei wichtigsten Lizenztypen

| Lizenz | Erklärung | Beispiel |
|--------|-----------|----------|
| **CC0** | Public Domain — uneingeschränkt nutzbar | Unsplash, Pexels, Pixabay |
| **CC-BY** | Nutzung erlaubt, aber Namensnennung erforderlich | Flickr (mit CC-BY-Filter), Wikipedia Commons |
| **Copyright** | Alle Rechte vorbehalten — nicht ohne Erlaubnis nutzen | Google Bildersuche (Standard) |

### CC0 (Public Domain)

- **Keine Namensnennung nötig** (trotzdem nett!)
- Darfst du: kopieren, verändern, verteilen, kommerziell nutzen
- **Quellen:**
  - [unsplash.com](https://unsplash.com) — hochwertige Fotos
  - [pexels.com](https://pexels.com) — Fotos & Videos
  - [pixabay.com](https://pixabay.com) — Fotos, Vektorgrafiken, Illustrationen

### CC-BY (Namensnennung)

- Du darfst das Bild verwenden, **musst aber den Urheber nennen**
- **Quellen:**
  - [flickr.com](https://flickr.com) — Filter auf "Commercial use allowed"
  - [commons.wikimedia.org](https://commons.wikimedia.org) — viele CC-BY-Bilder

### Copyright (©)

- **Nicht verwenden!** Auch nicht für Schulpräsentationen
- Auch "Bild von Google" ist keine Quelle — Google ist nur die Suchmaschine

### Wie gibt man Quellen richtig an?

**Vollständige Quellenangabe:**
- Wer? (Fotograf / Autor)
- Was? (Titel oder Beschreibung)
- Welche Lizenz? (CC0, CC-BY)
- Wo? (URL)

**Beispiele:**

```
Bild 1: Foto von John Doe, "Technology Workspace", CC0, unsplash.com
Bild 2: "Globe icon" by Jane Smith, CC-BY 4.0, flickr.com/...
```

**In der Präsentation:**
- Quellen am Ende der Präsentation auf einer **Quellen-Folie** sammeln
- Oder direkt unter dem Bild in klein: `Quelle: ... (CC0)`

## 3. Theorie: Masterfolie (15 min)

### Was ist die Masterfolie?

- Die **Masterfolie** ist die zentrale Vorlage für alle Folien
- Änderungen an der Masterfolie → alle Folien werden automatisch aktualisiert
- Zugriff: **Ansicht → Folienmaster**

### Was kann man auf der Masterfolie einstellen?

| Element | Beschreibung |
|---------|--------------|
| Hintergrundfarbe / -design | Einheitlicher Look |
| Schriftarten | Titel- und Textschriftart festlegen |
| Logo / Grafiken | Einmal platzieren, erscheint auf allen Folien |
| Platzhalter | Position von Titel, Text, Bildern |
| Fußzeile | Foliennummer, Datum, Name |

### Layouts

Unter der Masterfolie gibt es verschiedene **Layouts**:

| Layout | Verwendung |
|--------|------------|
| Titelfolie | Die erste Folie der Präsentation |
| Titel und Inhalt | Standard-Folie mit Überschrift und Text |
| Zwei Inhalte | Zwei Spalten (z.B. Text + Bild) |
| Nur Titel | Nur die Überschrift (z.B. für Trennseiten) |
| Leer | Komplett frei gestaltbar |

**Jedes Layout** kann unabhängig von der Masterfolie angepasst werden!

### Praxistipp

1. Öffne **Ansicht → Folienmaster**
2. Wähle das oberste Element (die Masterfolie)
3. Ändere Hintergrund, Schriftart, füge Logo ein
4. Passe die einzelnen Layouts an (z.B. Titelfolie anders gestalten)
5. Schließe die Masteransicht

Alle vorhandenen Folien übernehmen die Änderungen automatisch.

## 4. Live-Demo (10 min)

**Am Beamer zeigen:**

1. Neue leere Präsentation erstellen
2. **Ansicht → Folienmaster** öffnen
3. Hintergrundfarbe ändern (z.B. dunkelblau)
4. Schriftart für Titel ändern (z.B. auf Arial Bold, 36 pt)
5. Ein Logo (z.B. Schul-Logo) in der Masterfolie platzieren
6. Titelfolien-Layout anpassen (anderen Hintergrund)
7. Masteransicht schließen
8. Neue Folien einfügen → alle haben das neue Design
9. Quellen-Folie am Ende erzeugen
